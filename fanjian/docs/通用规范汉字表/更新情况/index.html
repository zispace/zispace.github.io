    更新情况

    《通用规范汉字表》（2013 年）共 8105 个字， 其中一级字表 3500 字，二级字表 3000 字，三级字表 1605 字。

    而《通用规范汉字表（征求意见稿）》（2009 年）共 8300 个字， 其中一级字表 3500 字，二级字表 3000 字，三级字表 2000 字。

    异体字调整 #

    • 一级字表：
      • 零：与表数目的汉字“一二三四五六七八九”连用时可用“〇”替代。
    • 二级字表：
      • 皙：义为人的皮肤白。不再作为“晰”的异体字。
      • 瞋：义为发怒时睁大眼睛。不再作为“嗔”的异体字。
      • 噘：义为噘嘴。不再作为“撅”的异体字。
      • 蹚：义为蹚水、蹚地，读tāng。不再作为“趟（tàng）”的异体字。
    • 三级字表：
      • 凓：义为寒冷。不再作为“栗”的异体字。
      • 𫫇：化学名词用字，读è，如“二𫫇英”等。
      • 勠：义为合力、齐力。不再作为“戮”的异体字。
