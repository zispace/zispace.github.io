    新一站式繁体字学习手册

    前言 #

    2019年的时候因为长期出差，业余时间没有孩子捣乱，甚是无聊，加之在网上为繁体字辩护了几句，觉得我为什么不写个繁体字学习手册，让人们知道繁体字其实很简单，我当年花一个礼拜学会的东西怎么就难了？我就是想让人们知道繁体字其实什么都好，仅仅是笔画多了一点，怎么就会那么理直气壮地对繁体字说三道四，就为减省楷书那一点点笔画就这么洋洋得意？于是我就写了《一站式繁体字学习手册》。

    起初没想搞得那么复杂，觉得繁体字和古字形有很好的对应关系，单用文字描述也能清楚说明问题，于是只用文本文件编辑器写了这么一篇文章。但其实效果并不好，没有古文字字形基础的人还是很难看懂。不过，没想到仍然得到不少人的点赞，现在每天都有赞。本来赌气写的文章没想到还能帮助到别人（我感觉有些专业需要繁体字的学生在看）。现在我想：既然已经写了这么一篇文章，不如把它搞得完美点。我决定把我觉得最完美的汉字学习方法展示出来。

    本文从2020年10月份开始写，由于不出差了，业余也没有太多时间，进度比较慢，但是繁体字已经罗列完了。选字主要依据从网上找到的田英章《五千常用汉字字帖》，但这个字贴中有一些字是重复的，我也增加了一些这个字帖中没有、但经常用到的字。

    虽然是以五千字为基础罗列的字形，但我觉得基本上所有的繁体字都没有遗漏。如果有遗漏，请务必在知乎的新一站式繁体字学习手册的评论里告知我，谢谢！

    什么是繁体字，什么是简化字 #

    白日依山盡，黃河入海流。欲窮千里目，更上一層樓。

    上面这首诗中“盡”“窮”等字的写法和现在不同，现在写作“尽”“穷”，前者就是繁体字，后者是简化字。“白”“日”等字的写法和现在相同，它们既不是繁体字也不是简化字，属于未被简化的字，但习惯上统称大陆1956年推行简化字后认定的那批规范字为简化字，之前使用的为繁体字。你应该注意到“黃”这个字的写法和“黄”略微不同，这是规范字形认定标准不同导致的，不是繁简字关系。我的理解是：一部分楷书（规范字其实就是对楷书的规范，我觉得还应该有行书规范）历来就有写法小异的情况，主要原因是同一小篆字形有多种隶变形式，因此出现了个别字规范字形有多个可选方案的情况。“黃”这个字形是港台标准，由于大陆繁体字输入法通常是按台湾标准，因此打出来的繁体字倾向台湾标准。需要注意的是，简化字除了简化字形外，还合并了一部分字，例如，干合并了干（干涉）、幹（幹活）、乾（gān乾淨）三个字。

    学习繁体字的工作量有多大 #

    按《简化字总表》的说法，有352个不作偏旁的（不可类推）简化字，146个可作偏旁的（可类推）简化字，后者类推出1753个字（以《新华字典》约8000字为范围）。不可类推的字如：標（标）、蠶（蚕）、飛（飞）等；可类推的字如：僉（佥）、婁（娄）、專（专）等，可类推劍（剑）、檢（检）、臉（脸），樓（楼）、摟（搂）、屢（屡），轉（转）、傳（传）、磗（砖）等。

    从示例可见，学习类推字并不难，只要学会146个可类推字就等于学会了1753个繁体字。再加上352个不可类推字，学繁体字的全部工作量就在学会这498个字上，相当于小学（不包括一二年级）一个学期的识字量。

    在498个字中，有一部分字是由你学过的字组成的，比如：襪（袜）的“蔑”是你学过的，讓（让）的“襄”是你学过的，臟（心脏的脏）、髒（肮脏的脏）的“藏”“骨”和“葬”都是你学过的。这些字你需要记住的不是笔画怎么写，而是构字方法。

    有一部分是和你学过的字极其相似的，比如：喪（丧）、儉（俭）、監（监）、執（执）、狀（状）等。这些字的简化方式一般是取原字的行、草书写法。比如，吅在行书中通常写作三点，灬通常连写成一横，因此“僉”简作“佥”。其实，如果你写繁体字的时候不是写楷书，你完全可以按照简化字的笔画写。也就是说，简化字的连笔写法就是这个字对应的繁体字的行书写法。

    最后一部分就是确实需要关注写法的字。这部分字的多少取决于你的繁体字基础。如果基础不好的话，馬、鳥这些字可能都需要学；基础好的话可能只是不会写竊（窃）、鬱（郁闷的郁）、鑿（凿）这种字。

    学习繁体字的方法 #

    有意识的从字形和字音上分析和归类汉字是快速学会繁体字的不二法门。认识简化字的人学习繁体字的过程与其说是学习写字，不如说是学习解字。真正让你去记笔画的情况少之又少。当然，这只是我个人经验。每个人的思维方式不同，比如我看到“监”后会自然联想到鉴、览、临、贤、坚、竖等字，而我对象则不行，她更自然的是联想到与监同音的字。这样可能不利于快速学习繁体字，要有意识的像下面这样从音形两方面去做分类，才能更快的学会繁字体。

    荣、荧、莹、萤等字都是字头简化，原字为榮、熒、瑩、螢。这是单纯的分析字形就能记住的字。当然，如果你能分析字音就更好了，会发现这些字都是形声字，以（熒本字）为声旁。

    汉、难、欢、观、劝等字都是声旁被符号“又”替换，原字为漢、難、歡、觀、勸。这些字不仅要分析字形，还要分析字音才能更好的记住。漢和難的发音近似，韵母相同，因此它们是同一个声旁。歡、觀、勸的发音近似，韵母相同，但读音不同于漢和難，因此它们用的是另一个声旁“雚”，而这个声旁还用于鹳、灌、罐等你学过的字，可帮助记忆。

    格式说明 #

    字例括号内第一个是简化字，前面加星号（※）的是异体字，加三角号（△）的是俗字。例如，罈（坛，罈罈罐罐，※罎）。

    甲金文的字形按时间先后排列，且尽量取西周及以前的；楷书尽量取唐朝及以前的；行草书也是尽量取时间靠前的。其实楷行草历代变化不大（至少表面上看起来是，笔法什么的就不懂了）。楷行草中列举的字例有写法略微不同的，不是前后演化关系，只是几种常见写法。

    后起义、后起字的“后起”指的是魏晋至唐宋这段时间出现或普及；晚起义、晚起字的“晚起”指的是元明及以后出现或普及。
