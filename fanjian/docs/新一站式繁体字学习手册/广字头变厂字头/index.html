    广字头变厂字头 #

    廠（厂）、廁（厕）、廄（厩）、廂（厢）、廚（厨）、廈（厦）、廝（厮） #

    金文中從“厂”的字應該是從“石”的省形（參《金文形義通解》、季旭昇），例如“段”（碫、鍛本字，擊打石頭）、“原”（水從巖穴流出）、“厚”（醇厚，酒貯藏在山崖洞穴）等字。甲骨文的“石”是指山石，所以《説文》訓“厂”爲“山石之厓（崖）巖”。另外，“厂”與象屋形的“广”本不同字，然而兩者字形相近，容易混淆，故金文偏旁偶有互作。“厂”、“厈”、“岸”古本一字，“厂字本象石岸之形。周秦或加干爲聲符作厈，後又或於厈上加山爲意符作岸，故厂、厈與岸實爲一字。”（高鴻縉《中國字例》）。

    “广”象房屋形。“广”象比“宀”簡單的建築。甲、金文從“︿”從“丨”，“︿”是屋宇，象屋頂向兩邊傾斜，“丨”是牆，王筠認爲“广”象三面有牆，而“宀”則四面有牆。甲、金文未見單獨之“广”，然有用爲部件。“宀”、“广”作爲金文部件常通用。“广”與“厂”有別。“广”象房屋，“厂”則與甲骨文“石”同，象崖巖、山石。不過“广”、“厂”作爲金文部件，因形近偶有相混。裘錫圭指出，如果所指的建築是比較簡單的，或者主要不是供人居住的，字形往往從“广”，如“廬”、“廊”、“廡”、“府”、“庫”等，在簡化字裏，“广”旁有時簡化爲“厂”，如“厨”、“厩”、“厕”等。

    “宀”與後世字的對應，主要有三説。一，馬叙倫認爲“宀”是“穴”的初文，蓋古人最初穴居，後營宮室。田藝衡曰：“古者穴居野處，未有宮室，先有宀，而後有穴。宀，當象上阜高凸，其下有𠕄可藏身之形，故穴字从此。室家宮宁之制，皆因之。”二，于省吾認爲是“宅”的初文，而甲骨文“宅”字則作動詞，表示居住。三，蔡哲茂、徐中舒認爲是“廬”的初文，“廬”、“六”音近，且“宀”、“六”形近，故甲骨文“宀”可假借爲數字“六”。按“宀”象房屋形，是否“穴”、“宅”、“廬”的初文，待考。
