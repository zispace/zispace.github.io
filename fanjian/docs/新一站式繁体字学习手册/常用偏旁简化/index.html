    常用偏旁简化 #

    如果想学繁体字的话，不用先从这一章节看，从后面章节开始看就行。因为只要不是后面章节提到的字，都是简单的偏旁替换。比如，当你把后面章节的字都掌握得很熟练后，看到“轶”这个字自然知道它不在后面章节里，那就是个简单的“车”字旁替换了。

    貝（贝）字旁简化 #

    甲骨文金文小篆隶书楷书行书草书

    象贝壳之形。

    “貝”作偏旁时的写法：

    行书 財草书 財

    类推字：

    貝（贝）、則、側、鍘、貢、財、貶、狽、唄、敗、販、貨、貪、貧、賞、賈、貳、貼、貽、貸、貿、測、惻、費、賀、鋇、賊、賄、賂、賅、賃、資、賒、賑、賭、賠、賦、賜、膩、贅、賺、賽、贈、贍、贏、贛

    “負”小篆作，从人，从貝，表示以背载物。

    負、賴、獺、懶、癩、籟

    “員”甲骨文作，上从圆形（“圆”的初文，亦是“员”的声符，以◎表示），下从“鼎”，鼎是古代的食器、祭器，鼎口多是圆的，故置圆形于鼎上，以会圆形之意，本义是圆形（林义光）。圆圈形和“員”字都是“圓”的初文。

    員、圓、隕、損、塤、韻（※韵）、勛（勋、※勳）

    “貞”甲骨文作、，开始假借“鼎”来表示，后来加“卜”（象卜兆）为意符，专门表示贞卜的“貞”。

    貞、偵、幀、禎

    “責”甲骨文作，金文作。从“貝”，“朿”声，本义是赋税，引申为责任。

    責、債、嘖、漬、績

    “賏 〔yīng / ㄧㄥ〕”小篆作，从二貝，表示颈项上的饰物。

    嬰（婴）、罌、纓（缨）、櫻、鸚（鹦）

    “賁 〔bēn / ㄅㄣ〕”小篆作，从貝，卉声，表示饰物。

    賁、噴、濆、憤

    “贊”小篆作，从貝，从兟（ 〔shēn / ㄕㄣ〕，进也），本义为导引宾客进见主人。

    贊、攢

    “貫”金文作，从二“貝”从“丨”，象用绳穿起两个贝币之形。本义是贯穿钱贝的绳索，即钱串，引申为贯穿、贯通、串连、连结。

    貫、慣、

    “貴”金文作，小篆作，“貴”的初文从相向的二“爪”从一中间有短横画的竖笔，“爪”象手形。象两手持一物之形。何琳仪认为会小物易弃之意，是表示遗弃的“遺”的初文。

    貴、匱、遺、饋、潰、憒、聵、簣

    “〔suǒ / ㄙㄨㄛˇ〕”《说文》：“貝声也。从小、貝。酥果切。”

    鎖、瑣

    金字旁简化 #

    甲骨文金文小篆隶书楷书行书草书

    金文“金”字最初从“王”（象斧钺之头朝下形）从“吕”（二粗点象两块金属饼）从“亼”（“今”之省形），“今”是“金”的声符。“金”最初专指铜，后来泛指一切金属，再后来专指黄金。

    烏（乌）字旁简化 #

    甲骨文金文小篆隶书楷书行书草书

    “烏”未见甲骨文，金文乌字一个重要特征是口部向上张开，这是因为乌鸦好反哺，母乌鸦张开嘴巴把已吞进体内的食物吐出来喂小乌鸦。此外，大部分字形作鸟形而没有眼睛，盖乌鸦全身乌黑而不见眼睛。但亦有些可以确定为“烏”的字有画出眼睛，如毛公鼎。

    西周早期借作语气词，西周晚期渐简写作“於”（参见“於”），今作“嗚”。

    “烏”作偏旁时的写法：

    行书 嗚草书 嗚

    类推字：

    烏（乌）、嗚、塢、鎢

    糸字旁简化 #

    糸: （〔mì / ㄇㄧˋ〕）

    甲骨文金文小篆隶书 給楷书 給行书 給草书 給

    象束丝之形，上下两端或有线绪，本义是细丝。后世凡涉及“絲”的字均以“糸”为旁或主要部件。在较早的古文字中，“糸”和“絲”是同一个字（裘锡圭），后来才分化为两个字。甲骨文中“糸”与“幺”亦互用无别。

    “糸”作偏旁时的写法：

    > “糸”在字的左侧时下部化为三点，在其他部位时保持不变。
    楷书 絲行书 絲草书 絲

    草书的“糸”字旁写成类似“子”字的形状，“孫”字的草书右侧写成“小”，这是和“絲”的草书唯一的不同。

    类推字：

    絲（丝）、噝、糾、紅、約、級、紀、純、紗、納、紛、紙、紋、紡、紐、紉、紂、紈、絝、紕、組、細、終、紹、紳、絆、絀、綁、絨、結、給、絝、絡、絕（绝）、絞、統、絢、絳、絹、綏、綰、緒、綾、綺、維、綿、綢、綽、繃（绷、※綳）、綜、綻、綴、緋、綹、綬、緞、緩、緝、編、緣、緬、緘、緲、締、縫、縟、縛、縊、縝、縮、縹、繆、繅、繚、繕、繳、繮、辮

    绝字右半边不是色，而是从刀从卩。港台的繁体印刷体还是遵从这个规矩。简化字的“绷”从异体字简化。

    类推字：

    見（见）、現、規、覓、視、硯、靚、覦、窺、覲、覷

    車（车）字旁简化 #

    甲骨文金文小篆隶书楷书行书草书

    甲金文“車”字象古时的车形；写法很不固定，有的将车的部件全数画出，有的只保留轮和轴；西周早期出现和现今同形的“車”字。

    “車”作偏旁时的写法：

    行书 軒草书 斬草书 輸
    草书 連草书 軍

    个人感觉简化字“车”字的写法在草书中并不多见，作左旁时“斬”字草书字例的写法比较多见；在其他部位时，“連”字草书第一例写法比较多见。

    类推字：

    車（车）、軋、軌、輓（挽，用车运输）、陣、連、庫、軒、斬、軟、軸、軲、轆、載、蓮、較、漣、塹、輒、輔、輛、嶄、漸、輦、輥、暫、輟、輩、鏈、褲、褳、輸、轡、輻、輯、轅、轄、輾、輿、攆、鰱、鏨、轍

    輓：（1）拉车、牵引。或作“挽”，《说文》作“輓”。（2）用车运输。（3）悼念，哀悼死者。本作“挽”，后写作“輓”。（4）通“晚”。

    挽：从“手”，“免”声，本义为牵引。“挽”不见于甲金文及《说文》。《小尔雅·广诂》：“挽，引也。”《玉篇·手部》：“挽，引也。”先秦典籍用其本义，如《庄子·天运》：“今取猨狙而衣以周公之服，彼必齕啮挽裂，尽去而后慊。”唐诗亦用其本义，如杜甫《前出塞九首》：“挽弓当挽强，用箭当用长。”挽可用作扭转、挽回；引荐、提携等义，不可用輓。详见挽、輓。

    “軍”金文作，从車，勹（螾，蚯蚓的象形）声或匀声。表示军队。

    軍、諢、揮、葷、渾、暉、暈、輝

    頁（页）字旁简化 #

    甲骨文金文小篆隶书楷书 順行书 順草书 順

    甲金文象人之头及身，以人身之小来衬托出其头部，因此“頁”本义是头部。“頁”与“首”字的分别在于书写时是否附带画出人身，因此两字是异体字。甲骨文“頁”字在人头上或有头发，或省去发形；战国文字头上发形多讹为止形，小篆则省去人首上的头发。

    类推字：

    頁（页）、頂、頃、項、順、須、頊、頑、頓、噸（吨）、傾、頌、頒、煩、預、碩、領、頗、頡、頜、頦、頤、頻、頷、頹、穎、顆、擷、題、顎、顓、額、顛、瀕、囂、巔、顫、顴、顰、癲

    注意：鬍鬚（胡须）二字上从髟（〔biāo / ㄅㄧㄠ〕表示毛发）。

    “胡”本义是动物颏下赘肉，颏下胡须亦称胡，后加髟分化出鬍字。详见鬍。

    “須”本义是胡须，但后借作虚词，另造鬚表本义。详见鬚。

    門（门）字旁简化 #

    > 就是左右两个“日”字的笔顺。
    甲骨文金文小篆隶书楷书行书草书

    象两扇门形。行书用两点或三点略写两个“日”形，三点的笔顺为“小”字的笔顺，即先写中间一点（实为左侧日形的横折，见第一字），再写左侧一点（实为日形内部结构的一略），再写右侧一点（为右侧日形略写）。

    “門”作偏旁时的写法：

    行书 聞草书 聞草书 間草书 閣

    类推字：

    門（门）、們、閃、閆、閉、問、闖（闯）、悶、捫、閔、閏、閘、聞、閥、閣、閡、閨、閩、閭、閱、潤、憫、澗、閹、闍、閻、闡（阐）、燜、擱、闊、闋、櫚、闔、闕、藺、闞（詳見7.14）

    間〔jiān / ㄐㄧㄢ〕、〔jiàn / ㄐㄧㄢˋ〕（时间）、閒〔xián / ㄒㄧㄢˊ〕（空闲）、閑〔xián / ㄒㄧㄢˊ〕（闲邪存诚）、嫺（娴，※嫻）、癇（痫）、簡（简）

    “间”是“間”的简化字；“闲”合并简化了“閒”“閑”。详细解释如下：

    間（〔jiān / ㄐㄧㄢ〕、〔jiàn / ㄐㄧㄢˋ〕，时间）：按《王力古汉语字典》所说，《说文》无“間”字，間为閒的俗体。经查《汉字全息资源应用系统》，只在战国早期三晋文字中（間右庫戈）有一例“間”字，故間可能确为閒之俗体或分化字。现在間和閒就如做和作一样，功能有别。間主要用于時間、空間、間隙等词。閒主要用作空閒、悠閒等词，读作〔xián / ㄒㄧㄢˊ〕。读古书时应当知道它们曾经通用。由于間分化自閒，楷书从間的字其实本从閒，例如簡的小篆及以前的字形从閒。

    閒（〔jiān / ㄐㄧㄢ〕、〔jiàn / ㄐㄧㄢˋ〕、〔xián / ㄒㄧㄢˊ〕，空闲）：包含間和閒的全部义项，但现在主要用作〔xián / ㄒㄧㄢˊ〕这一读音下的义项：（1）閒暇。（2）无关紧要的（后起义）。閒事，閒話，閒愁。（3）通“嫺”。

    閑（〔xián / ㄒㄧㄢˊ〕，闲邪存诚）：（1）木栅栏，用于禁卫。周礼夏官虎贲氏：“舍則守王閑。”（2）马厩。周礼夏官校人：“邦國六閑，馬四種。”（3）法度。论语子张：“大德不踰閑。”（4）限制，防备。左传昭公六年：“是故閑之以義。”（5）大的样子。诗商颂殷武：“旅楹有閑。”（6）通“嫺”。娴习，熟练。（7）通“閒”。暇。（8）閑閑。1、从容自得的样子。诗魏风十畝之閒：“十畝之閒兮，桑者閑閑兮。”2、动摇的样子。诗大雅皇矣：“臨衝閑閑。”3、广博的样子。庄子齐物论：“大知閑閑，小知閒閒（〔jiān / ㄐㄧㄢ〕）。”閒閒：分别是非的样子。

    閑邪存誠（闲邪存诚）：防止邪念滋生，以保存诚实之心。閑：防备。

    蕩檢踰閑（荡检踰闲）：行为放荡，不受礼法约束。閑：法度。

    鬥（斗，鬥爭）、鬧（闹）、鬨（哄，鬥鬨、內鬨、起鬨）、鬫、鬩（阋）、鬮（阄） #

    鬨（哄）: 〔hòng / ㄏㄨㄥˋ〕

    甲骨文金文小篆隶书楷书行书草书

    甲骨文从二人对立以手相格斗，且有怒髪冲冠之状，本义是二人相搏。由于鬥和門形体相近，从鬥的字都有从門的异体。我搜到的鬥的隶、楷、行、草多数都写作闘。鬥还有几种异体：鬪、鬭、鬦，这些异体自然也都可以替换成从門。草书的鬥是不是只有基于闘的一种写法我不太清楚。

    “斗”甲骨文作，小篆作。表示一种舀水器具。

    勾心斗角（亦作钩心斗角）的斗是“鬥”，语出唐·杜牧《阿房宫赋》：“各抱地勢，鉤心鬥角。”形容宫室建筑的结构精致巧妙。后比喻诗文的布局结构精心巧制，争奇斗胜。也用于比喻竞斗心机，刻意经营。

    之所以把勾心斗角专门拿出来说，真的是因为我直到现在才知道是“鬥角”而不是“斗角”。我平时也是读四声的，我估计不会有人读三声。那为什么我会一直以为是斗角而不是鬥角呢？这得拜我一个语文老师所赐。我深深记住了他对这个成语的解释。有一次上课他问我们谁能说明勾心斗角的意思，谁能逐字解释一下。没有人能解释。当然，大家都知道勾心斗角是明争暗斗的意思，可为什么是勾心、为什么是斗角呢？为什么用这几个字？没人能解释出来。于是老师信誓旦旦的给我们解释说勾心斗角最初是指房屋结构错综复杂，勾心和斗（〔dǒu / ㄉㄡˇ〕他是这么读的）角是指房屋的两种结构形式，能体现出房屋结构的精妙。并一再强调“斗”不是争斗，应该读作 dǒu jiǎo （他非常郑重的用了三声）。于是乎自那以后我一直以为勾心斗角读四声是误读，斗角可能是类似于斗形的房屋结构跟争斗无关，因此应该读三声而不是四声。

    那么，勾心斗角的正确解释是什么呢？我从网上找了个我觉得靠谱的解释：“勾心斗角”亦作“钩心斗角”，源于唐代杜牧创作的散文《阿房宫赋》：“各抱地势，钩心斗角”。意思是：各自依着不同的地势，参差错落，精巧工致。心：宫室的中心。角：檐角。诸角向心，叫“钩心”；诸角彼此相向，像戈相斗，叫作“斗角”。“勾心斗角”本意指阿房宫建筑结构严密吻合，精致巧合，即“心相勾，角相斗”。

    “鬧”從“市”，從“鬥”，本義爲喧嘩、嘈雜、不安靜。《說文新附》：“不靜也。从市、鬥。”

    “鬮”鬥取也，从鬥，龜聲。古侯切。例如，抓鬮（〔jiū / ㄐㄧㄡ〕）。

    “鬨”《康熙字典》：“《廣韻》《集韻》《韻會》《正韻》並胡貢切，音。《說文》鬭也。《孟子》鄒與魯鬨。《趙註》鬭聲也，猶構兵而鬭也。又《廣韻》胡絳切《集韻》《韻會》胡絳切，並音巷。義同。或作。”

    《王力古漢語字典》：“〔hòng / ㄏㄨㄥˋ〕，胡貢切，去，送韻，匣。東部。①爭鬥，戰鬥。《説文》：‘鬨，鬭也。’《孟子·梁惠王下》：‘鄒與魯鬨’。②喧鬧。宋·王之道《鵲橋仙》：‘十年湖海鬨樵歌’。按，字俗作‘閧’，《廣韻》《集韻》皆作‘閧’，周祖謨《廣韻校本》改作‘鬨’，是。”

    《王力古漢語字典》認爲内訌的“訌”和“鬨”是同源字。“訌”户工切，平，東韻，匣。東部。爭擾、潰亂。《詩·大雅·召旻》：“天降罪罟，蟊賊内訌。”《明史·流賊傳》：“政繁賦重，外訌内叛。”“訌”和“鬨”應是同源，古音同在匣母東部，中古只有平、去之分，都有爭吵、打鬭之義。《説文》：“訌，䜋也。”意義偏重於爭擾吵鬧。又“鬨，鬭也。”意義偏重於打鬭。在古籍中，“訌”的使用範圍窄，多用於“内訌”“外訌”；“鬨”的使用範圍廣，除爭鬭義外，還用於喧鬧義。

    “鬨”（〔hòng / ㄏㄨㄥˋ〕）這個字比較早出現，按《王力古漢語字典》，爭鬥、喧鬧義都用這個字。個人認爲喧鬧義和亂哄哄的“哄”意義相近，因此這兩個字在一些詞中通用（至少在臺灣的字典中如此），比如，鬨堂大笑、一鬨而散，也可作哄堂大笑、一哄而散。但是有些詞同樣是喧鬧義，在臺灣的字典中卻只有一種寫法。我覺得不能糾結於此，字典只是把常見的用法羅列出來了，本質上講其實就是歷史上這兩個字用混了。其實，如果“鬨”有喧鬧義的話，亂哄哄寫成亂鬨鬨也沒問題吧？爲什麼不這麼寫？我覺得是因爲在“亂哄哄”這個詞出現的時候“哄”比“鬨”更常用，尤其是口語詞中更不可能用“鬨”。

    “哄”（〔hōng / ㄏㄨㄥ〕、〔hǒng / ㄏㄨㄥˇ〕）是後起象聲字，鬧哄哄、哄騙是這個字。本文原來在此處還舉了“鬨搶”的例子，有網友說不對，應該是“哄搶”。我查了下臺灣的字典，確實是“哄〔hōng / ㄏㄨㄥ〕搶”，說這個詞是“大陸地區指群眾起鬨搶奪的行爲”。我一直以爲哄搶的哄讀四聲，是我搞錯了，但從字義上講“鬨搶”也不錯，只是大陸這邊把兩字一合併只能當多音字解了。

    按《類篇》《正字通》《康熙字典》等書，鬫和闞是兩個字，前者讀〔hǎn / ㄏㄢˇ〕，老虎的吼聲，《詩經·大雅·常武》：“進厥虎臣，鬫如虓虎。”，還有一個讀音〔xiàn / ㄒㄧㄢˋ〕，獸怒聲；後者讀〔kàn / ㄎㄢˋ〕，望，同瞰，也是姓氏。認爲鬫寫作闞是從俗了。

    《説文》中只有闞字，“闞，望也。从門敢聲。苦濫切”。

    《詩經》中這個“鬫如虓虎”到底用的哪個字就不知道了。現在大陸和港臺都統一用“闞”這個字，沒有特殊原因是見不到“鬫”這個字的。

    現在字典中闞主要有兩個讀音〔kàn / ㄎㄢˋ〕和〔hǎn / ㄏㄢˇ〕。

    龜（龟）、鬮（阄） #

    > 台湾标准第十笔和十三笔出头，写到第七笔处；大陆标准第十笔和十三笔写到第八笔处即可。
    甲骨文金文小篆隶书楷书行书草书

    甲金文象侧视或俯视全龟之形，龟之头、尾、足、甲俱全。本义是龟。“龜”与“黽”字形相似，差别为“龜”有尾、后腿直伸，“黽”无尾、后腿回折。战国楚文字“龜”与“黽”同形。

    “鬮”鬥取也，从鬥，龜聲。古侯切。例如，抓鬮（〔jiū / ㄐㄧㄡ〕）。

    艸（草的本字） #

    > 艸字头传统上分两部分写，大陆的标准是合起来写。
    甲骨文金文小篆隶书楷书行书草书

    在《汉字全息资源应用系统》中未找到艸的甲骨文，但在《汉语多功能字库》中有。艸的甲骨文像枝茎柔弱的植物之形，是“艸”的象形初文（裘锡圭、陈剑）。“屮”是由省去下半的写法演变而来，是“艸”的异体。甲骨文的“艸”下或从“口”，乃“造”字所从的声符。

    “草” 从“艸”，“早”声，本义为栎树的果实。传世古书多借“草”为“艸”，表示草本植物。

    • 先写两竖点再写横和一竖，
    • 先写两点再写一横。
