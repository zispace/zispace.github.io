    其他 #

    搜、蒐 #

    蒐〔sōu / ㄙㄡ〕：草名。即蒨草。《説文》：“蒐，茅蒐，茹藘。人血所生，可以染絳。”《山海經·中山經》：“（釐山）其陽多玉，其陰多蒐。”郭璞注：“茅蒐，今之蒨草也。”

    “蒐”還有“搜集，尋求”的意義。《文選·陸機〈辨亡論〉》：“於是講八代之禮，蒐三王之樂。”李善注：“蒐與搜，古字通。”《宋史·職官志四》：“以稱朝廷蒐補闕軼，緝熙彌文之意。”明·胡應麟《少室山房筆叢·經籍會通一》：“隋文父子，篤尚斯文，訪輯蒐求，不遺餘力。”

    臺灣在一些偏重“尋求”義的詞彙中仍然用“蒐”，經常能在新聞中看到“蒐羅”“蒐集”“蒐索”等詞。這些詞彙用“搜”也是正確的。

    大陸並沒有將“蒐”合併入“搜”，只是一般場合不會將“蒐”用於“搜集，尋求”義。可能古文中用“蒐”的地方也都替換成“搜”了。

    “蒐”在臺灣的字典裏還有幾個義項：

    ① 古稱春天或秋天的打獵爲“蒐”。《左傳·隱公五年》：“故春蒐、夏苗、秋獮、冬狩，皆於農隙以講事也。”《公羊·桓公四年》：“四年春，正月，公狩於郎。狩者何？田狩也：春曰苗，秋曰蒐。”《三國演義》第二〇回：“古之帝王，春蒐夏苗，秋獮冬狩，四時出郊，以示武於天下。”

    ② 檢閱車馬。《左傳·宣公十四年》：“告於諸侯，蒐焉而還。”晉·杜預·注：“蒐，簡閱車馬。”《左傳·昭公八年》：“秋，大蒐於紅。”晉·杜預·注：“大蒐，數軍實，簡車馬也。”

    ③ 聚集。如“蒐集”。《爾雅·釋詁下》：“蒐，聚也。”晉·郭璞·注：“春獵爲蒐，蒐者，以其聚人眾也。”《左傳·成公十六年》：“蒐乘補卒，秣馬利兵。”宋徐夢莘《三朝北盟會編》卷九十三：“任賢使能，信賞必罰，蒐卒豐財，以謀大舉。”（《王力古漢語字典》並無此義項，可能是認爲此義項與“搜集，尋求”義同，這一點從其引文可以看出——《宋史·李植傳》：“蒐選強仕，以重軍勢。”跟臺灣字典的“蒐乘補卒”差不多的意思。）

    ④ 隱藏。《左傳·文公十八年》：“靖譖庸回，服讒蒐慝。”晉·杜預·注：“蒐，隱也；慝，惡也。”

    勻（※匀） #

    “勻”甲骨文，金文。“勻”字金文從“吕”（象兩塊圓形金餅），從（螾，後作蚓，蚯蚓的象形），“螾”是聲符，“勻”是“鈞”的本字，本義是重量單位。

    金文“鈞”或替換形符“吕”作“金”。“吕”其後線條化爲兩橫，故《説文》誤説“鈞”從“二”。

    金文讀作“鈞”，是重量單位，多友鼎：“鐈鋚百勻（鈞）”。又用作人名和地名用字。

    電腦字庫中“勻”和“匀”是兩個字，但其他從“勻”的字並沒有作區分。臺灣定“勻”爲正體，大陸定“匀”爲正體。

    沒（※没） #

    秦简牍小篆楷书行书草书

    《説文》：“沒，沈（沉）也。從水從。莫勃切。”“，入水有所取也。从又在囘下。囘，古文回。回，淵水也。讀若沫。莫勃切。”

    臺灣以“沒”爲正體，大陸以“没”爲正體。古代楷書書法“沒”的右半邊已經類化成與“投”“般”等字的“殳”一樣了，只是與現今兩種寫法都不同。現今的兩種寫法似乎是從古代印刷體來的。

    著（△着） #

    “著”從“艸”，“者”聲，本義爲明顯、顯著。《禮記·樂記》：“好惡著，則賢、不肖別矣。”謂所好得其善，所惡得其惡，是好惡著，則賢與不肖自然分別矣。

    “着”是“著”的俗字。“著”在大陸只保留〔zhù / ㄓㄨˋ〕一個讀音和其對應的意義，除此之外其他地方用“着”。這也是唯一一個一繁對多簡。

    甲骨文金文小篆隶书楷书行书草书

    金文從“木”從數點從“口”，構形初義不明。“木”形或訛變爲“止”形，“口”形或訛爲“其”、“皿”，或加繁筆作“甘”。

    金文“者”字用作特指代詞，用於動詞、形容詞或動詞性短語之後，構成名詞性短語。中山王兆域圖：“不行王命者，（殃）（連）子孫”。中山王鼎：“昔者，吾先考成王，（早）棄群臣”。又用爲“諸”，如“者（諸）侯”、“者（諸）士”、“者（諸）尹”等。邾公牼鐘：“以喜者（諸）士”。
