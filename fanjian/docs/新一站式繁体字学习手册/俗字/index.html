    俗字 #

    本文列的其他各种简化方式里也包含俗字，俗字和简化方式不是同一维度的概念，俗字不是一种简化方式，只是有些俗字可以归类为某种简化方式。这里之所以列了俗字一类，就是因为有一部分俗字的简化方式我不知道该怎么划分，这些字既不是俗字常见的符号替换，也不是纯粹的记号字，更像是以讹传讹、部件类化或笔画小异导致的错字。因此，在这里单独列了一类俗字。

    兩（两）、倆（俩）、輛（辆）、魎（魉） #

    甲骨文金文小篆隶书楷书行书草书

    金文作“㒳”，或于“㒳”上添一横，为增繁饰笔。“兩”字是截取古文字“車”字的部分构形而成，象车辕前部衡上系以双軶（牛马拉东西时驾在脖子上的器具）之形（参于省吾）。假借为两个、重量单位等。

    㒼、滿（满）、瞞（瞒）、蟎（螨）、蹣（蹒） #

    甲骨文金文小篆隶书楷书行书草书

    㒼〔mán / ㄇㄢˊ〕，《说文》：“平也。从廿，五行之数，二十分为一辰。㒳，㒼平也。读若蛮。母官切。”

    廛、纏（缠）、瀍、躔 #

    甲骨文金文小篆隶书楷书 纏行书 纏草书 纏

    廛〔chán / ㄔㄢˊ〕，《说文》：“一亩半，一家之居。从广里八土。直连切。”古代指城镇一户人家所住的房屋和宅院，也指城镇民居。

    除常用的“纏”字外，其他从“廛”的字都未简化。

    俗书中“厘”即可替代“廛”，也可替代“釐”。

    弔（吊）、吊 #

    甲骨文金文小篆隶书楷书行书草书

    甲金文象人身缠绕着以丝绳绑着的箭矢，疑“弔”是“繳”（系箭用的丝绳）的初文。（杨树达、郭沫若、周法高）“弔”“繳”古音相近。本义为射取鸟兽的矰矢。吴大澂认为系箭而射是男子的事情，所以“弔”被借用作“叔”，作为男子的美称。杨树达、郭沫若、洪诚则认为“弔”“叔”纯粹是音近假借。伯叔字用“叔”来表示，是秦汉以后之事。

    “吊”是汉唐之间出现的俗字。后多用“弔”表示祭奠、慰问、凭吊等义，而“吊”多表示悬挂或作量词。作量词时也可以用“弔”。台湾应该是保持以上这些用法。

    “屌”这个字自然是个后世俗字，只有从吊的写法。不知道这个字什么时候出现的，《水浒传》等文学作品中用“鸟”字。

    雋（隽）、鐫（镌） #

    甲骨文金文小篆隶书楷书 鐫行书 鐫草书 鐫

    “雋”从“隹”从“弓”，字会以弓射鸟。本义为射杀野鸟，后由于野鸟多具肥肉，引申为肥肉，又有美好之义。《说文》：“雋，肥肉也。从弓，所以射隹。长沙有下雋县。”按《龙龛手鉴》“雋”字下部讹变为从“乃”，字作“隽”，为简化字所本。

    这个字下面写成“乃”应该是多方面原因造成的必然结果吧。篆文的乃作，再加上美观和书写便利性的考虑，出现隽这个字形几乎是个必然。

    大陆视其为旧字形；港台以其为标准字形。

    豬（猪）、瀦（潴） #

    甲骨文金文小篆隶书楷书 豬行书 豬草书 豬

    豕，象豬之形，普遍的特点是明显的腹部线条及微垂的尾巴，偶尔有字形在“豕”的颈上加上长而密的毛发。

    豬，从豕，者声。古代“豬”“豕”相通，二字为方言之别。《方言》卷八：“猪，北燕朝鲜之间谓之豭，关东西或谓之彘，或谓之豕。南楚谓之豨，其子或谓之豚，或谓之貕，吴扬之间谓之猪子。”

    秈（籼） #

    秈，一种稻米类型。叶片较宽大且长，色淡绿，茎秆较高。谷粒呈细长或扁形，颖上细毛少而短，米饭煮起来较为干硬。籼稻一般学术上称为“印度型稻”，其直链性淀粉在百分之二十五以上。

    祕（秘） #

    秘，俗“祕”字。《类篇》：“兵媚切。密也。又蒲結切。香也。”《正字通》：“舊註俗祕字。按秘祕音義別。《轉注古音》：‘秘，蒲結切，香艸也。’”

    攜（携） #

    攜，小篆作，从“手”，“巂”声，本义为提起、提着。

    巂，金文作，从“屮”从“隹”从“丙”，“隹”象鸟形，“屮”象鸟冠，本义是鸟，一说“丙”象鸟尾，待考。金文用作人名，小篆于下增“口”，《集韵》：“巂，鸟名。”《说文》：“周燕也。从隹，屮象其冠也。㕯声。一曰蜀王望帝，婬其相妻，慙亡去，为子巂鸟。故蜀人闻子巂鸣，皆起云‘望帝’。户圭切。”

    貓（猫） #

    豸〔zhì / ㄓˋ〕，甲骨文，金文。象张口之猛兽，侧视作二足长尾之形（徐中舒）。本义是野兽。徐中舒认为有足之猛兽如豹、貔、豺等字皆从“豸”，是知《尔雅·释虫》及段注所说皆非其本义。唐兰首先释读此字为“豸”。一说象猫科动物，长脊兽，如猫、虎之类（马叙伦、马如森）。豸本应读作“貓”音，如“貌”从“豸”标声。姑备一说。甲骨文用作地名。秦简表示无脚的虫。

    狸也为貍之俗体。但在狐狸一词中，台湾用狸字；狸貓一词中，狸和貍皆可。大陆似乎也并未将其定为繁简字，而是互通字。

    撐（撑） #

    撐，从手，牚声。

    牚〔chèng / ㄔㄥˋ〕，柱。〔chēng / ㄔㄥ〕，支撑。“牚距”互相撑着。引申为以言语相抗拒。

    按，今本《说文》无牚字，止部有䟫字，段玉裁注：“今俗字䟫作撐。”朱骏声《说文通训定声》：“字亦作牚，作撐。”《五经文字》：“直庚反。䟫距之䟫，见《考工记》。”

    䟫，金文，齐鲁文字。《説文解字注》：“歫也。今音丑庚切。古音堂。今俗語亦如堂。考工記。維角䟫之。大鄭曰。䟫讀如牚距之牚。牚距、卽䟫歫字之變體。……”

    屜（屉） #

    屜：① 履中薦。《集韻》以之爲“屟”（〔xiè / ㄒㄧㄝˋ〕）的別體。② 抽屜。

    按《王力古漢語字典》“屜”“屟”“屧”（〔xiè / ㄒㄧㄝˋ〕）都解作“履中薦”。《説文》作“屟”“从尸，枼聲。穌叶切”。段玉裁注：“即今婦女鞵（鞋）下所施高底。其字本音他頰切，轉爲他計切。今籢匱有抽屜，本即屟字。”

    《王力古漢語字典》無“屉”字。《康熙字典》：“屉，《字彙》他計切，音替。鞍屉也。”《正字通》：“俗字。舊註他計切。音替。鞍屉。泥。”

    蝨（虱） #

    《説文》：“蝨：齧人蟲。从䖵，卂聲。所櫛切。”

    卂：金文。《説文》：“疾飛也。从飛而羽不見。凡卂之屬皆从卂。息晉切。”

    《干祿字書》《龍龕手鑑》等認爲“虱”爲俗字。《正字通》《字彙》等認爲“虱”同“蝨”。由於“虱”很像“蝨”之訛形，暫列在俗字裏。

    臺灣異體字字典：“虱”字字義概同“蝨”，然一般寫“虱目魚”時，不用“蝨”字，故標準字體將之分化爲二正字，“蝨”另收於常用字。按：此字兼爲“蝨”之異體字。

    脈（脉，※衇） #

    “脈”古作“”，亦作“衇”，俗寫作“脉”。《説文》：“（），血理分衺（邪）行體者。从从血。（脈），或从肉。（衇），籒文。”今繁體字作“脈”，簡體作“脉”。

    “”金文。《説文》：“（）：水之衺（邪）流別也。从反永。凡之屬皆从。讀若稗縣。徐鍇曰：永，長流也。反即分也。匹卦切。”

    “永”甲骨文，金文。甲金文象河流很長，並有支流。本義是江河源遠流長。引申爲長久、永久。古文字“永”或向左，或向右，左右無別，同時表示兩個詞，一是長久的“永”，二是支流、支派的“”（派）。後來爲了使字義明確，才規定以字形向左的爲“永”字，向右的爲“”字（裘錫圭）。一説“永”象人在水中游泳 （參高鴻縉、高田忠周、徐中舒、劉興隆），是“泳”的初文，本義爲游泳。
