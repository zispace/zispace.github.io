    古字形 #

    達（达）、撻（挞）、噠（哒）、躂（跶）、韃（鞑） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文從“辵”從“大”，與《説文》或體形同，“大”是聲符。象一人在大路上行走，含通達、到、至等義（參《漢字圖解》）。甲骨文用作人名。金文從“辵”從“午”從“羊”，構形初義不明。

    《説文》：“達：行不相遇也。从辵羍聲。《詩》曰：‘挑兮達兮。’徒葛切。达，達或从大，或曰迭。”“羍：小羊也。从羊大聲。讀若達。他末切，《廣韻》他達切。（），羍或省。”

    從（从）、慫（怂）、縱（纵）、聳（耸）、蹤（踪） #

    甲骨文金文小篆隶书楷书行书草书

    从：甲金文從二“人”，象兩人前後相從之形，是“從”的初文。本義是跟從。二“人”大多數向左，向右只有一例，古文字左右無別。小篆統一向左。在今天“从”一般被視爲“從”的簡寫，但其實“从”古已有之，“從”不外是“从”加注“辵”部（彳+止）而成者而已，“从”的本義是跟隨。

    甲骨文表示隨从、隨行（董作賓），表示來自，如“从西”、“从東”，即“自西”、“自東”。此外，“从雨”，或解釋爲驟雨（郭沫若），或解釋爲順雨，指風調雨順（于省吾）。

    金文表示跟隨，宰椃角：“王各（格），宰椃从。”金文後加“辵”旁作“從”。又表示隨行之器，任氏从簋：“乍任氏从簋”。青銅器中的“从器”與“行器”的性質與作用相近。又用作人名。

    《説文》：“从，相聽也。从二人。凡从之屬皆从从。”段玉裁注：“按从者今之從字，從行而从廢矣。”

    從：甲金文從“彳”從“从”，金文或從“辵”，“从”是“從”的初文。“从”的本義是跟隨，而“從”則強調相隨於路途之上，故從“辵”。《説文》：“從，隨行也。从辵、从，从亦聲。”

    甲骨文用義不詳，金文表示隨行，弔簋：“弔從王員征楚荊”；又表聽從，中山王鼎：“隹俌（傅）母氏（是）從”；又通作“縱”，表示放縱，洹子孟姜壺：“用從（縱）爾大樂”。

    網（网）、罔、惘、魍 #

    “网”是“網”的甲骨文字形。“網”最初又作“罔”，从网，亡声，后又加“糸”为意符。从“罔”的字只有“網”简化成“网”，其他字仍保留“罔”字形。“罔”和“岡”字形相似，“網”和“綱”的草书字形也相近，详见岡。

    憑（凭，憑藉）、凭（凭欄） #

    凭：《説文》：“凭：依几也。从几从任。《周書》：‘凭玉几。’讀若馮。臣鉉等曰：‘人之依馮，几所勝載，故从任。’皮冰切。”

    徐鉉等把“凭”字爲什麼從几從任解釋爲“几能勝任人的依靠”，這種拆字算命式的釋讀方式肯定不如拿到更古老的字形、更科學的依據再釋讀準確；但是我能搜到的資料裏，沒發現有早於小篆的字形以及今人更科學的解釋，只能分別看看“任”及其幾個部件是什麼意思了。

    任：甲骨文，金文。甲金文從“人”、“壬”聲，甲金文“壬”字作“工”形，金文於豎畫中間增一圓點，後圓點伸展爲一橫。甲骨文“任”相當於金文和後世的“男”，是殷代專門爲王朝服役的職官名，後來演變爲爵名。金文“任”用作姓氏，又作“妊”，任氏簋：“乍任氏从簋”。表示責任、委任之任戰國金文寫作“賃”。

    壬，甲骨文，金文（商）（西周中期），小篆。干支名，獨體字，構形初義不明。早期甲金文“壬”作“工”形，後豎畫中間添一點，又伸展爲橫畫，而成“壬”字。林義光認爲“工”形即“榺”（〔shèng / ㄕㄥˋ〕，古稱經軸或繞經輥爲榺。是古代織機上捲放經紗的工具。有時也稱作柚，今作軸。）之古文，並云：“機持經者也，象形。”高鴻縉從林説。一説“壬”爲兩刃之斧（吳其昌），與“辛”義近。

    辛，甲骨文，金文。“辛”象一個鑿形刀具（詹鄞鑫），用來伐木、施刑等。如“亲”字上從“辛”下從“木”，象用刀鑿（“辛”）砍樹（“木”），以獲取柴薪，是“薪”的初文，“辛”亦標聲。郭沫若認爲“辛”是施黥刑的工具，“童”、“妾”等字從之。“辛”用作偏旁，又增繁筆畫，寫作“丵”，如“鑿”字從“丵”，表示鑿子。從“辛”（丵）取義的字皆與刀鑿、刑罪等義有關；背負刑罰，則心情悲痛，故“辛”又有辛苦、辛酸之義；而味道辛辣，會使人流淚或鼻水，甚至涕泗俱流，而痛苦時也會流淚，所以辛辣一義用表示痛苦、艱辛的“辛”字來表示。

    《王力古漢語字典》：“凭”“憑”本同一字。後來有了分工。“凭”字多指具體的行爲（倚靠）；“憑”字兼有抽象的意義（憑藉）。“凭”字古又讀去聲，必須是指具體的行爲，而且不能寫作“憑”。例如“小睡凭藤輪”不能寫成“小睡憑藤輪”。“憑”字不讀去聲。”

    臺灣《教育部異體字字典》：“凭”爲“憑”之異體。段注本《説文解字·几部》：“凭，依几也。從任几。《周書》曰：‘凭玉几。’讀若馮。”段注：“《顧命》文，今《尚書》作憑，衞包所改俗字也。”《正字通·几部》：“凭，蒲明切，音平。《説文》：‘依几也。’又倚也。《山海經》：‘西王母梯几而戴勝。’註：‘梯几、凭几也。’今借用馮，俗作、憑。”《經典文字辨證書·几部》：“凭正，憑俗。”《古今正俗字詁》：“凭，依几也。從任几，讀若馮。《周書》：‘凭玉几。’今行本作憑爲衞包所改俗字。”按凭爲憑之本字，惟《常用國字標準字體表》定憑爲標準字體，以凭爲或體，故今據之定作“憑”之異體。

    气、氣（气）、愾（忾）、汽、餼（饩） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文從三橫，象天上的雲氣。“气”是“氣”的古字，本義是雲氣。《説文》：“气，雲气也。象形。凡气之屬皆从气。”今天“氣”的簡化字作“气”，是採用了古字。

    因爲字形和“三”相近，所以後來把上畫和下畫寫成彎曲之形，以作區別（春秋金文第二例）。

    甲骨文多用爲副詞，表示終究、最終、終竟（沈培），《合集》12532正：“王占曰：‘疑茲气雨。’之日允雨。”意謂王對着卜兆作出預測：“這個卜兆（顯示）終究會下雨。”這一天果然下了雨。又用作介詞，表示至、到，《屯南》2366：“气日酒”，“日”、“酒”都是祭名，意謂到日祭的時候就進行酒祭。

    金文表示乞求，洹子孟姜壺：“用气（乞）嘉命”，表示因而祈求美好的命令。鄀公緘鼎：“用气（乞）沬（彌）壽，萬年無彊（疆）”，意謂因而祈求壽終，得享天年，萬年無限期。又讀爲“訖”，表示終止，天亡簋：“丕克气（訖）衣（殷）王祀”，即終止殷王的祭祀，亦即商朝滅亡。《尚書·西伯戡黎》：“天既訖我殷命。”

    戰國竹簡讀作“汽”，表示幾乎、將要，《上博竹書三·周易》簡44：“气（汽）至，亦母（毋）繘井，羸丌（其）缾，凶”，王弼本《周易·井卦》作“汽至，亦未繘井，羸其瓶，凶。”意謂汲水井繩將要出井口而尚未提出井口，就把汲瓶傾覆墜毀，那是很凶險的（陳鼓應）。又讀作“迄”，表示至於、達到，《清華簡一·皇門》簡2：“气（迄）又（有）寶（孚）”，《逸周書·皇門》作“訖亦有孚”。大意是説取信於民。

    “气”後假借來表示乞求、祈求，表示這個意思的“气”後來省作“乞”。段玉裁《説文解字注》：“气、氣古今字。自以氣爲雲气字，乃又作餼爲廩氣字矣。气本雲气。引伸爲凡气之偁。”“借爲气假於人之气，又省作乞。”

    氣：異體亦作“餼”、“䊠”，本義爲贈送給人的糧食，後以“餼”字表此專義，而“氣”表示空氣、雲氣、氣流、氣息等。

    《説文》：“氣，饋客芻米也。从米气聲。《春秋傳》曰：‘齊人來氣諸矦。’䊠，氣或从旣。餼，氣或从食。”王鳴盛《蛾術編》：“‘气’字隸變，以‘氣’代‘气’……‘气’廢而不用”。

    隊（队）、墜（坠） #

    甲骨文金文小篆隶书楷书 墜行书 隊草书 墜

    甲骨文從阜，從倒人或子。

    金文從阜，㒸聲。小篆同。本義是墜落，後多寫作“墜”。《説文》：“隊，從高隊也。从，㒸聲。”段玉裁注：“‘隊’‘墜’正俗字。古書多作‘隊’，今則‘墜’行而‘隊’廢矣。”

    “隊”表示本義的例子見傳世古書，如《左傳·莊公八年》：“公懼，隊于車。”《荀子·天論》：“星隊、木鳴，國人皆恐。”“星隊”指隕星下墜。

    殺（杀）、剎（刹）、鎩（铩，鎩羽而歸） #

    > 注意殺字有點，電腦字庫有的有點有的沒有。

    本文所用的宋體在殺字中無點，在剎字中有點。可能是大陸的標準沒有點，臺灣的標準有點，但不知道爲什麼殺字從大陸標準，剎字從臺灣標準。

    甲骨文金文小篆隶书楷书行书草书

    由於甲骨文字形確實沒有殳旁，第三例又類似“杀”，暫把殺的簡化方式定爲古字形。《漢語多功能字庫》對於“殺”字的解釋實在沒看懂，謹把原文列上。

    甲骨文有從“虫”從“攴”的字形（不知道是不是這個字，上面的應該就是後文所説倒寫的虫形。），陳劍認爲是“殺”字的前身。金文的“殺”就是在[虫攴]的簡體上加上意符“人”，字形參見攸比鼎（摹本）和比簋蓋。這種字形的人旁寫在左下角，與上面倒寫的“虫”旁和小點結合，就變成好像一個獨體的披頭散髮的人形之類，像《説文》殺字古文“”的右邊。“殺”的本義是殺戮。（陳劍）一説甲骨文的[虫攴]字是“殺”的初文，本義是擊打。（劉雲）

    金文表示殺戮。如攸比鼎：“？弗具付匕（比），其且（沮）射（厭）分田邑，則殺。”意思是如敢不把比所應得的都付給他，而沮止破壞分田邑之事的，就應被殺。（裘錫圭）

    戰國楚系文字有“殺”的異構，與《説文》“殺”的古文“”（）相近，參見郭店簡《唐虞之道》、《語叢一》和《語叢三》各篇的字形。（陳偉）

    古書中“殺”表示殺戮的例子如《左傳·桓公二年》：“二年春，宋督攻孔氏，殺孔父而取其妻。”《國語·魯語》：“臣違君命者，亦不可不殺也。”

    “殺”也表示衰敗、凋零。如《儀禮·士冠禮》：“以官爵人，德之殺也。”《呂氏春秋·長利》：“吾先君周公封於魯，無山林谿谷之險，諸侯四面以達，是故地日削，子孫彌殺。”高誘注：“殺，衰也。”又表示草木枯萎。如《呂氏春秋·應同》：“及禹之時，天先見草木秋冬不殺，禹曰‘木氣勝’。”

    “殺”還表示交戰、戰鬥。如《三國演義》第1回：“殺到天明，張梁、張寶引敗殘軍士，奪路而走。”《喻世明言·葛令公生遣弄珠兒》：“李存璋唐朝名將，這一陣，殺得大敗虧輸，望風而遁，棄下器械馬匹，不計其數。”

    “殺”可作副詞，表示程度深。如唐代杜甫《清明》之二：“春水春來洞庭闊，白蘋愁殺白頭翁。”元代關漢卿《竇娥冤》第二折：“兀的不痛殺我也！”

    確（确）、确 #

    确〔què / ㄑㄩㄝˋ〕：《説文》：“礊（〔kè / ㄎㄜˋ〕，堅硬）石也。从石，角聲。臣鉉等曰：今俗作確，非是。胡角切。，确或从㱿。”又指多石而貧瘠的土地。又有角逐、爭取的意思。《漢書·卷五四·李廣傳》：“李廣材氣，天下亡雙，自負其能，數與虜确。”《新唐書·卷一九二·忠義傳中·張巡傳》：“君世受官，食天子粟，奈何從賊，關弓與我确？”

    確：從“石”，“隺”聲，表示堅固、堅硬。引申爲真確。

    《王力古漢語字典》：二字溪匣旁紐，藥屋旁轉。在堅确這個意義上二字同源。《莊子·應帝王》：“確乎能其事者而已矣。”釋文：“李云：堅貌。”《説文》：“确，礊也。”又：“礊，堅也。”按，《説文》無確字。

    電（电） #

    甲骨文金文小篆隶书楷书行书草书

    金文從“雨”從“申”，“申”亦是聲符。“申”象閃電，是“電”的初文。由於閃電多伴隨雷雨，所以加“雨”爲意符，後來“申”被借用爲地支第九位，故以“電”來表示閃電的本義。

    申：甲金文象閃電的電光閃爍屈折之形，是“電”的初文。由於閃電多伴隨雷雨，故加“雨”爲意符造“電”字。假借爲地支第八位。

    戰國文字兩旁電光的分枝或訛爲兩“口”（與田疇的“疇”字初文“”相混），多見於楚系文字；或訛爲“”，多見於秦系文字。由於閃電放射延伸，故“申”有了延伸、伸展之義，後加“人”分化出“伸”表示人體的伸展、屈伸；古人認爲閃電是由神所主宰的，故後來加“示”以表示神靈的“神”（白川靜）。

    雨：甲骨文，商代金文。甲骨文上部象天或象雲，下部象雨點。金文字形有所變化，較貼近篆書的寫法，但依然象下雨之形。本義是雨水。

    棄（弃） #

    > 上部與㐬、育等字同，大陸、臺灣標準不同。
    甲骨文金文小篆隶书楷书行书草书

    甲骨文第一例從倒“子”從“廾”（），第二例從“子”從“其”（），第三例從“子”從“其”從“廾”，象雙手持簸箕把嬰孩扔掉，本義是抛棄嬰兒，引申爲抛棄。傳説中，周代的始祖后稷，出生後曾被遺棄，因而名叫“棄”。

    “弃”是《説文》古文。《説文》：“棄：捐也。从廾推棄之，从。，逆子也。臣鉉等曰：‘，他忽切。’詰利切。，古文棄。，籀文棄。”“弃”源於甲骨文第一例，“棄”源於甲骨文第三例。

    《康熙字典》引《史記·周本紀》：“周后稷名弃，其母姜原欲弃之，因名曰弃。”
