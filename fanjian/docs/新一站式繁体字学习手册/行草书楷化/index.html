    昜: 〔yáng / ㄧㄤˊ〕

    甲骨文金文小篆隶书 陽楷书 陽行书 陽草书 陽

    甲骨文及早期金文从“日”从“丂”，“丂”象树柯、枝柯，是“柯”的初文（屈万里）。金文后来加从“彡”，象太阳的光线。全字象太阳升到树上，阳光从树梢上照射下来，“昜”是“陽”“暘”的初文，本义是阳光。

    盪和蕩的区别：

    盪：① 洗涤器具。引申为一般意义的洗涤。又引申为清除。又引申为动荡。又引申为冲杀。《晋书·刘曜载记·陇上歌》：“丈八蛇矛左右盘，十盪十决无当前。”② （盪盪）空旷的样子。

    蕩〔dàng / ㄉㄤˋ〕：① 动摇，摆动。引申为震动，动摇。② 流，疏通。《周礼·地官·稻人》：“以潴畜水，以防止水，以沟蕩水。”③ 放荡，放纵。④ 荡涤，洗涤，清除。《释名·释言语》：“蕩，盪也，排蕩去秽垢也。”⑤ 毁坏。李白诗：“王城皆荡覆，世路成奔峭。”⑥ 平易，平坦。《诗·齐风·南山》：“鲁道有荡，齐子由归。”《毛传》：“荡，平易也。”⑦ 广大，渺茫。⑧ 浅水湖泊。⑨ 通“簜”。大竹。

    蕩〔tāng / ㄊㄤ〕：水名。通作“湯”。

    蕩〔tàng / ㄊㄤˋ〕：（蒗蕩）叠韵联绵字。古运河名。

    個人觀點：“盪”和“蕩”是本義不同，引申義相交的兩個字，但所有用“盪”的地方基本可以用“蕩”替換。“盪”一般用於強調洗滌義的詞彙。

    名詞的用字應該有例外，有些專有名詞或術語似乎應該保持歷史上的原字或規定的用字，不能混用。比如，有種農具叫“耘盪”，磨墨用的瓦盆“墨盪子”，除非有先例，否則不應該亂改。臺灣的字典中，“盪”字條下才有的“腦震盪”“振盪器”等這種近現代才有的詞彙的用字，爲了標準化，似乎也不能混用。

    雖然理論上“蕩”可以包含“盪”，但畢竟兩字字源不同，新出現的詞彙可以只用其中一字而不用另一字，但沒必要從漢字系統中隱去一字。

    曾、層（层） #

    曾，象古代一种蒸煮器具，“其器下体承水，上体盛饭，中设一箅”（参朱芳圃），箅即蒸锅中的竹屉。“曾”是“甑”的初文（朱芳圃、徐中舒、戴家祥）。后来被假借为“曾是”或“曾经”的虚词。由於“曾”字久借不還，後世乃增義符“瓦”而作“甑”字以保存、傳達其初義。

    正因为曾象甑中之箅将器隔为两重，故从“曾”之字多有重义（刘恒）。如“層”（《说文》：“重屋也”）、“甑”（甑甗皆为两器相叠）。《诗·周颂·维天之命》：“曾孙笃之”郑玄笺：“曾犹重也。”《尔雅·释亲》：“祖之父曰曾祖王父。”（参杨树达）

    喪（丧） #

    甲骨文金文小篆隶书楷书行书草书

    甲金文从数口，桑声。金文加从“亡”声。因“喪”“桑”同音，故喪字也借桑字表示，为了与桑区别，故加从数口。

    于省吾认为口乃采桑所用之器，商代已有丝织品，故此字甲骨文是采桑之本字。李孝定则认为数口表示喧哗嗷嘈之意。一说古代丧器讲究以桑木制作，桑是办理丧事的象征，其从多口正象众口号啕大哭之形（尹黎云）。

    按“喪”字的构形初义未有满意的说法，金文从“桑”从“亡”从数“口”，因为“桑”形至金文已讹写变形，故另加“亡”为声符，而“亡”又有丧失、亡失之意，使“喪”字字义得以彰显。金文或增“走”为意符。

    《说文》：“亾也。从哭从亾。会意。亾亦声。息郎切。”颜真卿就真把喪写作哭亡（楷书第一例）。篆书哭作，因此喪颇像哭亡。

    金文“圖”从“囗”从“啚”，“啚”是“鄙”之初文，象仓廪设在田野边郊，故有边鄙之意，“囗”象城邑之形，杨树达认为既有城邑，又有边鄙，“圖”之本义当为版图。一说字象以“囗”围绕鄙邑绘画地图之意（何琳仪）。“圖”后泛指一切图籍。

    啚，甲骨文，金文，是“鄙”的初文，甲金文从“囗”（城围之形）从“㐭”，“㐭”象仓禀、米仓，古时谷获于田而藏于廪，仓廪当离边鄙田野不远，故以城邑外之仓廪表示边鄙之意。凡国都必侈，乡邑必陋；故“都”引申有大、盛、美之义；“鄙”引申有小、狭、陋、俚之义（参杨树达）。
