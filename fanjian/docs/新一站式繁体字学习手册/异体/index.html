    异体 #

    加星号的是我认为应该定为异体的。

    詠（※咏） #

    《説文》：“詠：歌也。从言，永聲。爲命切。咏，詠或从口。”

    煙（※烟） #

    從“火”，“垔”聲，本義爲物質燃燒時產生的混合氣狀物。“煙”“烟”二字古通用。《説文》：“（煙），火气也。从火垔聲。烏前切。烟，或从因。，古文。，籒文从宀。”《集韻》：“籀亦作。”《國語·魯語上》：“既其葬也，焚，煙徹于上。”

    金文作，與説文籀文同。

    “垔”戰國金文作。金文出現時間較晩，始見於戰國，從“西”從“土”，“土”或訛變爲“壬”形。構形初義不明。《説文》：“垔，塞也。《尚書》曰：‘鯀垔洪水。’从土，西聲。於眞切。，古文垔。”《説文》以爲從西聲，按“垔”屬影紐真部，“西”屬心紐微部，兩字聲韻相隔，“西”並非聲符。

    雁、鴈、贗（赝，※贋） #

    “雁”楚文字，齊魯文字。從“隹”，從“人”，“厂”聲，本義爲候鳥。候鳥形狀略似鵝，頸和翼較長，足和尾較短，羽毛淡紫褐色。善於游泳和飛行。古文中另有“鴈”字，即是“鵝”。

    《説文》：“鴈：䳘也。从鳥、人，厂聲。臣鉉等曰：‘从人从厂，義無所取，當从雁省聲。’五晏切。”

    “鴈”“雁”兩字常互用於古文，學者多以“雁”爲鴻雁，“鴈”爲家鵝。《説文》：“雁：鳥也。从隹从人，厂聲。讀若鴈。臣鉉等曰：‘雁，知時鳥，大夫以爲摯，昬禮用之，故从人。’五晏切。”

    清段玉裁《説文解字注》“鴈”字下云：“許（慎）意隹部雁爲鴻雁，鳥部鴈爲䳘。……今字雁、鴈不分久矣。”

    “雁”“鴈”二字又借爲“假的，僞造的”之義，此義後作“贗”。“贋”爲晚起字。

    灾（※災） #

    甲骨文從“宀”從“火”，表示火災。小篆字形從“火”，“”聲。籀文字形從“”從“火”，隸書又省去“”旁中間的一橫，寫爲“災”。

    甲骨文有三種字形代表不同的災害，分別是“”（水災）、“灾”（火災）和“”（兵災）。（商承祚）“灾”從“宀”從“火”，表示火災。或有“火”形在“宀”形之上者，疑亦爲“灾”。（參《新甲骨文編》、《甲骨文字編》）甲骨文用爲地名，如《合集》7968反：“自灾。”

    小篆字形從“火”，“”聲。《説文》：“烖，天火曰烖。从火，聲。灾，或从宀、火。，古文从才。，籀文从。”或體字形（灾）與甲骨文構形相近，段注：“火起於下，焚其上也。”籀文字形從“”從“火”，隸書又省去“”旁中間的一橫，寫爲“災”。

    簡帛文字曾以“”表示災害，從“示”，“才”聲。如《上博竹書五·三德》簡2：“天乃[降止]（降）（災）。”《馬王堆帛書·經法·名理》第74行：“是必爲福，非必爲（災）。”

    古文字表示災害的“災”的異體很多，後來在傳世文獻中大多寫作“災”。“災”既表示火災，如《春秋·桓公十四年》：“秋八月壬申，御廩災。”又泛指各種自然或人爲的禍害，如《國語·周語》：“古者，天災降戾，於是乎量資幣，權輕重，以振救民。”韋昭注：“災，謂水旱蝗螟之屬。”《莊子·庚桑楚》：“若是者，禍亦不至，福亦不來。禍福無有，惡有人災也！”

    涌（※湧）、踊（※踴） #

    “湧”從“水”，“勇”聲。“湧”同“涌”，“勇”亦從“甬”聲。《説文》作“涌”。《集韻·腫韻》：“涌，或作湧。”本義是水向上冒。

    “涌”從“水”，“甬”聲。本義是水向上冒。《説文》：“涌：滕也。从水，甬聲。一曰：涌水，在楚國。余隴切。”

    “甬”金文，小篆。金文“甬”象鐘形，上象鐘懸，下象鐘體。（楊樹達）後來“甬”的下部演變爲從“用”以表聲。“甬”或加“金”旁作“銿”，表示用青銅製造。“銿”是《説文》“鐘”的或體。傳世文獻以“甬”表示鐘柄，所以徐灝、高鴻縉認爲“甬”象鐘柄的象形。

    “踴”從“足”，“勇”聲。“踴”同“踊”，“勇”亦從“甬”聲。《説文》作“踊”，後又寫作“踴”。《集韻·腫韻》：“踊，或從勇。”

    “踊”甲骨文，金文。甲骨文從兩“止”從“”。“”象高出地面之形，金文可單獨成字，隸定爲“冂”，是“堂”的初文。（唐蘭）甲骨文兩“止”，一在“”之下，一在“”之上，會踊上高堂之意，是登上高處的“踊”的表意初文。（陳劍）金文兩“止”中間從“同”，陳劍認爲可看作全字的聲符，從“”變爲“同”存在有意的“變形音化”的因素。小篆從“足”，“甬”聲。後又作“踴”。

    臺灣這四個字都爲正體，互通。

    椁（※槨） #

    “槨”同“椁”，《説文》：“，葬有木也。从木聲。”古代棺材有兩重，內曰棺，外曰椁。

    《説文》：“：度也，民所度居也。从回，象城之重，兩亭相對也。或但从囗（音韋）。凡之屬皆从。古博切。”商金文，西周早期金文。不是“享”，隸書將其類化爲“享”。

    “享”甲骨文，金文，小篆。“享”古作“亯”，甲金文象斜檐小屋建於高臺之上，象宗廟之形（吳大澂）。引申爲祭享、享獻、享用、享受等。一説象人穴居之形（徐中舒）。林義光等以爲亯、皀同字，象簋形，爲烹飪器皿。按就字形觀之，“亯”與“京”、“高”形近，應爲建築物之形。金文下部或訛變作曰、田，爲小篆所本。篆文又作“享”。宗廟爲祭祀之處，引伸有祭亯之意。古時宴饗賓客，典籍多作“饗”，饗祀先人，多作“享”。郭沫若曰：“凡生人言饗，死人言亯、言格”。按甲金文“鄉”混用不分，惟金文“亯”多用作祭祀。

    大陸以“椁”爲正體，臺灣以“槨”爲正體。

    冪（※幂） #

    《説文》：“：覆也。从一下垂也。凡冖之屬，皆从冖。臣鉉等曰：‘今俗作冪，同。莫狄切。”

    臺灣《異體字字典》：“幂”爲“冪”“幎”之異體。《説文解字·巾部》作“”，云：“幔也，從巾冥聲。”今考《廣韻·入聲·錫韻》云：“幎，亦作幂。”《正字通·巾部》亦云：“幎，亦作幂。”《經典文字辨證書·巾部》則云：“幎，正；幂，通；，別。”按“幎”、“幂”二字部件全同，特“巾”字位置移異耳。今以字書有據，故可收“幂”字爲“幎”字之異體。

    《康熙字典》“冪”“幂”並列，皆爲“冖”之或體。

    我理解“冖”是覆蓋的意思，“幎”是幔帳，是兩個不同的字。“冪”是“冖”的俗字，就當覆蓋講；“幂”是“幎”的異體，本來是幔帳，後來不知道怎麼的，又成了“冪”的異體。

    靦腆（腼腆） #

    《王力古漢語字典》：

    靦〔tiǎn / ㄊㄧㄢˇ〕：① 人面之貌。《詩·小雅·何人斯》：“爲鬼爲蜮，則不可得，有靦面目，視人罔極。”毛傳：“靦，姡也。”馬瑞辰《通釋》：“靦與姡皆人面之貌。”② 羞慚。《宋書·徐湛之傳》：“靦然視息，忍此餘生。”又爲不知羞慚。唐·慧琳《一切經音義》卷八八：“靦，考聲云：‘謂不知慙也。’”《文選》南朝·梁·丘遲《與陳伯之書》：“將軍獨靦顏惜命，驅馳氈裘之長，寧不哀哉！”

    靦〔miǎn / ㄇㄧㄢˇ〕：【靦覥】疊韻聯綿字。害羞貌。元·王實甫《西厢记》一本一折：“未語人先靦覥，櫻桃紅綻，玉粳白露，半晌恰方言。”或作“靦腆”。

    臺灣《異體字字典》：

    “靦腆”：心中羞澀，難爲情而表現於顏面。元·王實甫《西廂記·第一本·第一折》：“未語人前先靦腆（與《王力古漢語字典》不同）。”亦作“腼腆”、“愐㥏”、“緬覥”、“緬靦”、“靦靦”。

    《王力古漢語字典》無“腼”字，亦無“腼腆”；臺灣《異體字字典》有“腼腆”，但無“腼”字。估計是疊韻聯綿字偏旁趨同，而出現了“腼”字。

    强（※強） #

    “強”金文，小篆。沒找到對金文字形的解釋。戰國秦簡、秦印文字從“弓”從“口”從“虫”，《説文》認爲本義是“蚚”，即蛀米蟲，廣東稱爲米牛、穀牛，紹興稱爲米象（參季旭昇）。漢帛書表示強壯、有力，《馬王堆·老子甲本卷後古佚書》：“強禦者，勇力者”，《詩·大雅·烝民》：“不侮矜寡，不畏彊禦。”《説文》：“強：蚚也。从虫，弘聲。徐鍇曰：‘弘與強聲不相近，秦刻石文从口，疑从籀文省。’巨良切。，籀文強，从䖵从彊。”

    衮（※袞）、滚（※滾） #

    “滾”從“水”，“袞”聲，表示大水涌流，引申爲流泄，流淌。唐杜甫《登高》：“無邊落木蕭蕭下，不盡長江滾滾來。”

    “袞”金文，小篆。金文從“衣”，“公”聲，曶壺蓋聲符訛變爲“谷”。金文表示天子、諸侯、臣子穿著的有龍形刺繡的禮服。吳方彝蓋：“玄袞衣”，表示黑色的禮服。

    “袞”異體作“衮”，《説文》：“衮，天子享先王，卷龍繡於下幅，一龍蟠阿上鄉。从衣公聲。”

    《康熙字典》：“袞，《篇海》與衮同。按《説文》，衮，从衣公聲。《廣韻》《集韻》並作衮。《干祿字書》以袞爲正，以衮爲通。今依《説文》。”

    汙（※污） #

    汙，小篆，從“水”，“于”聲。異體作“污”，又作“汚”。本義是停積不流的濁水、小水坑。

    于，甲骨文，金文，小篆。由於小篆的寫法，導致“于”作部件時有“于”“亏”“亐”三種主要寫法。詳見于。

    臺灣以“汙”爲正，大陸以“污”爲正。

    鼇（鳌，※鰲）、鼈（鳖，※鱉） #

    “鰲”《説文》作“鼇”：“海大鼈也。从黽，敖聲。五牢切。”

    《增廣字學舉隅·卷二·正譌》：“鼇：鰲非。黿、鼈、鼉字並從黽。鼇音敖，大鼈也。”蓋魚、黽皆爲水生動物，故偏旁互作耳。《增廣字學舉隅》雖云“非”，然正可以見其時有此一異體。《正字通》：“鰲，俗鼇字”

    “鱉”《説文》作“鼈”。

    臺灣定“鼇”“鰲”皆爲正體，但定“鱉”正“鼈”異。

    豊、禮（※礼）、醴 #

    甲骨文金文小篆隶书楷书行书草书

    “豊”甲骨文從“玨”從“壴”。“豊”是“禮”的初文，林澐認爲古代進行禮儀活動時常用“玨”（玉）和“壴”（鼓），故以此表示禮的概念。《論語·陽貨》：“禮云禮云，玉帛云乎哉？樂云樂云，鍾鼓云乎哉？”

    甲骨文讀作“醴”，表示甜酒。如《合集》32536：“叀（惠）舊豊用。”“舊豊”即“舊醴”，表示陳酒。

    金文表示禮，指宗廟郊祀之禮，如天亡簋：“王又（有）大豊（禮）。”又讀爲“醴”，表示醴酒，如長甶盉：“穆王鄉（饗）豊（醴）。”又用作人名。

    小篆下部訛爲“豆”形，《説文》誤以爲“豊”字從“豆”，不確。又“豊”和“豐”在漢魏碑銘中有相混的情況，林澐認爲是字形相近所致，從出土的先秦古文字資料來看，“豊”和“豐”是起源不同的兩個字。參見豐。

    “禮”從“示”，“豊”聲。“禮”是“豊”的後起分化字，表示祭神致福。“豊”和“豐”由於字形相近，在漢魏碑銘中有相混的情況，“禮”也有寫爲從“豐”的例子。現代簡化字襲用了《説文》古文“禮”的寫法，寫作“礼”。這種字形在漢魏碑銘中也有使用。《説文》：“禮，履也，所以事神致福也。从示从豊，豊亦聲。（礼），古文禮。”

    古文“礼”應該是從示，乙聲。

    諠（※喧） #

    諠也作喧。“喧”同“吅”，表示惊呼。《说文》：“吅，驚嘑也。从二口。”《集韵·元韵》：“吅，《説文》‘驚嘑也’，亦作喧。”

    諠亦通諼。

    釁（※衅） #

    小篆作。《説文解字·舁部》：“釁，血祭也。象祭竈也。從爨省、從酉；酉，所以祭也。從分，分亦聲。”《玉篇·血部·衅字》收“衅”字，注云：“虛鎮切。牲血塗器祭。亦作釁。”《集韻·去聲·稕韻》“釁衅”下云：“許慎切。《説文》血祭也。……或作衅。”《字彙·血部》“衅”下云：“與釁同。《樂記》車用衅而藏之府庫，而弗復用。”《漢語大字典·血部》“衅”下云：“同釁。……按：今爲釁的簡化字。”按：《樂記》釁作“衅”，後代字書都言“釁”“衅”音義相同。

    由于找不到“衅”的古字形，也没找到任何书解释为什么从“半”，暂定为异体吧。

    噩、鰐（鳄，※鱷） #

    噩，金文。金文從◎從四“口”，聞一多、李孝定、高明俱以爲“噩”、“喪”古同字，至金文始分別爲兩字，“喪”字加“亡”聲，而不加“亡”聲的是“噩”字。參見“喪”。

    金文用作國名，典籍作“鄂”，禹鼎：“敦伐噩，休隻（獲）厥君馭方”。又用作人名，弔噩父簋：“弔（叔）噩父”。

    “噩”、“咢”古同字，《史記·楚世家》：“熊咢”，《索隱》曰：“咢亦作噩”，《十二諸侯年表》作“鄂”。《爾雅·釋樂》的《經典釋文》：“咢，本作鄂”，故容庚云：“咢即噩，又孳乳爲鄂也。”

    後世“噩”又作“愕”，表示驚恐，如噩夢、噩耗，《玉篇》：“噩，驚也。”（參漢語大字典）

    《説文》無“噩”，有“咢”（説文隸書爲㖾）。

    鄂，戰國楚金文。金文從“噩”從“邑”，“噩”是聲符。“鄂”是地名，在今湖北省鄂州市，因此“鄂”後來成爲湖北的代稱。“噩”、“咢”是一字的不同隸定。“鄂”地戰國時屬於楚國。

    《説文》：“鄂，江夏縣。从邑，（㖾）聲。五各切。”“（㖾），譁訟也。从吅，屰聲。五各切。”

    鱷，《説文》作：“似蜥蜴，長一丈，水潛，吞人即浮，出日南。从虫，屰聲。吾各切。”段玉裁注云：“俗作、鰐、鱷。”《玉篇·魚部》：“鱷，五各切，魚名。鰐，同上。”

    《龍龕手鏡·魚部》收錄“鱷”字，注云：“五各反。魚名。”其下又收“鰐”，以爲“鱷”之今字。《康熙字典·魚部》則作“鰐”。《角川漢和辞典·魚部》、《中文大辭典·魚部》、《漢語大字典·魚部》、《中華字海·魚部》等都作“鰐”形。

    今港臺以“鱷”爲正體，大陸以“鰐”爲正體。“噩”“咢”是不同的隷變形式。
