    合并字 #

    合并字的辨析主要参照《王力古汉语字典》，也参照台湾的《国语辞典》《异体字字典》。这里面有一部分字只是常见异体，或部分义项上的常见异体，或未严格分化的字。这部分字里有些我觉得完全可以当作异体处理，不需要作区分的，我在前面加了星号。

    金文从“”从“戍”，“”象丝线，指细微的事情，“戍”象人持戈戍守，表示防范细微的事情，有防微杜渐之意（关子尹）。戴侗认为表示戍守者当察于物色几微之间，故可引申为几微、危殆。

    “几”小篆作，象踞几之形，只用作几案、茶几等义。“机”是其异体，也是一种树名。

    “幾”表示幾個（几个）、幾乎等义。京畿的畿未简化，就是以幾为声。

    为了不漏掉一个本来从“幾”但简化为从“几”的字，也为了不使本来从“几”的字被误认为是简化字，我从这个网站（千千秀字）查了所有以“几”和“幾”为部件的字，逐一筛查。

    《汉字全息资源应用系统》按部件查询这个功能，但它的检索方式要么给出的字太少，要么太多，超出GBK的范围。超出GBK范围的字一般没有类推简化字。总体来说，我感觉“千千字秀”的结果应该是在GBK范围内得出来的，足够涵盖所有可能被类推的字。还有“云”和“雲”、“台”和“臺”、“只”和“戠”这三组不太好确定繁简字的地方，我也用这个办法查了一遍。

    甲骨文从“之”从“宀”。“之”是声符，“宀”表示与建筑物有关。本义是四方高而平的建筑物。战国文字从“室”，“之”声。从“宀”与从“室”意义相近。后来“臺”上半的声符“之”字下又增从“口”，逐渐演变为现在的“臺”。《说文》以为“臺”字“从高省”，是把“室”的上半跟“口”形结合成一部分而言。（姚萱）

    檯，专指桌子或像桌子具有平台之器物。注意，“講臺”是指教室前面高起来的平台（一般教室前面都有高出地面的平台），不是指讲台前面的桌子。

    “于”字的构形有两种说法。第一种认为甲骨文象管状乐器之形，中间有“于”作为声符，是“竽”的初文（郭沫若、李孝定、裘锡圭）。后来为书写简便，省作“于”，以代表整个乐器，后由吹奏乐器引申表示气之舒吁。第二种说法认为“于”是“吁”的初文，象管状乐器中的一口气之形，本义是吁气。还有第三种说法，孙雍长认为竽中的“于”象竽的表面竹管的纹路、纹理之形。

    上述第一种以“于”原指“竽”的说法出自郭沫若、李孝定、裘锡圭等名家。甲骨文文例表示“竽”这种乐器。又用作介词，引介时间、地点、人物等。

    金文“雩”字的或体“”从“羽”从“于”，用作音阶名，相当于后世的“羽”。“于”在“雩”和“”字中除了标声之外，还充当意符，意义与乐器相关，可作为“于”的初文是乐器一说的佐证。参见“雩”。

    至于第二种以“于”为“吁”的初文的说法实可远溯于许慎，近年则由关子尹提出。查“于”字于《说文》中写作“亏”，其解释为：“亏，于也。象气之舒亏，从丂从一，一者其气平之也。”查“于”字的结构，实与“乎”“兮”“号”等字一样，都和吾人的各种呼吸送气活动有关。“于”很可能是“吁”的初文，其原义亦应为“叹息”意义的“吁”。西周金文中“于”字有用作动词，解作往、去，如献簋：“献伯于遘王”，令簋：“隹王于伐楚伯，在炎。”与《诗经》用法相同，如《周南·桃夭》“之子于归”，《毛传》：“于，往也。”；“于”更多用作介词，如麦方尊：“作册麦易金于辟侯。”多友鼎：“追于京师”，《诗·采蘩》：“于涧之中”。也可解作超过、超越，如《荀子·劝学》中有“冰，水为之，而寒于水”，取意极为抽象。 “于”又因声音关系与“於”通用，参见“於”。

    关于“于”字的古文字字形。金文和甲骨文除了“从丂从一”这较简单的写法外，也保存了较复杂的另一种写法。针对甲骨文“于”字这一奇特的写法，特别是右侧看似重叠的一组线条，徐中舒认为“于”本指“大圆规”。这一说法，我们认为并无说服力。从甲骨及金文较复杂的字形看，我们认为“于”很可能是从作为一种管乐器的“竽”经两阶段的省略而得的省文。换言之，于字的形构除了一般的如许慎说的从丂从一外，也可以是从竽省。古文字“竽”其实像一件管状的器物，由于“竽”是用来吹奏的，甲骨文中解作竽管的“竽”字中间有“于”，正表示有气吹于其中。而较复杂的“于”字之所谓“从竽省”，其实是省剩竽管的右壁和竽管中的那一口气（即“吁”的本字）。所以“竽”字的存在，和古文“竽”中之有“于”，可说是“于”之原义为吁气的“吁”的极有力的侧证。

    按两种说法中，第一种说法是古文字学界的主流意见，但本网站认为第二说亦有充分的认知理据，姑备二说。

    甲骨文只有“于”而无“於”字，“於”是“烏”的分化字，后因音近借用为“于”。

    金文“於”字始见西周晚期，其字象鸟形，为“烏”之省文，《説文》：“烏，孝鳥也，象形。孔子曰：‘鳥，盱呼也，取其助气，故以爲烏呼。”又曰：“於，象古文烏省。”高鸿缙认为“烏”象鸟侧立之形，而“於”象鸟飞之貌。噩君啓舟节“於”字（金文最后一例）与《说文》“烏”古文相近。“於”与“烏”同，用作叹词，如西周早期效尊：“烏（嗚）虖（呼）！”战国中山王鼎：“烏虖！語不廢哉！”

    《王力古漢語字典》：“于、於不同音（于，羽俱切；於，央居切），但意義相通。在表示至于某地時，多用‘于’；在表示甚於時（《荀子·勸學》：‘冰，水爲之，而寒於水。’）和表示被動時（《孟子·滕文公》上：‘勞力者治於人。’），多用‘於’。但並不嚴格。”

    现在“于”主要用于姓氏；“於”用作介词。春秋以前多用“于”为介词，之后多用“於”为介词。

    於就是淤泥、瘀血里那个於。

    辟（鞭辟入裏）、闢（辟，開闢，開疆闢土） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文从“䇂”从“卩”，或加从“口”，金文从“䇂”从“卩”从◎，◎是“璧”的初文，象圆形玉璧形，是“辟”的声符。“䇂”象镰刀类的工具，是“乂”的初文（裘锡圭），有治理之义，疑“辟”的本义为治理。季旭昇认为“辟”字从“辛”（刑具），象依法施刑于人，本义为刑法。按“䇂”象镰刀类的工具，是“乂”的初文（裘锡圭），有治理之义，疑“辟”的本义为治理。

    《王力古漢語字典》大義如下：

    辟〔bì / ㄅㄧˋ〕：① 法、法度。② 治理。③ 罪行。④ 天子、诸侯国君的通称。⑤ 征召。⑥ 躲避。后作避。⑦ 腿瘸。后作躄。

    辟〔pì / ㄆㄧˋ〕：① 打开。后作闢。引申为开辟，开拓。② 捶胸。通擗。③ 偏颇，不实在。后作僻。④ 偏僻，幽僻。后作僻。⑤ 比如。通譬。

    辟〔mǐ / ㄇㄧˇ〕：停止，平息。通弭。

    辟〔bò / ㄅㄛˋ〕：分开。通擘。

    辟〔pī / ㄆㄧ〕：织物的花边。通纰。

    個人理解：“辟”讀〔pì / ㄆㄧˋ〕時的所有義項，後世都有專字表示；躲避義亦有“避”字表示。“辟”在表達這些意義時，本來就是假借，義項之間關係不大，後世發展出專字是自然的，也是好事。現在，在表達這些意義時，一般不會再用“辟”字，但在一些詞語中仍保留“辟”字的假借用法，最常見的是躲避義的假借，例如，辟邪（避邪）、辟穀（避穀，不吃五穀，道家修煉的方法）。

    另，按臺灣國語辭典，“闢”有駁斥、摒除義，“辟”通“闢”亦有此義。因此，“辟謠”同“闢謠”、“辟除”同“闢除”。

    “鞭辟（〔bì / ㄅㄧˋ〕）入裏”的“辟”用其本義。

    金文從“門”從，象兩手相背，象以雙手推開門，本義是推開、開啟。金文大盂鼎字形與《説文》古文相同。《説文》：“闢，開也。从門，辟聲。房益切。，《虞書》曰：‘闢四門。’从門、从。”段玉裁注：“引申爲凡開袥之偁（稱）。古多假借‘辟’字。……者，今之攀字。引也。今俗語以手開門曰攀開，讀如班。”

    後期金文或不從相背的雙手，而從相向的雙手。戰國晚期大武戈“闢”字加從◎，◎是“璧”的初文，作爲“闢”的聲符。小篆改爲從“門”，“辟”聲。

    金文用作本義，表示開闢、開拓。中山王鼎：“闢啟封疆，方數百里，剌（列）城數十，克敵大邦。”

    “闢”的本義是推開、開啟，開墾土地也叫“闢”，《史記·田敬仲完世家》：“田野闢，人民給，官無留事，東方以寧。”由推開門扇引申，有排除之義，《荀子·解蔽》：“是以闢耳目之欲，可謂能自彊矣。”，楊倞注：“闢，屏除也。”把門推開，兩扇户即分離，故“闢”又有分離義，《易·繫辭上》：“其動也闢”，焦循《章句》：“闢，猶分也。”北周衛元嵩《元包經·運蓍》：“混茫既判，天地闢矣。”

    尸（尸位素餐）、屍（尸，屍體） #

    甲骨文金文小篆隶书楷书行书草书

    甲金文象人屈膝坐下之形。本義是坐。《論語·鄉黨》：“寢不尸”，即睡覺時不坐。

    “尸”的字形本義有三説，一説象人高坐（李孝定），如坐在几上；一説象人屈膝坐在地上（沈培）；一説象人蹲踞（徐中舒）。按首二説較佳。

    古人的“坐”相當於今天的跪坐，而比起跪坐、蹲着，屁股着地的坐姿較爲舒適。查古代祭祀時，生者因不忍見至親之不在，乃以活人“尸”代表死者接受祭禮，甚至享用祭品，如《儀禮·士虞禮》：“尸飯”。“尸”受到尊敬和優待，故讓尸在祭祀時屈膝坐在几上或地上。因爲只有尸才能這樣坐，所以以坐姿來指稱“尸”的職稱。

    “尸”除了要坐着，還要站立。古書有“坐尸”、“尸坐”，與“立尸”相對，《禮記·禮器》：“周坐尸”、“夏立尸”、“殷坐尸”。《白虎通》：“座尸而食之”，表示使尸有座位而被喂食。“座”是坐具，似可側證“尸”象人坐在凳上。

    “尸”又可指代表死者的“神主牌”，古代“載尸以行”就是指持着神主牌巡行（見《史記·龜策列傳》），後世仍由嫡親持“神主牌”進行祭祀。戰國竹簡即加從“示”爲意符，“示”象神主牌位，注意此字形未有用作神主牌。

    甲骨文通假作“夷”，方國名，《合集》6459：“正（征）尸（夷）”，表示攻打夷方。又可借指來自尸方即夷方的人牲。《合集》828：“用十尸（夷）于丁，卯一牛。”表示用十個來自夷方的人牲來祭祖先丁，宰殺一頭牛。金文也通讀作“夷”，表示蠻夷。㝬鐘：“南尸（夷）、東尸（夷）具視，廿又六邦。”

    金文“尸”又可當動詞用，表示扮演“尸”（代表死者的活人）的角色，夷伯簋：“尸伯尸于西宮”，表示尸伯在西宮擔任尸。《詩·召南·采藏》：“誰其尸之，有齊季女”。引申之，可解作“執掌”或“佔據”名位，如“尸職”、“尸祿”、“尸位素餐”等。

    戰國竹簡表示代表死者受祭的活人，《上博竹書六·平王問鄭壽》簡1：“尸廟”，即“有尸之廟”，強調是有“尸”參加祭祀的廟（沈培）。

    又古代除可由“尸祝”主祭外，也有由死者嫡親充當“尸”的習俗，《禮記·曲禮》有：“孫可以爲王父尸”之説，及《儀禮·特牲禮》注：“尸，所祭者之孫也。祖之尸則主人乃宗子”等，都是明證。

    今人一般以爲“尸”乃“屍”的簡體字，可以補充的是，“尸”本爲繁體字，屍體的“屍”古借“尸”來表示，後來才加“死”爲聲符分化出“屍”專門表示遺體。

    《説文》：“尸，陳也。象臥之形。凡尸之屬皆从尸。”

    尸：① 代表死者受祭的活人。② 通屍。尸体。③ 主持。《诗·召南·采蘋》：“谁其尸之？有齐季女。”引申为居其位而无所事，如尸位素餐。

    《説文》：“屍， 終主。从尸从死”，終是死者，終主即死者，指人死後的“屍體”，後來省文爲“尸”，並可通用。查“尸”字本一卷曲的人形，本指古代喪儀上的代表“死者”接受祭祀的祝巫。唯當“尸”字解作“祭祀之尸”時，則只能用“尸”字。根據《正字通》，“屍”字比“尸”字更古，但此説不可信，因甲骨文及金文均未見“屍”字，而《睡虎地秦簡·封診式》中的屍字亦只借“死”字表逹。

    屍：死人的身体。《国语·齐语》：“杀而以其屍授之。”

    借（租借、借問）、藉（借，憑藉） #

    借：① 借入，借出。② 假使，假令。《诗·大雅·抑》：“借曰未知，亦既抱子。”备考义：助。《汉书·朱云传》：“少时通轻侠，借客报仇。”

    藉〔jiè / ㄐㄧㄝˋ〕：① 荐，草垫，祭祀时用以陈列礼品。引申为铺，垫。柳宗元《捕蛇者说》：“往往而死者相藉也。”② 坐卧在某物之上。③ 践踏，凌辱。④ 凭借，依靠。⑤ 连词。表假设。《史记·陈涉世家》：“失期当斩。藉第令毋斩，而戍死者固十六、七。”裴骃《史记集解》引服虔曰：“藉，假也。”

    藉〔jí / ㄐㄧˊ〕：① 杂乱，狼藉。② 古代田制，借用民力耕治公田。③ 进贡，奉献。④ 绳，繫。⑤ 顾。李山甫落花诗：“落拓东风不藉春，吹开吹谢两何因？”

    作姓氏時讀〔piáo / ㄆㄧㄠˊ〕。用於“朴刀”時，大陸讀〔pō / ㄆㄛ〕，臺灣讀〔pú / ㄆㄨˊ〕。朴刀又名播刀，俗稱雙手帶，出現於宋代。是一種木柄上安有長而寬的鋼刀的兵器，介於大刀與單刀之間。刀身窄長、柄短、可隨身攜帶。《水滸傳》第二回：“只帶三五個做伴，將了朴刀，各跨口腰刀，不騎鞍馬，步行下山，逕來到史家莊上。”

    〔pǔ / ㄆㄨˇ〕：① 未加工成器的木料。《説文》：“樸，木素也。”《老子》第二十八章：“樸散則爲器。”《論衡·量知》：“無刀斧之斷者謂之樸。”引申爲凡未未曾加工、訓練過的都稱樸。《左傳》哀公十二年：“素車樸馬。”《荀子·臣道》：“若御樸馬。”楊倞注：“樸馬，未調習之馬。”② 本質，本性。《老子》第十九章：“見素抱樸，少私寡欲。”《呂氏春秋·論人》：“故知知一，則復歸於樸。”高誘注：“樸，本也。”③ 淳樸，質樸。《莊子·漁夫》：“甚矣由之難化也，湛於禮義有間矣，而樸鄙之心至今未去。”《呂氏春秋·上農》：“民農則樸，農則易用。”

    〔pú / ㄆㄨˊ〕：① 叢生的樹木。《詩·大雅·棫樸》：“芃芃棫樸，薪之槱之。”朱熹《集傳》：“樸，叢生也。”【樸樕】叢生的小樹。《詩·召南·野有死麕》：“林有樸樕，野有死鹿。”《毛傳》：“樸樕，小木也。”

    【備考】【樸屬】依附，附著。《周禮·冬官·考工記·序》：“凡察車之道，欲其樸屬而微至。”鄭玄注：“樸屬，猶附著堅固貌也。”

    致、緻（致，精緻、細緻、標緻） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文从“丮”从“至”，金文从“人”从“至”，“至”亦是声符，表示人把东西送到或送来之意，本义是致送。甲骨文“丮”象人跪坐而伸出双手，意会手里拿着东西，把它送来、致送。金文“人”下或加从“止”（象脚），强调专程到来之意。在古汉语里，“致”既可以表示送给、给与，也可以表示来到、送来。所以在汉简里，送东西和领东西用的文书都可以叫做“致”（裘锡圭）。

    “致”字本义是送到、致送，引申为精致，即达到精细的程度，由于丝线幼细，故后加意符“糸”，造“緻”字专门表示精细之义。从到来义又引申为导致、引致。

    从我能搜到的书法字中感觉自明以后多写作攵旁。

    凶（吉凶）、兇（凶，兇手、兇惡） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文从“凵”从乂形，“凵”是“坎”的象形初文，表示地面下陷，乂形疑象掉下来的沙、石、泥、草等。本义是地面的坑洞、陷阱，引申为灾祸、不吉利。

    “兇”是“凶”的同源分化字，它分担了“凶”字的人事部分。“兇”不用于灾祸、年成、寿命、征兆之类，只用于形容凶恶的人或个性，以及凶恶的事。如“兇狠狠”“兇首”“兇器”等。“凶”“兇”分化之后，由于人们表示凶恶义时，仍常常用“凶”，使“凶”“兇”在凶恶义上重叠，故简化字将二字归并为“凶”（王凤阳）。

    “兇”甲骨文，亦有战国楚文字，不是后起分化字。

    按《王力古汉语字典》：

    凶：① 不吉利，与“吉”相对。② 恶。

    兇：① 骚动恐惧。② 兇恶。

    凶是吉凶的凶，兇是兇惧的兇，二字不同义。“吉凶”不能写成“吉兇”，“兇惧”可以写成“凶惧”。在凶恶的意义上二字通用。“群兇”也可以写成“群凶”。但也要依照习惯，例如杀人“兇手”不能写成“凶手”，“行兇”不能写成“行凶”。

    發（发，發現）、髮（发，頭髮） #

    参见發（发现）、髮（头发）、撥（拨）、潑（泼）、廢（废）。

    蔑（蔑視）、衊（蔑，污衊） #

    衊：形声字。污血。又用作动词，谓以血或秽物涂染。《新唐书·列女传·崔绘妻卢》：“是夕，出自窦，粪秽衊面。”引申为污蔑，诋毁，以不实之词诬毁他人。《汉书·文三王传》：“污衊宗室，以内乱之恶披布宣扬于天下。”这个意义后来多写作“蔑”。

    甲骨文金文小篆隶书楷书行书草书

    甲骨文从“戈”从“”，象以武器斫人之胫。“”一般认为是“眉”的繁体，下作人形，或作女形。甲骨文和早期金文的“戈”形都穿过人形的下半部分，可知“蔑”本指斩人的下半身，与表示以戈斩首的“伐”有别（唐兰、谢明文）。

    蔑的本义疑为伐灭。（朱骏声、季旭昇）如《国语·周语》：“今将大泯其宗祊，而蔑杀其民人，宜吾不敢服也。”韦昭注：“蔑，犹灭也。”“宗祊”即宗庙。

    丑（丑角、小丑）、醜（丑，醜陋、小醜） #

    丑，甲骨文，金文。象手有指甲，本义是指爪。“丑”“爪”金文同形。“丑”后假借为干支字，而本义废。

    醜，甲骨文，金文。甲骨文从“鬼”，“酉”声，本义不详，疑从鬼面引申为美醜之醜。商代金文或于“酉”下加“其”形，象盛“酉”之物（余永梁）。西周晚期又于“醜”字叠加“犬”旁，其义不详。《説文》：“醜，可惡也。从鬼，酉聲。”段玉裁注：“非真鬼也。以可惡，故从鬼。”

    “小丑”“小醜”“丑角”“醜角”辨析：

    “丑角”是中國戲劇的一種程式化的角色行當，一般扮演插科打諢比較滑稽的角色。文丑以做工爲主，武丑以武打爲主。和淨角中的三塊瓦之間的區別不明顯。可見，丑角是滑稽角色，並不一定是外表醜陋或內心醜陋的角色。

    “小丑”是喜劇演員之一，常見於舞臺、馬戲團、綜藝節目、嘉年華、兒童節目及生日會等。他們表演時，多穿著特大鞋子及奇裝異服，其臉部也經塗裝，尤以鼻子部分較爲突出。這些古怪、搞笑的裝束給觀眾帶來歡笑。通常小丑都會以自身出糗來娛樂觀眾。

    可以説，丑角可以包含醜，但他不僅僅是醜。

    按臺灣“教育部重編國語辭典”，“小丑”有兩個意思：

    ① 戲劇中表演滑稽角色的人。清·李斗《揚州畫舫錄·卷五·新城北錄下》：“廣東劉八工文詞，好馳馬，因赴京兆試，流落京腔，成小丑絕技。”

    ② 後比喻舉止不莊重、善於逗趣的人。如：“他是辦公室裏的小丑，一天到晚鬧笑話！”

    “小醜”一詞解釋爲：稱小人、盜匪等。《國語·周語中》：“怨在小醜，猶不可堪，而況在侈卿乎？其何以待之。”《紅樓夢》第六三回：“所以凡歷朝中跳梁猖獗之小醜，到了如今，竟不用一干一戈，皆天使其拱手俛頭，緣遠來降。”

    “丑角”一詞除表示戲曲角色外，還比喻品德低劣，無操守的人。這也是有時候“小丑”和“小醜”混用不分的原因。

    “醜角”一詞在這個所謂的“教育部重編國語辭典”中沒有搜到。

    看來，在表示戲劇角色或與這種角色相似的人時，應該用“丑”字；在表示小人、盜匪，或類似之醜惡之人時，應該用“醜”字。“小丑”更多當中性詞用，“小醜”更多當貶義詞用。有時，“小丑”和“小醜”也通用，我在“教育部重編國語辭典”中搜到“小醜跳梁”和“跳梁小丑”兩個詞（意義是一樣的），小丑/醜在這裏指成不了氣候又愛興風作浪的人。“小醜”一詞在《國語》中就有，但戲曲中的“丑角”一詞是怎麼來的沒有搜到了。

    只、隻、祇 #

    只〔zhǐ / ㄓˇ〕，楚文字，小篆。① 从“口”，象征“气向下引之形”，用作语气词。《説文》：“語巳詞也。从口，象气下引之形。凡只之屬皆从只。”《詩·鄘風·柏舟》：“母也天只，不諒人只！”② 用作副词，表示仅仅（后起义）。唐韓愈《過始興江口感懷》：“憶作兒童隨伯氏，南來今只一身存。”③ 也用作助词，类同“着”。元楊顯之《瀟湘雨》第二摺：“這襆頭呵，除下來與你戴只；這羅襴呵，脫下來與你穿只。”

    隻〔zhī / ㄓ〕，参见隻。① 鸟一隻。引申为一个。又为单。② 量词。

    隻〔shuāng / ㄕㄨㄤ〕：通“雙”。一对。《史记·龟策列传》：“玉椟隻鸡，出于昆山。”《穆天子传》卷二：“于是载玉万隻。”郭璞注：“双玉为瑴。”陈逢衡补注：“萬隻之隻，即古省雙字。玉必以雙獻。”

    祇〔qí / ㄑㄧˊ〕：① 地神。② 大。③ 通“疧”。病。

    祇〔zhǐ / ㄓˇ〕，旧读〔zhī / ㄓ〕：副词，只，仅仅。字亦作“祗”“秖”“秪”“衹”。

    只、祇等皆借用作副词仅仅。祇字借用作此义可能更早。《诗·小雅·无将大车》：“无将大车，祇自尘兮。”大陆以“祇”为仅仅义的繁体。个人以为唐宋以后和小说中可能多用“只”，现港台以“只”为仅仅义。

    匯（汇，匯合，※滙）、彙（汇，彙編、詞彙） #

    匯：从匚，淮声。表示水流会合，引申表示聚集、聚合。到了近代还表示把款项从甲地寄到乙地。

    彙：从㣇，胃省声。本义是刺猬。也表示类别。【品彙】即品种和类别。刺猬身上有大量毛刺，故又以“彙”“彙集”比喻众多同类的聚集（参王凤阳）。亦有写作“猬集”。

    字彙，彙編当用彙。其义源自众多同类的聚集。

    面（面目）、麵（面，麵粉，※麪） #

    面：甲骨文，象脸的轮廓，中间是眼睛。本义是脸。

    麵：面粉。《说文》作麪。从麥，丏声。麵为后起字。现大陆和港台皆以麵为正字。

    系、係、繫（下面是糸不是系） #

    行书草书
    系〔xì / ㄒㄧˋ〕：① 连接。《说文》：“系，繫也。”② 世系；前后相承的关系。③ 辞赋末尾贯通全文的言词。④ 用单股丝合成的丝绳。⑤ 悬。曹植《辅臣论》：“群言系于口。”

    係〔xì / ㄒㄧˋ〕：束缚，捆绑。引申为係属。

    繫〔xì / ㄒㄧˋ〕：① 连接，联结。② 悬挂。③ 拘囚。《汉书·江充传》：“收繫其父兄。”④ 拘束，羁绊。⑤ 带子，绳子。⑥ 《易》之“繫辭”的简称。

    繫〔jì / ㄐㄧˋ〕：① 粗劣的絮。② 打结（晚起义）。繫帶子。

    係、系同音，可以通用。系、繫也可以通用。但“世系”不作“世係”或“世繫”，“繫辭”不作“係辭”或“系辭”。

    个人以为现在繁体字使用地区情况是这样的：

    “系”用於抽象概念，比如：體系，系統，系列，大學裏的科系。

    “係”表示“是”“關連”“牽涉”（皆引申自係屬義），例如：確係、實係、關係、干係、係數（代數中與未知數相乘的數字），主要表示和某事有牽連的意思，強調係屬關係而不是單純的聯繫。

    “繫”用於聯繫、維繫、繫念、繫（jì）鞋帶，解鈴還須繫（xì）鈴人這種由捆綁義引申出的意思。繫的口語詞一般讀（jì），文言詞讀（xì）。“係”和“繫”的區別在於前者強調係屬關係，後者只是單純的相連、牽挂等義。 | |

    鞏（巩，鞏固、姓鞏）、巩、築（筑，建築）、筑 #

    鞏：從“革”，“巩”聲，本義爲用熟牛皮束物。

    巩〔gǒng / ㄍㄨㄥˇ〕：從丮，工聲。《説文》：“擁也，袌（抱）也。一作㧬〔gǒng / ㄍㄨㄥˇ〕。”

    築：從“木”，“筑”聲，本義是舂搗泥土用的杵。

    筑〔zhú / ㄓㄨˊ〕：《説文》：“以竹曲五弦之樂也。从竹从巩。巩，持之也。竹亦聲。張六切。”① 古弦乐器名。② 贵阳的别称。③ 水名。

    夸（夸父）、誇（夸，誇獎、浮誇、誇張） #

    甲骨文金文小篆行书草书

    夸〔kuā / ㄎㄨㄚ〕：甲金文從“大”，“于”聲，六國文字“于”旁演變爲“丂”，或加飾筆作“亏”，故篆文從“大”，“亏”聲，本義爲樂聲張大（谷衍奎），後引申爲奢侈，今亦作“誇”的簡體字。《說文》：“夸，奢也。從大，亏聲。”段玉裁注：“夸，奢也。奢者，張也。從大，亏聲。”

    《王力古漢語字典》① 奢侈。② 自大，炫耀。

    誇〔kuā / ㄎㄨㄚ〕：《說文》：“譀也。从言，夸聲。苦瓜切。”《王力古漢語字典》① 誇大，誇耀。引申爲誇獎，讚美。② 通“姱”。美好。

    两字同音，皆苦瓜切。个人认为，“誇”是用言语夸奖或用不实之词赞美，后面跟被誇的对象；“夸”是自大，炫耀自己，后面跟的是你要在谁面前炫耀。在古文中能看出夸和誇的差异。《史记·韩长孺列传》：“驅馳國中，以夸諸侯。”并不是誇讚诸侯，而是在诸侯面前炫耀自己。还有《列子·杨朱》：“而欲尊禮義以夸人，矯情性以招名。”并不是誇别人，而是炫耀于人。所以，“夸”后面跟的是炫耀于的对象，炫耀的是自己；而“誇”后面跟的是要夸奖的对象。“夸”就是表示自大、炫耀于别人。“誇”是赞美、奉承或用虛妄的语言誇别人或自己。现代汉语中已经不再用“夸”这个词，转而用“炫耀”或其他的表达方式，所以现代汉语出现“夸”的词都是“誇”。

    划（划船、划算）、劃（划，劃痕、劃分、計劃） #

    划〔huá / ㄏㄨㄚˊ〕：后起字。以桨拨水。划拳、划算、划得来都是这个字。

    劃〔huà / ㄏㄨㄚˋ〕：① 从刀，从畫，畫亦声，本义为用尖利物将东西割开。也读作〔huá / ㄏㄨㄚˊ〕。② 忽然（后起义）。杜甫《苦雨奉寄陇西公》：“劃見公子面，超然懽笑同。”③ 通“畫”。劃分，謀劃，策劃等义源自畫。《说文》劃在畫部，爲畫之古文。徐鍇繫傳刀部亦有劃字，云“錐刀也”。《廣韻》麥韻：“劃，錐刀刻。”劃是畫的分別字。

    曲（曲折、歌曲）、麯（曲，酒麯，※麴） #

    曲：甲骨文，金文。象器物弯曲之形，本义是弯曲。由于音乐旋律抑扬婉转，曲折回旋，故引申为乐曲的曲。学者多从《说文》释此字象弯曲的器具，商承祚认为其形如矩，高鸿缙则以为象盛饭之器具，白川静以为象细竹、蔓草编成的筐笼，一说以为养蚕的器具，亦有以为字与音乐有关。另，季旭昇认为此字似曲畸之农田，可备一说。

    麯〔qū / ㄑㄩ〕：酒母。后起字。《说文》作麴。大陆以麯为正字，台湾以麴为正字。

    吃（口吃）、喫（喫飯） #

    吃〔chī / ㄔ〕：① 口吃。② 同“喫”。

    喫〔chī / ㄔ〕：后起字。食，饮。喫飯，喫茶。今写作“吃”。

    台湾吃和喫并为正字，可能是按习惯决定哪里用哪个。

    朱（朱紅）、硃（硃砂、硃批） #

    朱：甲骨文，金文。是“株”的初文，本义是露出地面的树根、树干或树桩。假借为赤色、红色。甲金文从“木”从一圆点，显示树干、树根之所在，构形取义与“本”“末”同理。

    硃：朱的后起分化字。① 朱砂。② 用朱墨写的文字。

    伙（伙食）、夥（伙，夥伴，※㚌） #

    夥，《说文》：“齐谓多为㚌。从多，果声。乎果切。”个人认为：后来也俗写作“伙”。比如：大夥也作大伙，夥計也作伙計。

    伙，《说文》无。《王力古汉语字典》无。按《汉典》：“同‘火’。古兵制十人为火，同火的人互称火伴。俗作伙伴，同伴。”台湾教育部的字典认为“伙”通“夥”，但对“伙”的释义中有伙食一义，而对“夥”的释义中没有伙食一义。《百度百科》对伙食的解释是“多指集体所办的饭食”，个人认为其实也可以作“夥食”，但可能伙食一词出现后一直被写作伙食而非夥食吧。另外，如果《汉典》的释义正确，那么“伙”可能就是后世发展出来的一个与“夥”不同源但在部分词汇中可互换使用的字吧。

    后（皇后）、後（后，後來、後宮） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文从“幺”从“夂”，为“後”之初文，“夂”象足形，“幺”象系绳，字会足被绳绑，落后不能向前之意（林义光）。后叠加形符“彳”，以标出行走的意思。

    一说以为“幺”为声符（刘钊、何琳仪）。金文与战国竹简“彳”或改从“辵”，与《说文》古文同。

    甲骨文金文小篆隶书楷书行书草书

    “后”字金文始见于春秋晚期，本义是君主。字形向右的为“后”，向左的是“司”字，这种分别是在金文中才出现，而甲骨文向左向右无别，故唐兰认为“后”“司”古本同形，李孝定亦认为“后”“司”一字。此外，甲骨文“后”字又以“毓”表示（王国维）。

    司，甲骨文，金文（商·司母戊方鼎）。其字形释义未有定论。疑本义是主管、主宰。刘兴隆释字象用手罩口上大声说话，会为主管之事，本义为主管。从甲骨字来看，“司”的上部是否从“又”（手）存疑，“又”多为三指。徐中舒释“司”的上部象倒置之柶（古代舀取食物的礼器），柶所以取食。以倒柶覆于口上会意为进食。氏族社会中食物为共同分配，负责食物分配的人亦称为司。季旭昇则认为象权杖之形，故有职司、主宰之义。

    个人认为“后”通“後”是个别情况，使用并不广泛，比如：《礼记·大学》：“知止而后有定。”

    最近發現“後”“后”合併也會出現詞語理解上的混亂。我問過一些人，他們認爲後宮是皇后住的地方所以〔hòu / ㄏㄡˋ〕是皇后的后。但其實後宮就是皇帝家的後院，所以是“後”不是“后”。

    沖（冲，沖盈、一飛沖天、怒氣沖沖）、衝（冲，衝動、要衝、衝擊） #

    沖，甲骨文，金文。从“水”从“中”，象水流撞击“中”字之形，“中”亦是声符，本义为水冲击。“沖”字甲骨文疑用作人名。《合集》32906：“令（命）沖”，似是命令沖的意思。“沖”字金文见于沖子鼎：“沖子之行贞（鼎）。”陈梦家、林洁明认为当读作“童”，“沖子”即“童子”。一说以为“沖”是姓氏。传世文献也有“沖”读作“童”的例子，如《尚书·盘庚下》：“肆予沖人。”孔安国传：“沖，童。童人，谦也。”说明“沖人”是古人谦称自己幼小无知。又如《尚书·召诰》：“今沖子嗣，则无遗寿耈。”孔安国传：“童子，言成王少，嗣位治政。”“沖子”即“童子”，为成王自称。全句指现在自己年少嗣位，不会违背老人长辈的教诲。金文又以“沈子”表示“沖子”。表示幼小的意义后来也写作“种”。“冲”是俗字。

    沖：① 水涌動。沖洗，沖咖啡，沖破堤防。② 空虛。沖盈。本作“盅”。《説文》：“盅，器虛也。从皿，中聲。《老子》曰：‘道盅而用之。’”段玉裁注：“盅虛字今作沖，沖行而盅廢矣。”③ 幼小。④ 直飛而上。《韓非子·喻老》：“雖無飛，飛必沖天。”《會真詩三十韻》：“海闊誠難度，天高不易沖。”⑤ 猛烈之狀。怒氣沖沖。⑥ 破除厄運。沖喜。⑦ 相剋、相忌。對沖，相沖。⑧ 謙虛，淡泊。沖和。

    令狐沖和任盈盈的名字即取義自“沖盈”，沖爲空，盈爲滿。

    “沖”之所以有一個特立獨行的“直飛而上”的意義，而且“沖天”這種明顯往天上衝的意思居然不用“衝”，是因爲此處的“沖”是同音假借，應當是“翀”字（專指鳥向上直飛）。“翀”字爲後世所創，韓非子那時候還沒有這個字，因此在表達“鳥向上直飛”這個意義時，用“沖”字做了代替，後世爲了區分，另創“翀”字。三國魏·杜摯《贈毌丘荊州詩》：“鵠飛舉萬里，一飛翀昊蒼。”其實，“沖天”這個詞本來表達的是鳥騰空而起的這種氣勢，一種專有說法，不是一般意義的“衝天”，如果你就是要表達一般意義的衝天，自然可以用“衝”。

    “沖破堤防”等表達衝擊的詞，其實可以用“衝”，而且古代也是這麼用的。“衝”就是表示一切衝擊，不分什麼水不水的，真要這麼區分，就有強行區分的嫌疑了，且古代“沖”和“衝”音不同。

    本人能力有限，對“沖”和“衝”的區別只能做這麼多，我自己也有很多疑問找不到答案，有興趣的自己研究吧。

    衝，小篆。從“行”，“重”聲，本義爲通道。“衝”小篆從童，《說文》：“通道也。从行，童聲。《春秋傳》曰：及衝，以戈擊之。”“重”“童”形近音近，從童之字也多寫作從重，例如，種和穜、鍾和鐘等。

    衝：① 交通要道。要衝。《春秋傳》曰：及衝，以戈擊之。② 衝擊，碰撞。《莊子·秋水》：“梁麗可以衝城，而不可以窒穴，言殊器也”又如，怒髮衝冠。引申爲頂撞，觸犯。衝撞之處，還請見諒。③ 向，對著。今讀爲〔chòng / ㄔㄨㄥˋ〕。門窗衝南。④ 穿，刺。戰國·宋玉《風賦》：“及其將衰也，被麗披離，衝孔動楗。”《漢書·賈誼傳》：“白公爲亂，非欲取國代主也，發憤快志，剡手以衝仇人之胸，固爲俱靡而已。”⑤ 衝蓯（〔chǒng cōng / ㄔㄨㄥˇ ㄘㄨㄥ〕）。疊韻聯綿字。相入貌。即互相糾結的樣子。⑥ 激烈，强烈，無理。讀〔chòng / ㄔㄨㄥˋ〕。煙味太衝。做事太衝。説話太衝。

    澱（淀，沉澱）、淀（白洋淀） #

    淀：《说文》无。浅水湖泊。如，白洋淀。

    澱：① 河底淤泥。② 浅水湖泊。澱与淀古字通。③ 蓝色染料称蓝澱或蓝靛。

    並（并，並且、並列）、併（并，合併、吞併）、并（并州） #

    “並”用於並排、並列，表示兩物並立（未混合在一起），也用於並且、並非等副詞；“併”用於合併、歸併、併入，表示兩物融合在一起；“并”只用於地名——并州。

    以下是《漢語多功能字庫》對字源的解釋（節選）：

    並：甲骨文，小篆。甲金文從二“大”或“立”，象兩人並排正面站立在地上之形。本義是並立。丁山認爲古“並”“并”同字，按甲金文“並”“并”字形、用法皆有別，後世典籍混用無別。甲骨文用作人名、地名，又表示併也（徐中舒）。金文表示並排、共同，中山王方壺：“將與吾君竝（並）立於世”。又用作人名、族氏名。“並”小篆作“竝”，與甲金文同形。

    并：甲骨文，金文，小篆。甲骨文表示兩個人前後相從或相連，本義是和合。甲骨文用作地名、方國名。金文表示兼併、吞併，後增人旁累增“併”字，中山王鼎：“吳人并（併）雩（越），雩（越）人修教備恁（任），五年覆吳。”商鞅量：“廿六年，皇帝盡并（併）兼天下。”小篆作幷。與甲金文比，將兩橫畫分開是不對的。

    併：與并同字。

    以下是《王力古漢語字典》的釋義（節選）：

    並：副詞。一齊，一併。《莊子·齊物論》：“昔者十日並出。”引申爲動詞。平列。《荀子·儒效》：“俄而並於堯禹。”《解嘲》：“僕誠不能與此數子並，故默然獨守吾太玄。”

    并〔bìng / ㄅㄧㄥˋ〕：兼併。

    并〔bīng / ㄅㄧㄥ〕：① 古州名。② 并閭即椶櫚（棕櫚）。

    併：並排。引申爲合併。又爲副詞，表示一齊。

    併、并、並三字同義不同音。併，卑政切；并，卑盈切。二字古韻耕部。並，蒲迥切，古韻在陽部。“並”《説文》作“竝”。《説文》：“併，竝也。”“竝，併也。”二字互訓。《説文》：“并，相從也。”朱駿聲曰：“按，合一爲并，對峙爲併。”“兼并”作“并”；“併肩”作“併”。副詞一般寫作“並”，如“並包”“並出”。但這些區別並不嚴格。

    以上《王力古漢語字典》的辨析，反而把人看糊塗了，因爲他沒有從字源上解釋字義，只是解釋了這幾個字在古籍中的使用情況。正如《漢語多功能字庫》所説，“並”“併”後世典籍混用。如今繁體字使用地區又將這幾個字規範了一下，就是我開頭第一段所説那樣。這個規範也比較符合字理。因此，現在“兼并”只作“兼併”，“併肩”只作“並肩”。

    異（异，差異）、异 #

    甲骨文金文小篆隶书楷书行书草书

    注意楷行书的笔顺，不是先写个“甲”再写个“共”，而是先写“旦”，再写两短竖，再写长横，再写中竖。至于现在的规范体，就写上田下共。

    甲骨文“異”象頭部殊異的人形，雙手上舉。周忠兵認爲雙手上舉可能是起區別意義，而非過去一般所説的表示手持物戴於頭頂，與“戴”的甲骨文字形有別。“異”的本義應該是特異、別異。

    甲骨文的“異”主要用於田獵卜辭，用作虛詞，表示可能、將要。如《合集》30757：“王異其田，亡災。”意謂王將要田獵，不會有災禍。“異”疑亦表示不同、別異。如《屯南》610：“弜（勿）異酒。”“酒”是祭名，“勿異酒”大概是不要把某種儀式跟酒祭分開舉行。（裘錫圭）

    金文“異”可作虛詞，訓爲乃。如召圜器：“召弗敢忘王休，異用乍（作）□宮旅彝。”（陳英傑）又表示小心、恭敬。如梁其鐘：“不（丕）顯皇且（祖）考，穆穆異異。”“異”還假借爲“翼”，表示輔翼、輔佐。大盂鼎：“古（故）天異（翼）臨子，灋（法）保先王，□有四方。”

    戰國秦漢文獻“異”表示不同、殊異。如《郭店簡·性自命出》簡9：“四海之內其眚（性）弌（一）也，其甬（用）心各異，教史（使）肰（然）也。”《睡虎地秦簡·法律答問》簡172：“同母異父相與奸，可（何）論？棄市。”

    甲骨文金文小篆隶书楷书行书草书

    《説文》：“异，舉也。从廾，㠯聲。虞書曰：‘岳曰异哉。’羊吏切。”

    《書·堯典》：“岳曰：异哉！試可乃已。”僞孔傳：“异，已；已，退也。”與《説文》異。蔡沈傳：“异義未詳，疑是已廢而後强舉之之意。”

    《列子·湯問》：“重囚纍梏，何以异哉？”注：“异，異也。古字。”

    奸（奸犯）、姦（奸，姦詐、通姦，△姧） #

    奸:〔gān / ㄍㄢ〕，奸犯

    奸 〔gān / ㄍㄢ〕：金文，從“女”，“干”聲，本義疑是干犯、冒犯。《左傳》襄公十四年：“君制其國，臣敢奸之。”杜預注：“奸，猶犯也。”《漢書·溝洫志》：“使神人各得其所，而不相奸。”引申爲干求，求仕。《莊子·天運》：“以奸者七十二君。”《史記·穰侯列傳》：“以此時奸説秦昭王。”

    奸 〔jiān / ㄐㄧㄢ〕：通“姦”。奸邪，奸詐（後起義）。《三國志通俗演義》卷一：“亂世之奸雄也。”《三國志·魏書·武帝紀》注作“亂世之姦雄”。“奸”又音“姦”，因爲同音之故，“奸”常常通假爲“姦”，戰國竹簡表示通姦。

    姦 〔jiān / ㄐㄧㄢ〕：金文，從三“女”，疑本義是私通。不守婦道，紅杏出牆稱爲“通姦”。古人對女人有偏見，認爲女人是禍水，近女色即損德，所以“姦”字從“女”。從“女”而有貶義的字還有“妄”“婪”“嬾”（懶）“奴”“怒”等。一説“姦”是會意兼形聲字，“奻”是“姦”的聲符（徐在國）。① 邪惡。② 私通。③ 通“奸”。干犯。《韓非子·定法》：“賞存乎慎法，而罰加乎姦令者也。”《淮南子·主術》：“各守其職，不得相姦。”

    奸、姦二字在上古時代音義皆不同。奸，古寒切，是干犯的意思；姦，古顏切，是姦詐的意思。後來由於“姦”又作“姧”，所以又演變爲“奸”，與干犯的“奸”變爲同字。（《辭源》引《管子·重令》“奸邪得行”爲例是誤引。查《管子》原文作“姦”，不作“奸”。）

    姧，俗“姦”字。姧蘭：犯禁走私。《史記·匈奴列傳》：“漢使馬邑下入聶翁壹姧蘭出物與匈奴交。”《集解》：“姧音干。干蘭，犯禁私出物也。”《漢書·匈奴傳》作“閒闌”。

    参照台湾的字典，我觉得在台湾除了个别词汇不互通外（可能只是按照习惯写法），基本都可互通。例如，漢奸不作漢姦，強姦不作強奸，但是像通姦、姦雄、姦詐、甚至姦淫都可写作“奸”。

    塗（涂，塗抹）、涂（涂姓） #

    涂、塗、途三字在“道路”一义上通用。“途”字古代多写作“涂”或“塗”。“泥塗”一义只作“塗”；“涂抹”一义可作“涂”，也可作“塗”。简化字将“塗”“涂”合并为“涂”。现在“道路”一义只作“途”。

    塗〔tú / ㄊㄨˊ〕：《说文》无，《新附》有之。① 泥。《书·禹贡》：“厥土惟塗泥。”又《仲虺之诰》：“有夏昏德，民墮塗炭。”引申为污染。《庄子·让王》：“其竝乎周以塗吾身也，不如避之以絜吾行。”② 粉饰，以颜色、油漆塗在房屋器物的表面。③ 道路。④ 用笔抹去（后起义）。

    塗〔dù / ㄉㄨˋ〕：① 镀。《汉书·外戚传·赵皇后》：“切皆銅沓，黃金塗。”颜师古注：“塗，以金塗铜上也。”

    涂〔tú / ㄊㄨˊ〕：① 水名。在今四川省境内。② 道路。后写作“塗”“途”。③ 十二月称涂。《义疏》：“涂者古本作荼，荼亦舒也，言阳虽微气渐舒也。”

    涂〔chú / ㄔㄨˊ〕：① 水名。即今滁河。② 通“除”。扫除。《荀子·礼论》：“卜筮視日，齋戒脩涂。”

    途：《说文》无。甲骨文，金文。甲骨文从余从止，金文从“辵”，“余”声，“辵”从“彳”从“止”，“彳”象道路，“止”象人的脚，本义是道途，路途。

    涂：甲骨文，金文。本义为水名。篆文与甲金文同形，本义为水名。

    塗未找到小篆以前文字。本义是泥。

    壇（坛，花壇、祭壇）、罈（坛，罈罈罐罐，※罎） #

    壇，从“土”，“亶”声。本义是高台，古代多用于祭祀、会盟、拜将等。

    亶，从“㐭”，“旦”声。本义是谷物多，在古书中多用引申义，如：忠厚、诚信，亦作副词表示诚然。

    罈，从缶，覃声。本义是口小肚大的瓦制容器。查《汉字全息资源应用系统》，未在字书中找到“罈”字，但在宋、元、明史及清史稿中有其用例；“罎”字可在《正字通》《康熙字典》中找到，但未有古書用例。大陆似定“罎”为正字，台湾定“罈”为正字。二字《说文》皆无。

    覃，甲骨文，金文。金文從“鹵”從“”作“”，“”象盛載東西的罈子，“鹵”字象鹽（盐）粒，全字象罈中裝載鹽之形。早期金文所從“鹵”字省去象鹽之小點，是“鹵”的省形。季旭昇根據毛公鼎“簟”字從“鹵”從“皿”，認爲“覃”是“”“鹽”的初文，可備一説。金文用法有二：一，用作族氏名；二，表示擴大，晉姜鼎：“每（敏）揚厥光剌（烈），虔不彖（惰），魯覃京師，乂我萬民”，“魯覃京師”指使其京師得以擴展（郭沫若）。

    从对“覃”字的解释看，似乎应为“罈”的初文。

    二字区别字自不必说了，注意“花壇”和“花罈”是两个不同的词，后者指一种杂技表演，以耍小罈来作各种姿势。

    壩（坝、水壩）、垻（坝、沙坪垻） #

    壩，从土，霸声。本义是拦截水流的建筑物。

    垻，蜀人謂平川爲垻。平川，即平坦之地勢。見《玉篇·土部》。宋·黃庭堅《謝楊履道送銀茄四首》詩之一：“君家水茄白銀色，殊勝垻裏紫彭亨”。（〔bèi / ㄅㄟˋ〕）坡也。見《類篇·土部》。

    “垻”本正字，《龍龕手鑑·土部》云：“垻，必嫁反。蜀人胃（謂）平川爲平垻。”《廣韻·去聲·禡韻》：“垻，蜀人謂平川爲垻。”《集韻》、《四聲篇海》、《字彙》皆同。《正字通·土部》更詳云：“垻，必架切。音霸。障水堰。今謂堰埭曰垻。俗作壩。”然則，“垻”本正字，有“障水堰”“平川”之義。“壩”反是其俗字。然至後世，於“障水堰”一義，“壩”行而“垻”廢，反以“壩”爲正，“垻”成異體。《重訂直音篇·土部》亦云：“壩：音霸。障水堰。垻：同上。”“壩”正，“垻”異，明矣。另“垻”字於“蜀人謂平川爲垻”一義，仍當獨立爲正字。《漢語大字典·土部》云：“我國西南地區稱平地或平原爲‘垻’。”音〔bà / ㄅㄚˋ〕。

    霸，甲骨文，金文。甲骨文从“革”从“月”，会月光变化之意（季旭升、陈秉新）。金文加从“雨”为声符。“霸”与“白”音近，一说“霸”表示白色的月光（金国泰、张世超）。金文或不从“月”而从“帛”声，参师父鼎，而吴王光鉴“霸”字直接写作“白”，可支持第二种说法。

    折: 〔zhé / ㄓㄜˊ〕、〔shé / ㄕㄜˊ〕

    折，甲骨文，金文。甲金文从“斤”从断木，会以斧头砍断树木之意。本义是折断。① 折断。② 曲折。③ 屈。面折廷争。④ 挫败。⑤ 毁掉。⑥ 亏损。⑦ 折狱，即断狱、判案。⑧ 抵当，折合。⑨ 封土为祭处。⑩ 古代葬具。形如床，无足。（11）元杂剧结构的一个段落（晚起义）。

    摺，从手，習声。《说文》以为本义是败、毁。① 摺疊。② 曲折。③ 读〔lā / ㄌㄚ〕，同“拉”。

    克（克勤克儉，千克）、剋（剋星，相生相剋） #

    甲骨文金文小篆隶书楷书行书草书

    克，甲金文从“由”，林义光、李孝定认为象人以肩膀担负重物，上部象所肩负之物，疑本义为肩负重物，引申为胜任。后期金文上部不从“由”而从“古”。罗振玉、朱芳圃认为“由”象头盔，其根据是“冑”字上从“由”，按此说不可信，张玉金指出“由”只是“冑”的声符。

    由，甲金文从“口”形上加一竖笔或一圆形，本义未有定论。“由”字现多指原因，又用作介词，相当于“从”。甲骨文“由”字首先由唐兰释出，他认为字从“冑”之初文省略而来，象古时战士所戴之头盔之形。唯张玉金、陈剑已先后指出“冑”只是以“由”为声符，故此说未足采信。陈剑提出“由”字甲金文从“口”，“丨”声。“丨”为“针”的初文，“针”“由”古音相近，因此“由”或以“针”为声符，可备一说。

    剋，《説文》無剋字。① 取勝。② 限定日期。剋期。

    杆〔gàn / ㄍㄢˋ〕：树名。《玉篇》：“杆，檀木也。”《广雅·释木》：“杆，柘也。”

    赶〔gǎn / ㄍㄢˇ〕：追赶（晚起义）。明·洪楩《清平山堂话本·简帖和尚》：“皇甫殿直拽开脚，两步赶上。”

    槓〔gàng / ㄍㄤˋ〕：《王力古汉语字典》无此字，《说文》亦无此字，但《汉字全息资源应用系统》中列出《史记》中有一个用例“擊王離軍成陽南，復攻之槓里，大破之。”（卷五十四 曹相国世家）。但同时，“杠”字的引文有“卒不許項羽，而遣沛公西略地，收陳王、項梁散卒。乃道碭至成陽，與杠里秦軍夾壁，破二軍。楚軍出兵擊王離，大破之。”（卷八 高祖本纪）等多处皆作“杠里”。“杠里”是秦国邑名，不知道是《史记》里面就这么写的，还是《汉字全息资源应用系统》的资料错误。估计后者的可能性更大，这种因繁简字合并导致的错误，在这个系统中普遍存在。

    黨（党，朋黨）、党（党項族）、儻（傥） #

    甲骨文金文小篆隶书楷书行书草书

    “黨”的本义为颜色黯淡不鲜明，故从“黑”。古代多表示地方基层组织。又解作亲族、朋党。现在多表示政治组织，即政党。

    “黨”之簡化字今作“党”，但古時“黨”與“党”表示不同的詞。“党”專指古代外族，即“羌族”的其中一分支“党項”。

    里（鄉里、鄰里、公里）、裏（里，裏外、衣服裏子，※裡） #

    甲骨文金文小篆隶书楷书行书草书

    金文從“衣”，“里”聲。與小篆同。本義是衣服的內層。異體作“裡”。“裏”字後來普遍地引申爲裏外的“裏”。

    按香港《漢語多功能字庫》：“又‘裏’字於香港，本來用得遠較‘裡’字爲普及，自中文電腦發達以來，由於香港用‘大五碼’（Big5）標準，‘裏’字於許多字型制式中不被支援，遂漸被‘裡’字取代。但這問題其實並不嚴重，因‘裏’、‘裡’都是繁體字，前者爲古代寫法，可追溯至金文，後者爲後起字，但二者都从‘衣’从‘里’，皆爲形聲結構，其之爲異體字，斷無疑問。因‘裏/裡’爲異體，與‘羣/群’或‘稾/稿’之爲異體道理一樣。”仍然不知道“於許多字型制式中不被支援”是什麼意思，又是爲什麼不被支援，難道是指字形過於修長，在當時的顯示設備中不便於顯示？

    甲骨文金文小篆隶书楷书行书草书

    从“田”从“土”，本义为古代村民聚居的地方。“里”字初见于西周金文，从“田”从“土”，会土田可居之意。西周金文“田”“土”中划贯通，晚周以后习惯将土旁竖笔上的圆点延伸为短横，遂为篆文“里”字所本。一说先民农业有所发展、田地有所耕植以后，始能定居下来，故从“田”从“土”会“居也”之意（参戴家祥）。

    困（困難、困乏）、睏（犯睏、睏覺） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨、金文、小篆從“木”從“囗”，木在囗中，“囗”亦作聲符。“困”是“梱”、“閫”的初文，本義是門檻。

    段玉裁注“梱”字曰：“謂當門中設木也。”俞樾：“困者，梱之古文也。《木部》：‘梱，門橛也。从木，困聲。’困既从木，梱又以木，緟椱無理，此蓋後出字，古字止作困。从囗者，象門之四旁，上爲楣，下爲閫，左右爲根也，其中之木即所謂橜也。”“困”的本義是門檻。由此義引申出阻礙之意。

    後來“困”字又解作疲憊。唐白居易《賣炭翁》：“牛困人飢日已高，市南門外泥中歇。”指牛隻疲倦了，賣炭翁餓了，就在市集南邊的城門外休息。“困”後因假借義太多，加義符“木”，另造“梱”字；或加義符“門”，另造“閫”字，以表示本義（俞樾、楊樹達）。

    困：① 艱難，窘迫。《禮記·中庸》：“事前定則不困。”② 貧乏。《左傳·僖公三十年》：“行李之往來，共其乏困。”③ 勞倦。《後漢書·耿純傳》：“世祖至營，勞純曰：‘昨夜困乎？’”

    睏，《王力古漢語字典》無。《説文》無。① 睡。睏覺。② 倦而欲睡。睏得睜不開眼。

    個人理解：“困”泛指疲憊（並不是想睡覺）；“睏”指睡覺或疲憊欲睡。例如《後漢書·耿純傳》：“世祖至營，勞純曰：‘昨夜困乎？’”是問他是否勞倦，而不是問他是否盹得慌。“睏”字可能出現得很晚，專門用來表示盹睏，也不是很嚴格的分化字，所以，用“睏”的地方用“困”也不能算錯。

    別（告別）、彆（别，彆扭） #

    彆（别）:〔biè / ㄅㄧㄝˋ〕，彆扭

    甲骨文從“刀”從“冎”（骨的初文），會以刀分解骨肉之意（陳煒湛、季旭昇），本義是分解、分割。古書中“別”有用作以刀分割的本義，《淮南子·齊俗訓》：“宰庖之切割分別也。”由分解引申，有分別、離別之義。

    彆〔biè / ㄅㄧㄝˋ〕，《王力古漢語字典》無。① 弓末彎曲的地方。② 執拗。彆扭。③ 改變他人的想法、意見。彆不過他，只好順從了。③ 強忍住。同“憋”。可能是在古白話小説中有這種用法，現今沒有也沒必要這麼用了，而且現在“彆〔biè / ㄅㄧㄝˋ〕氣”或“彆〔biè / ㄅㄧㄝˋ〕口氣”和“憋〔biē / ㄅㄧㄝ〕氣”意義不同，前者有賭氣的意思。

    • 傭（佣）:〔yōng / ㄩㄥ〕，女傭、傭人

    甲骨文从“庚”，“”声。“”是筒、桶一类东西的象形字。“庸”或从“庚”，“用”声。“用”是由“”分化出来的一个字。“庸”是“鏞”的初文，本义是大鐘。（裘锡圭）

    金文“庸”可表示奴隶，如訇：“今余令汝啻官司邑人，先虎臣、後庸。”“先虎臣”是一种军旅，“先”指先锋。“後庸”指追随于正规军队之后的“庸”。（陈世辉、裘锡圭）

    后还表示受雇用，如《史記·陳涉世家》：“若爲庸耕，何富貴也？”又指被雇用的人，后作“傭”。

    佣〔yòng / ㄩㄥˋ〕，表示佣金，即买卖成交后付给中间人的酬金。

    “傭”在大陸只有〔yōng / ㄩㄥ〕一個讀音，在臺灣有〔yōng / ㄩㄥ〕和〔yóng / ㄩㄥˊ〕兩個讀音，但是字義沒有區別，〔yóng / ㄩㄥˊ〕只不過是另一種讀音。咱們在讀“女傭”時，不出意外都是一聲，而“傭人”一詞似乎更多的是讀四聲，但實際上“傭人”和“用人”不是一回事。“傭人”指的是受雇爲人做事的人，其實是文言詞彙。而“用人”其實專指舊社會的家僕，或者至少其本義是拿來使用的人，因此本作“用人”亦可做“佣人”。受人雇傭做事的人都可以叫“傭人”，但不是都可以叫“用人”，這就是區別。

    簡化字的“佣”合併了“傭”和“佣”兩個字，如果覺得不好分，所有讀一聲的詞用的肯定是“傭”字，讀四聲的詞用的肯定是“佣”字。而“佣”我認爲只不過是“用”在部分詞彙中的偏旁類化和專有化，其實都可以用“用”替換。比如，臺灣的字典中說到，“佣金”也稱爲“用錢”，“用錢”亦作“佣錢”。

    臺灣字典中說“僱傭”（僱爲雇的異體，也是偏旁類化的結果）也作“僱佣”“僱用”，大陸字典亦有“雇佣〔yōng / ㄩㄥ〕”和“雇用”兩個詞。其實，“僱佣”和“僱用”是偏旁類化的結果，“僱傭”和“僱用”就是兩個不同的詞了，只是從不同角度表達相似意義的詞。現實生活中我們一般會說“我雇用了兩名員工，我們之間是雇傭關係”，不太會說“我們之間是雇用關係”，這也能體會出“雇傭”和“雇用”是不同的。

    余、餘（余或馀，剩餘） #

    甲金文象古人搭建的简易茅屋之形，“余”是“舍”的初文，本义是房舍。后借用来第一人称代词。

    甲金文用作第一人稱代詞，毛公鼎：“余一人才（在）立（位）”，即我一人在位。又假借爲“舍”、“捨”，牧簋：“舍[馬]四匹”，表示給予四匹馬。

    戰國竹簡也用作第一人稱代詞，又假借作“餘”，表示多餘的，《上博竹書二·容成氏》簡29：“民又（有）余（餘）飤（食）”，意謂人民有餘糧。

    餘，從“食”，“余”聲。表示多出來的、殘留的。《説文》：“饒也。从食，余聲。以諸切。”《廣雅·釋詁四》：“餘，盈也。”《玉篇·食部》：“餘，殘也。”多出來的部分，從功用上來説如果是不需要的，即可視爲殘留，故兩義其實相通。

    在产生歧义时，简化字可用“馀”字。

    泛（泛舟、寬泛）、氾（氾濫）、汎 #

    泛，從“水”，“乏”聲，本義爲浮行，也作“汎”。《説文》：“泛，浮也。从水，乏聲。孚梵切。”《説文》：“汎，浮皃（貌）。从水，凡聲。孚梵切。”

    氾，《説文》：“濫也。从水，㔾聲。孚梵切。”

    《王力古漢語字典》對三字辨析的大意爲：泛、氾、汎三字均爲敷母。氾、泛在談部，汎在侵部，三字音近。三字實際用法相同，只是通“覂〔fěng / ㄈㄥˇ〕”（覆、翻）時只寫作“泛”。

    三字雖然用法相同，但在實際使用中，在不同詞語中會用不同的字。個人感覺抽象意義和泛舟多用“泛”；氾濫義多用“氾”；“汎”似乎現在用的比較少，只用在抄寫古代文獻，比如《三字經》“汎愛眾”。

    證（证，證明）、証（证，証諫） #

    證：從“言”，“登”聲，本義爲告發、檢舉。《説文》：“告也。从言，登聲。”《論語·子路》：“葉公語孔子曰：‘吾黨有直躬者，其父攘羊，而子證之。’”

    証〔zhèng / ㄓㄥˋ〕：從“言”，“正”聲。本義是諫正。《説文》：“証，諫也。从言，正聲。”如《戰國策·齊策一》：“士尉以証靖郭君，靖郭君不聽，士尉辭而去。”《呂氏春秋·貴當》：“觀人主也，其朝臣多賢，左右多忠，主有失，皆交爭証諫，如此者，國日安，主日尊，天下日服，此所謂吉主也。”

    “証”後通“證”，表示憑證。《正字通·言部》：“証，與證通。”如《水滸傳》第十二回：“因此一時性起，將那人殺死。眾鄰舍都是証見。”又表示驗證。如《明史·葉伯巨傳》：“由此言之，分封踰制，禍患立生，援古証今，昭昭然矣。”

    金文從“茻”，“廌”亦聲。“廌”象野獸，全字象野獸在吃草，本義是牧草。《説文》：“薦，獸之所食艸。从廌从艸。古者神人以廌遺黃帝，帝曰：何食何處？曰：食薦，夏處水澤，冬處松柏。”《莊子·齊物論》：“民食芻豢，麋鹿食薦”，意謂人吃肉類，麋鹿吃草（陳鼓應）。後引申爲墊席，又假借爲進獻、送上。

    漢帛書表示推薦，《馬王堆·春秋事語》第50行：“其達不見薦言”，意謂顯貴的時候沒有人用言詞推薦。

    ① 野獸、牲畜所吃的草。② 草席，草墊。③ 一再，頻頻。《詩·小雅·節南山》：“天方薦瘥，喪亂弘多。”④ 進，進獻。《左傳·宣公十四年》：“誅而薦賄，則無及也。”⑤ 推薦，薦舉。⑥讀〔jìn / ㄐㄧㄣˋ〕，通“搢”。插。“薦紳”通“搢紳”或“縉紳”。指士大夫有官位的人。司馬貞《索隱》：“薦音搢。搢，挺也。言挺笏於紳帶之間，事出《禮·內則》，今作‘薦’者，古字假借耳。《漢書》作‘縉紳’。”

    《説文》：“（荐）：薦蓆也。从艸，存聲。在甸切。”

    ① 草席。② 聚集。《左傳·襄公四年》：“戎狄荐居，貴貨易土。”杜預注：“荐，聚也。”孔穎達疏引服虔云：“荐，草也，言狄人逐水草而居，徙無常處。”③ 頻頻，一再。《左傳·定公四年》：“吳爲封豕長蛇，以荐食上國，虐始於楚。”《國語·吳語》：“天奪吾食，都鄙荐饑。”

    “薦”今簡化爲“荐”，在上古“荐”與“薦”是兩個不同的字，讀音也有異（聲母清濁不同，韻部亦不同）。“荐”字用在“草席、草墊”和“一再，頻頻”的意義上，與“薦”相同。《詩經》《楚辭》《史記》多用“薦”，《國語》《左傳》《墨子》等多作“荐”。但“薦”字的“推舉”和“進獻”意義在漢以前的典籍裏不作“荐”。唐宋以後逐漸混用。《字彙》：“薦，音荐，與荐同。”《正字通》：“荐，同薦。”

    範（范，範圍、典範）、范（姓氏） #

    範：從“車”，“笵”省聲。一説從“”聲。本義是古代出行前祭祀的儀式。

    范：從“艸”，“氾”聲。本義是草名，又是昆蟲之名（是蜂的一種）。

    《王力古漢語字典》辨析笵、範、范：笵的本義指模子、法則。範的本意指範軷，一種出行時的祭祀，但文獻多以範來表示模子、法則義，其本義鮮用。范的本義指一種草，古籍中也常用指模子、法則。笵一般不用於姓氏。范用於姓氏，如范雎、范仲淹。範也用於姓氏，但和范是不同的姓氏，不得相混。

    傑：從“人”，“桀”聲，本義爲特異，超出一般。《説文》：“傲也。从人，桀聲。”

    > 结构：木缶鬯彡。上部左中右结构，书法一般都是从左往右写。

    鬱：甲金文從“林”從“勹”從“大”，“大”象正面站立的人形，“勹”象俯伏人形，疑表示林中有一人伏地，一正立之人踐踏其背，被踐踏者心情抑鬱，因而有鬱結之義（于省吾）。本義是憂鬱、抑鬱，即胸中鬱積心結。

    字形方面，甲金文象一正立之人（大）踩踏一俯伏之人（勹），被踩者心情憂鬱。“大”形後來訛變爲“缶”，“勹”形訛變爲“冖”，另加“鬯”和“彡”，成爲楷書的“鬱”字。而加“鬯”（香酒）爲意符，是由於“鬱”後用來表示鬱金香調製的酒。

    詞義方面，“鬱”的本義是心裏積聚憂愁、煩悶，引申之，草木茂盛，枝葉積聚也稱“鬱”。曹植《贈白馬王彪》：“山樹鬱蒼蒼”。《説文》：“鬱，木叢生者。从林，省聲。”段玉裁注：“《秦風》：‘鬱彼北林。’毛曰：‘鬱，積也。’”漢帛書表示鬱滯，《馬王堆帛書·五十二病方》第253行：“鬱血”，即瘀血，指滯留積聚之血（吳振武）。

    表示鬱金香的“鬱”本寫作“”，後來被“鬱”吞併。

    “鬱”和“郁”本來是不同的字，在古代不同音（“鬱”屬物韻，“郁”屬屋韻。現代許多方言如粵語仍讀作二音），它們都有當芳香講的用法，《文選·劉孝標〈廣絕交論〉》：“言鬱郁于蘭茝”，李善注：“鬱郁，香也。”

    ① 草木茂盛貌。② 果名，棣屬。③ 香草名，即鬱金香草。④ 阻滯，蘊結。⑤ 鬱邑，雙聲聯綿詞。憂愁貌。亦作鬱悒。

    鬯〔chàng / ㄔㄤˋ〕：甲骨文，金文。甲金文象酒器中盛載酒之形，酒器中的小點代表所容之酒，本義是酒。“鬯”是古代祭祀、宴飲用的酒，用鬱金草和黑黍釀成。

    郁：從“邑”，“有”聲，古地名，在今陝西省。

    “郁”表示富有文采的樣子。《論語·八佾》：“周監於二代，郁郁乎文哉！”

    “郁”又表示香氣濃郁。

    “郁”也是姓氏。《廣韻·屋韻》：“郁，姓。魯相有郁貢。”

    “郁”還表示沒有內核的果實。《論衡·量知》：“物實無中核者謂之郁，無刀斧之斲者謂之樸。”

    ① 郁夷，古地名。② 通“彧”。形容事物的盛美、繁多。可以是富有文采。③ 通“燠”。溫暖（後起義）。

    昆、崑（崑崙） #

    按《漢典》。崑崙現作昆侖；峨嵋也作峨眉。崑和昆不是繁簡關係，崙和嵋沒有對應簡化字。其實就是由於崑崙屬於專字，現在不用了，也就不需要簡化了。

    制（制服、制度）、製（制，製造） #

    甲骨文金文小篆隶书楷书行书草书

    金文從“木”從“刀”，或不從“木”而從“未”，“木”、“未”皆象樹木形，會以刀砍劈木材之意。本義是折斷。王子午鼎“木”形右邊中部加兩短橫，以示斧斤折斷草木之處，與戰國竹簡、《説文》籀文形體相同。

    “制”、“折”二字形音義皆十分相近，“折”象以斧斤斷木，“制”象以刀劈木，二字古音同在章紐月部，是一組同源字。參見“折”。古書中“制”也有裁木之義，《淮南子·主術》：“巧工之制木”，高誘注：“制，裁也。”《説文》：“制，裁也。从刀从未。未，物成有滋味，可裁斷。一曰止也。，古文制如此。”

    “制”本指剪裁、切斷樹木，引申指根據法度而作出的決斷、制裁、控制等，再引申指制度、體制。

    金文表示法度、準則，王子午鼎：“萬年無諆（期），子孫是制。”勉勵子孫以鼎銘所言爲準則（參張世超）。

    古“折”、“制”通用，《郭店楚簡·緇衣》引《呂刑》云：“折以型（刑）”，今本《禮記·緇衣》引作“制以刑”，指以刑法來裁斷。段玉裁《説文解字注》：“古多假折爲制。《吕刑》：‘制以刑’。《墨子》引作‘折則刑’。《魯論》：‘片言可以制獄。’古作折獄。”

    《王力古漢語字典》：① 裁衣。引申爲製作。② 節制，控制，制止。③ 制度。④ 人君的教令。⑤ 禮喪服有四制。後來以守父母之喪爲“守制”。

    製：從“衣”，“制”聲，本義爲剪裁。《説文》：“裁也。从衣从制。” 《左傳·襄公三十一年》：“子有美錦，不使人學製焉。”杜預注：“製，裁也。”

    《王力古漢語字典》：① 裁製衣服。② 衣服的形製、樣式。《漢書·卷四三·酈陸朱劉叔孫傳·叔孫通》：“通儒服，漢王憎之，乃變其服，服短衣，楚製。漢王喜。”③ 製造，製作。④ 寫作，著述。⑤ 通“致”。

    “製”雖然有衣服的樣式的意義，但“制服”一詞不作“製服”。因爲“制服”的含義是遵從規定的服裝，比如警服、軍裝等。

    《王力古漢語字典》：制，《說文》作“，裁也”，本義是“裁斷”，亦引申爲制造、制作。故與“製”本同源。後來，“制”多用於抽象的制作、制裁。如《國語·晉語》二：“因驕以制人家”。而“製”多用於具體的製造，今簡化爲“制”。

    征（征伐、長征）、徵（征，象徵、徵求）、懲（惩） #

    大陆标准：第8笔为横。这个字就是五音“宫商角徵羽”的“徵”。

    甲骨文金文小篆隶书楷书行书草书

    甲金文從“彳”從“正”，“正”亦聲。古代“征”同“正”字，“正”象人向城邑或目標前進，本義爲遠行，爲明確行走義而加義符“彳”。古代多爲了征戰而遠行，故字有討伐之意。

    傳世文獻中“征”可指征稅，《左傳·文公十一年》：“宋公於是以門賞耏班，使食其征。”杜預注：“征，稅也。”由於征稅的“征”與徵求的“徵”意義相近，有人認爲兩字可通用。但古代二字讀音不同，即使後來兩個字變成同音字，古人一般不會把兩個字混爲一談。由於簡化字把“徵”併入了“征”，所以簡化字中徵求與征稅兩義都以“征”字表示。

    以上是香港中文大學《漢語多功能字庫》中的解釋。我查臺灣字典，“征稅”一般作“徵稅”，但用作名詞表示賦稅時（一般是古文）只能用“征”。臺灣字典給出的“徵稅”的用例爲：《左傳·昭公二十年》：“布常無藝，徵斂無度。”

    ① 行。如：遠征、長征。② 征伐。如：征討。③ 抽稅。如：征稅、橫征暴斂。又用作名詞。

    甲骨文金文小篆隶书楷书行书草书

    甲金文的字形象背部有齒形裝飾的刀子，可隸定爲“”。甲骨文下或從“止”，金文或增“口”、“阜”、“辵”等偏旁。這些字形都可看作“徵”的古字。“徵”表示徵召、收取、徵兆等義。

    甲骨文中一般用作國族名，金文也用作族名。西周中期史牆盤“伐”讀作“懲伐”。

    ① 召。如：徵兵、徵召。② 求。如：徵文、徵婚。③ 問，詢。如：徵詢意見。④ 跡象。如：徵兆、吉徵、象徵。⑤ 證驗，證明。如：旁徵博引、信而有徵。

    • 症:〔zhèng / ㄓㄥˋ〕，病症
    • 癥（症）:〔zhēng / ㄓㄥ〕，癥結

    症〔zhèng / ㄓㄥˋ〕：“證”的俗字。病徵的症，古作證。

    癥〔zhēng / ㄓㄥ〕：癥結，病名，腹內結塊之病。《史記·扁鵲倉公列傳》：“以此視病，盡見五藏癥結。”

    徵、證、症區別：“徵”和“證”古音同在蒸部，“徵”是端母，“證”是照三，聲母也很接近。“徵”有徵兆、證驗義，“證”是證實、證據義。典籍中常以“證”訓“徵”，或同訓“驗”。“症”是“證”字病症義的後起專用字。症，病徵也，古但作“證”。

    念、※唸（唸書）、鮎（鲇，※鯰） #

    念：甲骨文，金文。古文字“念”從“亼”（倒口）從“心”，意會口向自己的心説話。本義是心念、意念。後期金文從“心”從“今”，基本上仍保留上述的會意內容，但“今”也可視爲聲符，故“念”字可視爲會意字或形聲字；後期金文或從“今”並增“口”而成從“含”從心。

    今：甲骨文，金文。從“亼”（倒口）從“一”，是“吟”的初文。本義有兩説，一説表示閉口不言，一説表示呻吟。

    唸：《説文》：“㕧也。从口，念聲。《詩》曰：民之方唸㕧。都見切。”唸㕧，表示呻吟。“唸”典籍又作“”、“殿”、“”。“㕧”〔xī / ㄒㄧ〕典籍又作“吚”、“咿”、“屎”，表示呻吟。

    “念書”作爲一種口語説法，可能以前寫作“念書”“唸書”都可以，現在臺灣好像將表示呻吟義的“唸”多用作唸書的“唸”。大陸似乎將“唸”作爲異體。

    我記得上學時學的鯰魚用的是“鯰”字，可是查了字書發現《説文》有“鮎”無“鯰”，兩岸都定“鯰”爲異體，簡化字也是“鲶”“鲇”都有。

    岳（岳父）、嶽（岳，五嶽） #

    甲骨文小篆隶书楷书行书草书

    甲骨文“岳”象層巒疊嶂，山外有山之形（孫詒讓、屈萬里、李孝定），與《説文》“嶽”的古文形近。“岳”是“嶽”的異體字，本義是高大的山。《説文》：“嶽，東岱、南靃、西華、北恆、中泰室，王者之所以巡狩所至。从山、獄聲。（岳），古文象高形。”《玉篇·山部》：“岳，同嶽。”屈萬里又認爲“岳”象山上有樹，樹外又有山峰之形。東漢魯峻碑“岳”字上部寫作從“丘”。

    甲骨文用作本義，指山神，古人把高山、大川當作神祇。屈萬里認爲卜辭的“岳”是太岳山，孫詒讓、彭裕商則認爲是嵩山。詹鄞鑫則釋甲骨文此字爲“華”，認爲是西岳華山。《合集》30419：“于岳求，又（有）大雨”，表示向山神求雨，有大雨。傳世文獻也有向山河求雨的記載，《晏子春秋·內諫上》：“齊大旱逾時，景公召群臣問曰：‘天不雨久矣，民且有饑色，吾使人卜之，祟在高山廣水，寡人欲少賦斂以祠靈山，可乎？’”

    “岳”又用以尊稱妻子的父母一輩長者。“岳父”指妻子的父親，又稱“泰山”。

    “五嶽”亦作“五岳”，但“四岳”（官名）不作“四嶽”，“岳父”不作“嶽父”。“岳飛”不作“嶽飛”。

    表示高山時，“嶽”同“岳”；姓、對妻子父母的尊稱只用“岳”。

    沾（沾沾自喜、沾邊）、霑（沾，雨露均霑、霑染） #

    沾：從“水”，“占”聲，本義爲淇水支流，源於山西省壺關縣南。《説文》：“水，出壺關，東入淇。一曰：沾，益也。从水，占聲。”

    《王力古漢語字典》：

    〔zhān / ㄓㄢ〕：① 水名。流經今山西、河北兩省。② 通“霑”。浸潤，濡溼。汗出沾背，亦作汗出霑背。③ “沾沾”自矜貌。沾沾自喜。

    〔chān / ㄔㄢ〕：通“覘”。視。

    〔tiān / ㄊㄧㄢ〕：增添。《説文》：“沾，益也。”段玉裁注：“沾、添古今字，俗製‘添’爲沾益字，而‘沾’之本義廢矣。”

    霑〔zhān / ㄓㄢ〕：《説文》：“雨也。从雨，沾聲。張廉切。”

    《王力古漢語字典》：亦作“沾”。沾濕。《離騷》：“霑余襟之浪浪。”比喻受恩澤。《韓非子·詭使》：“今戰勝攻取之士勞而賞不霑，而卜筮視手理狐蟲爲順辭於前者日賜。”

    按《王力古漢語字典》：“沾”只在“浸潤、濡濕”義及其比喻義上通“霑”，其他義項都不和“霑”相通。因此，“沾沾自喜”不能寫作“霑霑自喜”。

    而“霑”字只有“浸潤、濡濕”義及其比喻義“受恩澤”；“沾”的義項更多，且包含了“霑”的全部義項。據此，“霑”可理解爲“沾”在部分義項上的異體字。但事實是“沾”霸佔了“霑”的義項，我理解應該是這樣的：“沾”本來就是水名，後來用作連綿詞“沾沾”，也成了“添”的古字。而“霑”本來就是雨露均霑、霑染等詞的原字，只是後來也寫作“沾”了，以至於一些後來才出現的詞只有“沾”的寫法而沒有“霑”，也導致詞典中對“霑”和“沾”的解釋看起來很混亂，詞例一會兒用“沾”，一會兒用“霑”。

    其實還可理解爲“沾”行而“霑”廢，後者只出現在慣用詞裏。

    《臺灣教育部重編國語辭典》對“沾”的解釋：

    ① 浸溼。

    ② 接觸、接近。如：“滴酒不沾”、“這些話一點也沾不上邊。”

    ③ 感染、染上。如：“沾染惡習”。

    ④ 藉別人的關係而得到好處、利益。如：“沾光”、“沾便宜”。

    ⑤ 帶、有。如：“我和他既不沾親，也不帶故。”

    其實，②到⑤都是近現代漢語賦予它的意義，都引申自“接觸”“接近”或者我説的“夠得着”“靠近”義吧。

    《臺灣教育部重編國語辭典》對“霑”的解釋：

    ① 浸溼、沾溼。同“沾”。《文選·江淹·恨賦》：“此人但聞悲風汩起，血下霑衿。”

    ② 比喻受人恩惠。如：“霑恩”、“法雨均霑”。（按《王力古漢語字典》這個意義也可作“沾”）

    復（复，復發、恢復、日復一日、復仇）、複（复，複雜、複賽、複製、複數）、覆（复，答覆、覆命、覆核、反覆無常） #

    甲骨文复金文小篆隶书楷书行书草书

    “复”字之甲骨文從地室之形，從“夊”（象足趾）。上部的象古人居住的室（即土室），而足趾在其下，以會往來進出（陳永正、季旭昇）、回來之意。“复”是“復”的初文。

    甲骨文表示回來，《合集》7076正：“雀今來复（復）”（王蘊智、郝士宏），“雀”是部族名，意謂雀如今回來。又用作方國名。金文用作地名。

    “復”是“复”增加會意符號“彳”或“辵”（表示行走）而來。

    “復”“複”“覆”這三個字有意義相近甚至相通之處，我先説一下我個人區分這三個字的方法，再擺上《王力古漢語字典》和《重編國語辭典修訂本》的解釋。

    臺灣的字典雖然會寫在某些地方兩字通用，但實際使用中一般只會見到一種用法，除非你故意干擾輸入法的選詞。另外，個人認爲雖然字典中説兩字通用，但是教學中應該只教一種用法，教古文的時候可能會遇到通用的情況，跟學生説一下就是。比如“複試”這個詞，教給學生的肯定是“複”，但古文中有寫作“覆試”的，肯定是跟學生説這個字古時通用，絕對不會跟學生説你們平時寫哪個都行。所以，下面的區分方法，就是基於我理解的現如今的規範用法寫的，並不是説這個詞只能這麼寫。

    “復”用在“又”“再”“返回”意義上。比如：

    ※ 復發。又一次發作。 ※ 復習。再溫習一遍。《重編國語辭典修訂本》説也可以作“複習”。其實，“又”“再”義和“重複”義在這個詞中都可以解釋得通。我剛學會繁體字的時候認爲這個詞應該作“複習”，可後來看到的繁體字材料一般都做“復習”。舊《一站式》我對這個詞的解釋也是片面的，現在仔細查過後，我覺得還是作“複習”比較合適，“復習”可能只是習慣的錯誤用法（沒有證據）。臺灣有家長發帖問這個問題，説自己記得應該是“複習”，怎麼孩子寫成“復習”老師也不算錯，而且老師也是一會兒“複習”，一會兒“復習”的。後來查辭典發現通用。看來這個詞真是有些問題。 ※ 千金散盡還復來。還會再來。 ※ 百川東到海，何時復（再）西歸？ ※ 復興。返回興盛的狀態，重回興盛。還有光復、復旦大學（取復興義）。 ※ 恢復。返回以前的狀態。 ※ 復仇。把你給我的傷害返回給你，回報仇恨。

    “複”《説文》：“重衣皃（貌）。从衣，复聲。一曰：褚衣。”本義爲夾衣。我理解就是多層衣服吧。因此，用在“繁複”“重複”“多（與單相對）”的意義上。比如：

    ※ 複雜。多而雜； ※ 複製。重複製造； ※ 複試。爲了篩選而重複（或多次）進行的考試。還有初賽、複賽、決賽，複賽是相對於初賽而言再次舉行的比賽。寫成“復賽”行嗎？我覺得不行。如果你能感覺到“復賽”有什麼不同，我覺得你能輕鬆區分‘復’和‘複’。“劉翔復賽了，他終於擺脫了傷病的困擾！”我覺得“復賽”就是“又恢復比賽、又回到比賽”的意思。此外，複審、複查、複檢、複核等詞也是重複的意思，是相對於只進行一次的檢查而言的。 ※ 複數。① 語法上，兩個以上的數量稱爲“複數”，與“單數”相對。② 設a、b爲實數，則a+bi的數稱爲“複數”，i是-1的平方根。英文是complex number，即複雜的數（其實就是結構複雜的數，比實數複雜多了）。 ※ 複（多層）式房、複（多，與單相對）眼、複（複雜）合結構、山重水複（複雜）疑無路。而“復合”是重新結合的意思。

    “覆”本義是翻過來、覆蓋。主要用在“答覆”和“再次檢查”義上。比如：

    ※ 覆命、回覆。都有答覆義。 ※ 覆查、覆核或覆覈。都是再次檢查。覈和核都有核對義，詳見覈。

    “覆”的這些用法是習慣用法，這些意義可以用“復”或“複”代替。其實，大陸的字典裏“覆”的義項下註明了通“复”。“復”“複”合併簡化成“复”，“覆”並沒有。

    “復”和“複”在大多數情況下容易區分，比如“恢復”“復興”“重複”“複雜”。但在有些詞中比較難以區分，比如“複試”“複查”爲什麼不能用“復”呢？“又測試一遍”、“又檢查一遍”不可以嗎？而且字典中也説“復習”和“複習”相同，那爲什麼“複試”不能和“復試”相同？“再來一遍”不就是“重複一遍”嘛，怎麼區分？“複試”可不可以寫成“復試”我真的不知道，只能説沒查到這種用法，也沒查到明確説這種用法不對的，不過我覺得用“複”更合理一些。因爲“複試”“複查”就是相對於“一次測試”、“一次檢查”而言的，“複”與“單”相對，“複試”“複查”就是進行多次測試和檢查。“複習”一般字典都説重複學習的過程，那麼理應作“複”，而又説亦可作“復習”，不知道有沒有更深層次的原因，或者只是習以爲常的錯誤用法。

    漢字中確實有一些本義不盡相同，但可引申出相似意思的字。這些字在後期的使用過程中逐漸在某些詞中通用，這是很自然的過程。在學習這些字的時候，只要別把不該弄混的字義弄混就行，本來就相通的也無所謂對錯，接觸多了自然養成習慣就可以。沒有什麼港臺標準，弄清這些字的來龍去脈就是標準；也沒有什麼明清標準，歷代字書中心思想是追本溯源，這個標準是一致的。

    復〔fù / ㄈㄨˋ〕，房六切，入，屋韻，奉。覺部。① 返回。《説文》：“往來也。”《易·泰》：“無往不復。”② 告，回答。《書·説命》：“説復於王。”③ 報復。《左傳》定公四年：“我必報楚國。”④ 免除賦稅或勞役。《荀子·議兵》：“中試，則復其户，利其田宅。”《史記·商君列傳》：“僇力本業，耕織致粟帛多者，復其身。”⑤ 通“複”。《史記·秦始皇本紀》：“爲復道，自阿房渡渭，屬之咸陽。”

    複〔fù / ㄈㄨˋ〕，方六切，入，屋韻，非。覺部。① 重衣，即夾衣。《説文》：“複，重衣也。”《釋名·釋衣服》：“有裏曰複，無裏曰禪。”《三國志·魏書·管寧傳》：“寧常著皁帽，布襦袴，布裙，隨時單複。”……亦指絮有絲綿的衣服。《説文》：“……一曰禇衣。”……引申爲夾層的。《舊唐書·王稷傳》：“廣治第宅，嘗奏請藉坊以益之，作複垣洞穴，實金錢於其中。”…… ② 重複，繁複。《文選》漢·張衡《東京賦》：“複廟重屋，八達九房。”李善《注》引薛綜曰：“複廟，重覆也。”宋·陸游《遊山西村》詩：“山重水複疑無路，柳暗花明又一村。”③ 土窟。……孔穎達疏：“複穴者，謂窟居也。”……這個意義，説文作“”。

    複，復。二字幫並旁紐，覺部疊韻。音甚近，義亦相通。“複”的本義是重衣，引申爲重複、繁複；而“復”，《説文》彳部：“往來也。”本義指行走的道路相重。《廣雅·釋詁》四：“復，重也。”《管子·牧民》：“不行不可復者。”引申爲反覆，恢復。故同源。今二字均簡化爲“复”。

    覆〔fù / ㄈㄨˋ〕，芳福切，入，屋韻，敷。覺部。① 翻，翻轉過來。《説文》：“覆，覂也。”段玉裁注：“‘又’部‘反’下曰：‘覆也。’反覆者，倒易其上下。”…… ② 覆蓋，掩蔽。《説文》：“覆，……一曰蓋也。”…… ③ 審察。《爾雅·釋詁》下：“覆，審也。”《周禮·考工記·弓人》：“覆之而角至，謂之句弓。”鄭玄注：“覆，猶察也。”《漢書·元帝紀》：“今不良之吏覆案小罪，征召證案，興不急之事，以妨百姓。”④ 反，反而。《詩·大雅·瞻卬》：“人有土田，女反有之；人有民人，女覆奪之。”鄭玄箋：“覆，猶反也。”《漢書·酷吏傳·嚴延年》：“於是覆劾延年闌內罪人，法至死。”顏師古注：“覆，反也，反以此事劾之。”⑤ 回復，答復。……這個意義通常用“復”字。⑥ 重複。……這個意義，通常作“複”。

    勝（胜）、胜（腥本字；胜鍵（化學）） #

    胜:〔xīng / ㄒㄧㄥ〕，腥本字；〔shēng / ㄕㄥ〕，胜鍵（化學）

    战国金文秦简小篆隶书楷书行书草书

    戰國晉系和楚系文字從“力”，“乘”聲。秦系文字（睡虎地秦簡）從“力”，“朕”聲，爲《説文》小篆所本。本義是能承擔、勝任。

    “乘”和“朕”的古音相近。《古文四聲韻》收的《古老子》古文“勝”和《説文》“乘”字的古文十分接近，白於藍以爲是借“乘”爲“勝”。

    乘，甲骨文，金文，從“大”從“木”，象人登上樹木，“木”上的“大”（象正面站立的人形）除張開兩臂外，兩足向木的左右兩側置放，有乘坐之意。戰國竹簡下部加從“車”，表示乘坐在車上，引申爲一切的登乘。本義是登乘、騎乘、乘坐。

    王國維指出甲骨文象人乘木之形。徐中舒認爲甲骨文“乘”字下部所從是“木”的訛形，按“木”字上部略去中豎，寫作分岔形，應有表意作用，象可供人乘坐的樹杈之形。金文或加從象雙腳形的“舛”，戰國文字象兩足的“舛”或延伸與兩臂相交，或省去“木”（戰國“公乘鼎”作），參見《古璽彙編》5373。楚文字或不從“木”而從“几”（與《説文》古文形近）或“車”，分別象登乘在几上和車上，表示登上之意同。古人登車以几爲踏腳，《禮記·內則》：“乘必以几”，《儀禮·士昏禮》：“婦乘以几”（陳邦懷）。秦文字下部訛變爲“桀”，是“乘”字小篆作“椉”，從“入”從“桀”的根據。

    胜〔xīng / ㄒㄧㄥ〕，《説文》：“犬膏臭也。从肉，生聲。一曰：不孰也。桑經切。”《説文》以爲“胜”專指狗的脂肪的氣味。凡肉不熟也有腥臭之味，故亦稱爲“胜”。後來多用“腥”表示腥臭之義，“腥”行而“胜”廢。故段玉裁《説文解字注》：“今經典膏胜、胜肉字通用‘腥’爲之而‘胜’廢矣，而‘腥’之本義廢矣。”（腥本義爲豬肉中像米粒的息肉）

    “胜”用作本義的例子如南宋羅泌《路史·遂人氏》：“乃教民取火，以灼以焫，以熟臊胜。”

    “胜遇”是鳥名，《山海經·西山經》：“有鳥焉，其狀如翟而赤，名曰胜遇，是食魚，其音如錄，見則其國大水。”

    現在用作化學名詞“胜鍵”時讀作〔shēng / ㄕㄥ〕。胜鍵：組成多肽鏈的胺基酸之間，乃藉前一個胺基酸的羧基與後一個胺基酸的胺基脫水，而形成一種共價鍵，並連結起來，這種鍵稱爲“胜肽鍵”。也稱爲“肽鍵”。

    洼（水洼）、窪（洼，窪地、低窪、坑坑窪窪） #

    洼〔wā / ㄨㄚ〕，① 《説文》：“深池也。从水，圭聲。一佳切，又於瓜切。”引申爲低陷貌。洼水：積滯的水。② 水名。渥洼。

    洼〔guī / ㄍㄨㄟ〕，姓。漢代有洼丹。

    窪〔wā / ㄨㄚ〕，《説文》小篆作“漥”：“漥，清水也。一曰：窊也。从水，窐聲。一穎切，又屋瓜切。”即“窪”的一個意思是清水，一個意思是窊（〔wā / ㄨㄚ〕凹陷）。

    “洼”從“深池”義引申爲低陷義；“窪”在《説文》時即有凹陷義。又是一個不同源但引申義接近的例子。現在一般在“水洼”或者單用“洼”表示積水或水坑的地方用“洼”字；在表示凹陷義的地方（不一定是水洼）用“窪”字，比如“窪地”“坑坑窪窪”“低窪”“鼻窪”。

    從臺灣字典看，“窪地”和“洼地”似乎有不同的意義。前者表示因地殼變動而陷落，高度在海平面之下的陸地。如：吐魯番窪地。後者表示小的低窪的地方。如：院子後有塊洼地，一下雨就積滿了水。

    核（核心）、覈（核，審覈） #

    > 下面是激的右半部分。
    甲骨文金文小篆隶书楷书行书草书

    亥，甲金文字形未有定論，疑象草根（許慎）；一説象豬，古與“豕”同字（商承祚、唐蘭）。後借用爲地支第十二位，古人用以紀時（午後九時至十一時）、紀日、紀年。十二生肖以“亥”爲“豬”。

    一説象草根藏於地下（許慎、林義光、葉玉森、高鴻縉、白玉崢、季旭昇），“一”象地面，下象草根藏於地下，是“根荄”的“荄”的初文，有“深”、“藏”之意，從而可説明“刻劃”之“刻”、“核心”之“核”之形構取義。不過，細看“亥”字之甲骨文，與根荄之象相去甚遠，難以令人信服。

    一説“亥”是“豕”字之訛誤（袁國華），字象一頭側躺著的豬，或以爲古時“亥”、“豕”一字（商承祚、唐蘭）。然而若將甲骨文中的“豕”字與“亥”字相比較，可見“豕”字普遍的特徵在於其明顯的腹部線條，而且商代金文之“豕”字其實是豬的圖像。凡此種種，均“亥”字所未見。可見，“亥”字的構形本義未有很好的説法。

    核，從“木”，“亥”聲，本義爲樹木之名。“核”不見於甲骨、金文，《説文》：“蠻夷以木皮爲篋，狀如籢尊。从木，亥聲。”段玉裁注：“今字果實中曰核，而本義廢矣。按許不以核爲果實中者，許意果實中之字當用覈也。《小雅》：‘肴核維旅’。班固、蔡邕作‘肴覈’。左思作‘肴槅’。槅卽覈也。今本《毛詩》作‘核’，非古也。《周禮》：‘其植物宜覈物’。覈猶骨也。《廣韵》云：‘，果子也。’此字近是。《玉篇》亦云：‘核，户骨切。果實中也。’”

    由段注可見“核”用作“果實中者”（即果實中心堅硬包裹果仁的部分）多於用作本義。《禮記·玉藻》：“食棗桃李，弗致于核。”《世説新語》：“王戎有好李，常賣之，恐人得其種，恒鑽其核。”也將“核”用作“果實中者”。

    “核”亦用作核心、中心，如《論衡·量知》：“文吏不學，世之教無核也。”；也用作真實、實在，如《漢書·司馬遷傳贊》：“其文直，其事核。”

    “核”還用作考察、對照。今天所言“核算”、“核對”來自此義。《資治通鑑·漢元帝建昭二年》：“事必核其真，然後脩之。”

    甲骨文金文小篆隶书楷书行书草书

    “覈”亦作“”，指經反覆驗證檢核，而得事之確實。《説文》：“覈，實也。考事襾笮，邀遮其辤（辭），得實曰覈。从襾敫聲。，覈或从雨。”段玉裁注：“襾者，反覆之；笮者，迫之；徼者，巡也；遮者，遏也。言考事者定於一是，必使其上下四方之辭，皆不得逞，而後得其實，是謂覈。此所謂咨於故實也，所謂實事求是也。”又與“核”通。

    按照段玉裁説法，許慎不把“核”解作果實中者，認爲許慎的意思是“覈”當爲果實中者。不過光看許慎對“覈”的解釋，看不出許慎有這個意思。古時候“核”與“覈”應該是通用。現在港臺地區“核”主要用在果核、核心義上，也用於考核義；“覈”主要用在覈查、考覈義上，似乎不用在果核義上。

    臧、贓（赃）、臟（脏，心臟）、髒（脏，骯髒、髒淨） #

    甲骨文金文小篆隶书楷书行书草书

    臧，甲骨文從“戈”從“臣”，“臣”象豎目，表示以戈刺傷眼睛（楊樹達、李孝定）。古代戰爭中擒獲的俘虜，往往用戈刺盲他們眼睛，使其淪爲奴僕。古書稱戰爭所獲的俘虜爲“臧獲”，後泛指奴僕，成爲僕人的賤稱。《方言》：“臧，奴婢賤稱也。荊、淮、海、岱、雜齊之間，罵奴曰臧。”

    楊樹達認爲戰敗者被獲爲奴，不敢橫恣，故“臧”引申有善、好義。《説文》：“臧，善也。从臣，戕聲。，籒文。”段玉裁注：“凡物善者必隱於内也。”古時“臧”又用來表示收藏，段玉裁認爲是因爲凡美善的才德，必蘊藏在內。一説可能戰爭所得的財寶必須“善而藏之”，故“臧”亦是“藏”的本字，這一點從《管子》、《漢書》、《呂氏春秋》，乃至近世出土文獻如雲夢睡虎地秦簡和馬王堆帛書中“臧”字的用法皆可解藏而可得證。《睡虎地秦簡·日書甲種》簡72背：“臧（藏）於草中”。

    “臧”字後來進一步發展，主要循著“藏匿”這一意義展開，其中，若轉注之以“艸”便產生收藏的“藏”（埋之草下），若注之以“貝”便得賊贓的“贓”（匿藏的不法財貨），若從藏累增之以“肉”便得臟腑的“臟”（藏之於體內）等，其孳乳浸多，可謂久遠。

    金文改爲從“臣”，“爿”聲，戰國文字或上承此形。戰國文字以從“戕”聲從“口”或“言”之字來表示“臧”。

    髒，從“骨”，“葬”聲。“髒”表示污穢、不清潔。

    旋（旋轉）、鏇（镟，鏇果皮） #

    鏇（镟）:〔xuàn / ㄒㄩㄢˋ〕，鏇果皮

    甲骨文金文小篆隶书楷书行书草书

    甲金文從“止”從“㫃”。字象人足隨旌旗指麾而轉動，本義是轉動、周旋。

    甲骨文“止”或改從“正”，“正”有遠行之義，其構形之義與從“止”同。《説文》：“旋，周旋，旌旗之指麾也。从㫃从疋。疋，足也。”徐鍇《繫傳》：“人足隨旌旗也。”徐中舒、劉興隆釋字象人於旌旗下周旋之意。另，段玉裁《説文解字注》：“旗有所鄉（嚮），必運轉其杠，是曰周旋”，馬如森亦認爲字象人持旗行走以周旋。徐灝《説文解字注箋》：“旌旗可以齊眾，執以指麾，令士卒望而前却，非運轉其杠之謂也。”按“旋”似沒有人持旗運轉之義，段注似不可從。

    㫃〔yǎn / ㄧㄢˇ〕，甲骨文，金文。甲金文象旗在竿上飄游之形，是“旂”“旗”的初文。

    鏇〔xuàn / ㄒㄩㄢˋ〕，《説文》：“圜鑪也。从金，旋聲。辭戀切。”

    ① 圓形的爐子。《説文解字·金部》：“鏇，圓鑪也。”

    ② 溫酒器。如：“金鏇”、“酒鏇”。元·戴侗《六書故·卷四·地理》：“鏇，溫器也，旋之湯中，以溫酒與洎者也。”也稱爲“鏇子”。

    ③ 銅錫盤。明·馮夢龍《古今譚概·妖異部·鏇中佛像》：“常熟丘郡家食櫥內，錫鏇置熟雞半隻。”

    ④ 用鏇子溫酒。《水滸傳》第七二回：“你自去與我鏇一杯熱酒來吃。”

    ⑤ 以機器或刀旋轉著削切。如：“鏇陀螺”、“鏇螺絲釘”、“把蘿蔔皮鏇下來”。

    “旋”讀四聲時的意義爲：

    ① 打轉的。如：“旋風”、“旋球”。

    ② 溫酒。通“鏇”。《水滸傳》第五回：“那莊客旋了一壺酒。”

    ③ 表動作是臨時、現時的。唐·杜荀鶴《山中寡婦》詩：“時挑野菜和根煮，旋斫生柴帶葉燒。”元·喬吉《滿庭芳·活魚旋打》曲：“活魚旋打，沽些村酒，問那人家。”

    簡化字中亦有“镟”這個字，《漢典》中釋爲同“旋”，但“旋”字釋義裏只有“用车床或刀子转着圈地削”一項與“鏇”有關。

    從臺灣的字典以及《王力古漢語字典》看，只有“溫酒”或“溫酒器”一義“旋”通“鏇”。

    遊（游，遊行、旅遊、遊戲）、游（游泳） #

    甲骨文金文小篆隶书楷书行书草书

    斿，甲金文從“㫃”從“子”，或不從“子”而從“人”，象小孩子執旌旗出行之形。本義爲出遊，是“游”、“遊”的初文。

    東周金文於“斿”字增“彳”、“辵”、“水”，“遊”爲遨遊之專字，“游”是表示在水中游。

    字義分化後，“斿”用來表示古時旌旗末端直幅、飄帶一類的下垂飾物，可能因爲這些下垂的穗帶隨風飄揚，有浮游之貌。《説文》無收“斿”字，商承祚認爲《説文》以“游”爲旗之流者，借字也。旌旗之游應作“斿”，俗作“旒”。《玉篇》：“斿，旌旗之末垂者。或作游。”《集韻》：“旒，旌旗之旓。或作斿。”“斿”即旗幟上飄帶之類的裝飾物。

    游， 從“水”，“斿”亦聲，“游”是“斿”的後起字，春秋後增“水”旁（見金文第二例），表示在水中游泳，引申有出游、游樂之意。

    金文表示游樂，莒弔之仲子平鐘：“莒弔（叔）之中（仲）子平自乍（作）鑄其游鐘”，表示叔之仲子平鑄造供游樂的鐘。

    現在，“游”只表示游泳等與水有關的意義，如“游泳”“上游”“下游”，其他義項由“遊”字表示，如“遊戲”“遊樂”。

    遊，從“辵”，“斿”亦聲。甲骨文及早期金文寫作“斿”，象小孩執旌旗出遊。“遊”是“斿”的後起字，春秋以後增“彳”（象道路）或“辵”（象腳在路上走）（見金文第三例），表示出遊。

    金文表示遠遊，中山王鼎：“寡人專賃之邦而去之遊”，表示寡人把國家交托（給司馬賈），而去遠遊。

    矇〔méng / ㄇㄥˊ〕：① 失明者，盲人。② 蒙昧。《論衡·量知》：“人未學問曰矇。”

    《王力古漢語字典》沒有〔mēng / ㄇㄥ〕這個讀音，可能是現代漢語出現的讀音。臺灣字典解釋如下：

    矇〔mēng / ㄇㄥ〕：① 欺騙。如：別説瞎話矇人。② 胡亂猜對或得到。如：我能得獎是矇上的。

    濛〔méng / ㄇㄥˊ〕：① 微雨貌。《詩·豳風·東山》：“我來自東，零雨其濛。”② 【濛濛】1、雲雨密貌。2、不分明貌。③ 疊韻聯綿字“濛澒”。未分的元氣。又作“濛鴻”或“鴻濛”。

    懞〔méng / ㄇㄥˊ〕：慤厚貌。見《集韻》。《管子·五輔》：“敦懞純固，以備禍亂。”

    懞〔měng / ㄇㄥˇ〕：疊韻聯綿字“懞懂”。昏昧，糊塗（晚起義）。亦作“懵懂”。

    “放懞〔méng / ㄇㄥˊ〕掙”是個北方方言，任性、發脾氣的意思。

    “蒙”和“矇”又是一對從不同角度描述相似意義的字。在蒙蔽、無知一義上二字有重疊。前者是由覆蓋引申爲蒙蔽、無知；後者是因看不見（蒙住眼睛）而引申爲矇蔽、無知。

    臺灣字典中，“蒙蔽”和“矇蔽”、“蒙昧”和“矇昧”、“蒙混”和“矇混”都是並列出現的，意義基本一樣，只不過有些小差異，也都是因爲“蒙”和“矇”有小差異導致的。例如，“矇昧”在某古文中有“眼睛看不清”的意思之類。很自然的，特指眼睛不明導致的“蒙”自然會用“矇”字。

    當然，也有形式一樣，但意義不同的詞，比如，“蒙住”和“矇〔mēng / ㄇㄥ〕住”就不一樣，前者是用東西覆蓋，後者是呆住或被騙的意思。

    還有一些詞，應該也是並列的意義，但是臺灣那個教育部國語辭典中沒有，比如，“蒙騙”“欺蒙”，只有“矇騙”“欺矇”（前者還給注音〔méng piàn / ㄇㄥˊ ㄆㄧㄢˋ〕 ，雖然現在是這麼發音，但〔méng piàn / ㄇㄥˊ ㄆㄧㄢˋ〕 不可謂不是一個詞吧）。

    雖然“矇”是人未學，但估計只能單用（或某篇文章的術語）的時候表示人未學，不能用於“童蒙”。“童矇”給人感覺是眼睛不好。

    另外，還要注意“矇矓”“濛濛”“矇矇”“蒙蒙”等詞的意義。

    矇矓：① 眼睛欲閉又張的樣子。如：睡眼矇矓。② 模糊、不分明。如：淚眼矇矓。

    朦朧：① 月色昏暗的樣子。如：月朦朧。② 不清楚、模糊。煙霧朦朧。兩眼朦朧（其實和矇矓通用）。

    曚曨：形容太陽初現時，光線暗淡的樣子。杜甫《西枝村尋置草堂地夜宿贊公土室》詩二首之二：“晨光稍曚曨，更越西南頂。”

    其實都是〔méng lóng / ㄇㄥˊ ㄌㄨㄥˊ〕，只是用不同的意符體現特定的指向。所謂的聯綿詞都是這樣。像“曚”“曨”這種字也只用於“曚曨”了。也有個“瀧”字（雨瀧瀧貌），估計是沒有人寫細雨濛瀧吧，所以沒有“濛瀧”一詞，呵呵。

    濛濛：① 形容水氣綿細密布的樣子。細雨濛濛，霧濛濛。② 迷茫不清的樣子。白居易《江夜舟行》：“煙澹月濛濛，舟行夜色中。”岑參《與高適薛據登慈恩寺浮圖》：“五陵北原上，萬古青濛濛。”

    矇矇：昏暗不明。引申爲不明理、不開通。《文選·班固·幽通賦》：“昒昕寤而仰思兮，心矇矇猶未察。”臺灣字典中説“天矇矇亮”也是用這個“矇”。那爲什麼不用“曚”呢？可能是這個字不常用。

    朦朦：① 迷惘貌。漢·應劭 《風俗通·愆禮·河南尹太山羊翩祖》：“翩祖位則亞卿，雅有令稱，義當綱紀人倫，爲之節文，而首倡導犯禮違制，使東嶽 一郡朦朦焉，豈不湣哉！”② 微明貌。月色朦朦。

    蒙蒙：① 昏暗不明。《漢書·卷一〇〇·敘傳上》：“昒昕寤而仰思兮，心蒙蒙猶未察。”此昏暗不明與日、月、目無關，當然也可能〔méng lóng / ㄇㄥˊ ㄌㄨㄥˊ〕 就是個音而已，聯綿詞只要音同，其實用哪個字無所謂。② 萌生。《大戴禮記·夏小正》：“日冬至，陽氣至始動，諸向生皆蒙蒙符矣。”③ 形容多而密布的樣子。《楚辭·東方朔·七諫·自悲》：“何青雲之流瀾兮，微霜降之蒙蒙。”

    懞懞：迷迷糊糊。《二刻拍案驚奇》卷三：“孺人疼痛既止，精神疲倦，懞懞的睡了去。”

    蒙、矇、濛、懞這幾個字其實不應該合併。雖然前兩個字在蒙蔽、無知一義有重疊（只是在雙字詞中），但其他幾個字意義是很分明的，以至於電視劇《情深深雨濛濛》在大陸播出時，沒人知道按規定應該寫成《情深深雨蒙蒙》。

    至於“矇”字，由於它畢竟有個“目不明”的義項，不應將這個意義併入“蒙”，多個目字旁沒有什麼壞處。“矇”還可以作爲〔mēng / ㄇㄥ〕這個讀音的專字，什麼“瞎矇”“矇人”“矇事”“矇矓”“天矇矇亮”等詞比寫作“瞎蒙”“蒙人”“蒙事”“蒙眬”“天蒙蒙亮”要好。

    “蒙”就讀作〔méng / ㄇㄥˊ〕，減輕了負擔。“矇”就是目不明，引申出“瞎矇”等詞。本來挺清楚的事，爲什麼要搞含糊？

    黴（霉，黴菌、發黴）、霉（倒霉） #

    > 把微下面的几替換成黑。

    黴：《説文》：“中久雨青黑。从黑﹐微省聲。武悲切。”就是發黴、黴菌的黴。

    霉：晚起字。同“黴”。衣服因潮溼而變質。《正字通》：“霉，莫裴切。音枚。項甌東曰：江南以三月爲迎梅雨，五月爲送梅雨。或言古語：黃梅時節家家雨，故云。張蒙溪謂：梅當作霉，雨中暑氣也。霉雨善汙衣服，故又云霉涴，言爲霉所壞也。按《埤雅》風土記皆作梅雨。霉義與黴通，存備考正。舊本闕。”

    現在，臺灣的字典，“黴”主要用於黴菌義，例如，青黴素、黴菌病。雖然字義中有“因發黴而敗壞。如：黴壞。”這個義項，但，沒有“發黴”“黴爛”等與“因發黴而敗壞”相關的詞。這個意義都用“霉”，例如，霉爛、霉溼、霉味、發霉。

    另外，“黴”還有個“面垢黑色”的義項，不能由“霉”代替。例如，《淮南子·脩務》：“神農憔悴，堯瘦臞，舜黴黑，禹胼胝。”，不是因發黴而黑。還有“黴黧”，臉上汙黑。《楚辭·劉向·九歎·逢紛》：“顏黴黧以沮敗兮，精越裂而衰耄。”

    “霉”字也被用作“倒霉”“霉運”這種詞。據《百度百科》，“倒霉”起源于封建时代，读书人要做官，都要参加科举考试。明朝时，考试录取很难，如果录取了，就在门前立旗杆一根；如果考不中，就把旗杆倒下拿走，称为“倒楣”。“楣”本是门上的横木，这里指高杆。“楣”与“霉”读音相近，江浙一带的人就把遇事运气不好、不吉利叫“倒楣”亦即“倒霉”了。

    綜上所述，“黴”有兩個義項，“面垢黑色”和“受潮而產生的黑斑”，古人不知道黴菌，現代人知道，因此，黴菌、發黴等詞自然用“黴”字。“霉”是元明以後出現的字，可能最初就是附會“梅雨”的“梅”，或者説就是爲了區分而另造了一個字。由於雨天發黴，因此，“黴”“霉”並行，都表示發黴。而“霉”字更通俗，所以可能“霉”用得更多，尤其是近代新出現的詞，比如倒霉，會首先使用“霉”而不是“黴”。這也是臺灣目前出現這種用字情況的原因。至於爲什麼黴菌等生物學、醫學用詞使用“黴”，不知道是這些詞彙來源更講究，從開始就用這個字，還是後來臺灣的規定。

    挽（挽手、挽留）、輓（挽，推輓、輓夫） #

    挽：① 拉，牽引。《説文》作“輓”，云：“引之也。”② 捲起。如：挽袖子。③ 哀悼死者。後來寫作“輓”。《新唐書·承天皇帝倓傳》：“（李）泌爲挽詞二解，追述倓志，命挽士唱。”

    輓：① 拉車，牽引。亦作“挽”。② 用車運輸。《漢書·韓安國傳》：“又遣子弟乘邊守塞，轉粟輓輸。”又泛指運輸。③ 悼念，哀悼死者。亦作“挽”（後起義）。④ 通“晚”。“輓近”最近若干年來。《史記·貨殖列傳》：“必用此爲務，輓近世塗民耳目，則幾無行矣。”司馬貞《索隱》：“輓音晚，古字通用。”

    “挽”“輓”在牽引義和哀悼義上通用，但“挽手”不作“輓手”，“挽留”不作“輓留”，因爲與車無關；“挽袖子”也不作“輓袖子”，因爲“輓”沒有捲起義。“挽”字不見於《説文》，可理解爲與車有關的牽引義當作“輓”，一般意義的“拉”“牽引”用“挽”。因此，“挽手”“挽留”用“挽”。“輓”表示哀悼死者是魏晉以後出現的，最初用“挽”。

    “推輓”就是“推拉”；“輓夫”就是“縴夫”。

    才（才能）、纔（才，剛纔） #

    > 由糸㲋兔組成。
    甲骨文金文小篆隶书楷书行书草书

    “才”字《説文》指種子破土而出。陳劍認爲“才”、“弋”本一字，象木橛，在古文字中借來表示“在”，指所在、存在，引申爲剛才。

    現代學者多認爲甲金文“才”象木樁、橛杙之形。當代方家多認爲甲金文“才”與“弋”（“杙”的初文）（甲，金）是一字之分化，“弋”字省去右上的一小筆便是“才”字。象木樁、橛杙之形（陳劍）。

    “才”字後來的發展，是被借來表示存在動詞和介詞，並疊加“士”爲聲符而造“在”字。從認知上看，一種子的生長或一本樁的豎立必有一固定的地點，故“才”字乃得以引申爲“表示行爲所涉及的處所、時間、範圍，以及對象等”意義（參徐中舒《甲骨文字典》）。

    《王力古漢語字典》中，“才”有三個意思，① 才能。② 通“材”。資質，品質。③ 通“纔”。副詞。方始，僅。

    《王力古漢語字典》對“纔”的釋義如下：

    纔〔cái / ㄘㄞˊ〕：副詞。後作“才”。① 剛才、剛剛，表示時間短暫。《漢書·鼂錯傳》：“救之，少發則不足；多發，遠縣纔至，則胡又已去。”② 僅僅，表示數量少，程度輕。晉·陶淵明《桃花源記》：“便捨船，從口入，初極狹，纔通人。”

    纔〔shān / ㄕㄢ〕：淺青，微黑。《説文》：“纔，帛雀頭色。一曰微黑色，如紺。纔，淺也。讀若讒。从糸、兔、毚（〔chán / ㄔㄢˊ〕）、讒（谗）、攙（搀）、饞（馋）、纔（才，剛纔）)聲。七咸切。”《廣韻》所銜切。

    朮〔shú / ㄕㄨˊ〕：① 同“秫”。② 姓。

    朮〔zhú / ㄓㄨˊ〕：草名。菊科朮屬植物的泛稱。多年生草本。有白朮、蒼朮等數種。也作“”。《爾雅·釋草》：“，山薊，楊枹薊。”《集韻·術韻》：“，草名。《説文》：‘山薊也。’或作朮。”

    虫（虺初文）、䖵（昆蟲的昆本字）、蟲（虫）、蠱（蛊） #

    • 虫:〔huǐ / ㄏㄨㄟˇ〕，虺初文
    • 䖵:〔kūn / ㄎㄨㄣ〕，昆蟲的昆本字
    甲骨文虫金文 虫小篆 虫隶书楷书 蟲行书 蟲草书 蟲

    虫〔huǐ / ㄏㄨㄟˇ〕：甲金文象蛇形，本義是毒蛇。《説文》：“虫，一名蝮。博三寸，首大如擘指，象其臥形。”甲金文“虫”和“它”皆象一條蛇，其分別是“它”的字形象身體的部分比較粗。甲骨文的“它”身上有花紋，金文或簡化爲一豎。而“虫”的蛇身比較小，二字毫不相混。（裘錫圭）

    馬王堆帛書《老子》乙本第16-17行：“（蜂）（蠆）虫蛇弗赫（螫）。”後來孳乳了“虺”表示蝮蛇之義，傳世王弼本《老子·第五十五章》作“蜂蠆虺蛇不螫”。

    《集韻》：“蟲，……俗作虫。”徐灝《説文解字注箋》引戴侗曰：“蟲或爲䖵、或爲虫者，從省以便書。”

    “虫”作偏旁時往往泛指昆蟲。

    䖵〔kūn / ㄎㄨㄣ〕：甲金文從二“虫”。《説文》以爲“䖵”的本義是蟲類的總稱，亦即昆蟲的“昆”的本字。

    甲骨文用作神祇名，也表示族名或地名；金文疑用與“虫”同，表示蛇類。

    “䖵”用爲偏旁時，意義或同“虫”，或同“蟲”。（參季旭昇）

    蟲：從三“虫”，會蟲之總名。《説文》以爲“蟲”是有足動物的總名。

    “蟲”和“䖵”都是重複同一偏旁“虫”而成的會意字。甲金文多見從“䖵”的字，少見從“蟲”的字。從“蟲”的字或從“䖵”，如“蠱”。楚文字中獨立的“蟲”和作偏旁的“蟲”都有“䖵”（昆）的音。（參陳劍）例如郭店簡《老子》甲本簡21“又（有）[爿頁]（狀）蟲（混）成”。劉釗認爲古文字中相同偏旁從二個或三個經常無別，“蟲”可看成是“䖵”的繁體。季旭昇認爲“蟲”和“䖵”實本同義。

    《説文》以爲“蟲”是有足動物的總名。《説文》：“蟲，有足謂之蟲，無足謂之豸。”此説或本於《爾雅》。《爾雅·釋蟲》：“有足謂之蟲，無足謂之豸。”邢昺疏：“此對文爾，散文則無足亦曰蟲。”故“蟲”可以是一切動物包括人的通稱。如《戰國策·秦策二》：“虎者，戾蟲；人者，甘餌也。”《大戴禮記·曾子天圓》：“毛蟲之精者曰麟，羽蟲之精者曰鳳，介蟲之精者曰龜，鱗蟲之精者曰龍，倮蟲之精者曰聖人。”

    《説文》：“願，大頭也。从頁，原聲。魚怨切。”《王力古漢語字典》列爲備考，可能是古籍中未見用例。

    聖（圣，聖人）、圣（義同掘） #

    圣:〔kū / ㄎㄨ〕，義同掘

    大陸下面寫成王即可。

    甲骨文金文小篆隶书楷书行书草书

    甲骨文從人從耳從口，象人形而突出耳朵，旁邊有口，象音聲出之口，而入之於耳。本義爲聽聞，由聽聞廣博會通達之意，進而引伸爲聖賢、聖德。吳大澂云：“聖，聲也，通也，聞聲知情謂之聖，聖、聲古通。”古“聖”、“聲”、“聽”三字同源，形近義通，皆從口、耳會口言而耳聽之之意，而金文多以“聖”字表聽聞之義。早期金文與甲骨文相同，西周中期以後“聖”字多從耳從口從“〔tǐng / ㄊㄧㄥˇ〕”聲。耳下人形漸訛爲“”，象人挺立於土上。耳上或又加“屮”，可能受“聲”之甲骨字形影響，參“聲”。聖亦有省人形作；或於下加兩點兩畫之飾筆；或從人從耳從兩口，與“聽”甲文形近；皆無損聽聞之義，參“聽”。

    ，甲骨文，小篆。從人從土，甲骨文象人立於土上之形，音“挺”，當爲“挺”之古文。一説象挺立有所企求之形；一説人直立地上形容端正，故訓善也。甲文所見只用爲人名及祭名。《説文》：“善也，从人、士。士，事也。一曰物出地挺生也”徐鉉曰：“人在土上，壬然而立也，他鼎切。”金文“”只用作部件，構形與甲文同，本義爲人站立於土上。另“”與“壬”不同，宜小心區別。

    壬，甲骨文，金文（商）（西周中期），小篆。干支名，獨體字，構形初義不明。早期甲金文“壬”作“工”形，後豎畫中間添一點，又伸展爲橫畫，而成“壬”字。林義光認爲“工”形即“榺”（〔shèng / ㄕㄥˋ〕，古稱經軸或繞經輥爲榺。是古代織機上捲放經紗的工具。有時也稱作柚，今作軸。）之古文，並云：“機持經者也，象形。”高鴻縉從林説。一説“壬”爲兩刃之斧（吳其昌），與“辛”義近。

    辛，甲骨文，金文。“辛”象一個鑿形刀具（詹鄞鑫），用來伐木、施刑等。如“亲”字上從“辛”下從“木”，象用刀鑿（“辛”）砍樹（“木”），以獲取柴薪，是“薪”的初文，“辛”亦標聲。郭沫若認爲“辛”是施黥刑的工具，“童”、“妾”等字從之。“辛”用作偏旁，又增繁筆畫，寫作“丵”，如“鑿”字從“丵”，表示鑿子。從“辛”（丵）取義的字皆與刀鑿、刑罪等義有關；背負刑罰，則心情悲痛，故“辛”又有辛苦、辛酸之義；而味道辛辣，會使人流淚或鼻水，甚至涕泗俱流，而痛苦時也會流淚，所以辛辣一義用表示痛苦、艱辛的“辛”字來表示。

    圣〔kū / ㄎㄨ〕：《説文》：“汝潁之閒，謂致力於地曰圣。从土从又。讀若兔窟。苦骨切。”古代方言，義同“掘”。清施補華《别弟文》：“吾負母而逃，圣野菜充飢。”據百度百科，圣始見於戰國，本義指用手挖土，後作爲“聖”的簡化字。今魯西一帶仍然稱用力刮挖硬物爲“圣吃圣吃”。

    幺（△么，幺妹、幺麼小丑），麼（么，什麼） #

    幺〔yāo / ㄧㄠ〕：金文。應該就是“糸”去掉下面的線緒。《説文》：“幺，小也。象子初生之形。凡幺之屬皆从幺。於堯切。”

    “么”是“幺”的俗字，其實是一個字，但臺灣“么”“幺”並存。“幺”只表示細小義；“么”除表示細小外，還表示“一”的別稱、姓氏、排行最小的（么妹、么叔）等義。其實，“么”就是“幺”。注意，只有〔yāo / ㄧㄠ〕這一個讀音。

    麼〔mó / ㄇㄛˊ〕：① 細小。《文選》漢·班彪《王命論》：“又況幺麼不及數子，而欲闇干天位者也。”李善《注》引《通俗文》曰：“不長曰幺，細小曰麼。”《列子·湯問》：“江浦之間生麼蟲，其名曰焦螟。”② 代詞，猶這麼、那麼。宋·黃庭堅《南鄉子》：“萬水千山還麼去，悠哉！”

    麼ma：語氣詞，表示疑問。忘了麼？你的麼？

    “幺”“麼”裏的“幺”都可寫作“么”，其實是一個字，可能是書法異寫或者就是俗字。

    迹、※跡、※蹟（奇蹟、遺蹟） #

    迹，金文，小篆。金文從“辵”，“朿”聲，與《説文》籀文同。《説文》小篆從“亦”聲，是“朿”的訛變。漢以後基本承襲從“辵”，“亦”聲的寫法。“迹”亦作“蹟”“跡”。

    跡，同“迹”。“足”和“辵”是義近形符。

    蹟，從“足”，“責”聲。《説文》以爲“蹟”是“迹”的或體。“責”從“朿”聲。“迹”本作“”，亦從“朿”聲。《説文》：“迹，步處也。从辵，亦聲。蹟，或从足、責。，籀文迹从朿。”《字彙·足部》：“蹟，與迹同。”

    現在，痕跡、足跡、跡象用“跡”（大陸用“迹”）；奇蹟、名勝古蹟、事蹟用“蹟”。“蹟”表示前人留下來的事物，多指建築、文物而言。

    胡（胡人）、鬍（胡，鬍鬚） #

    甲骨文金文三晋文字小篆隶书楷书行书草书

    金文從“肉”，“古”聲，本義疑爲牛自下巴至頸下垂的肉。引申泛指動物下巴垂下來的肉。由於動物下巴多有鬚，故頷下的鬚也可以稱“胡”，後來才加“髟”分化出“鬍”，專門表示鬍子。《史記·孝武本紀》：“鼎既成，有龍垂胡髯，下迎黃帝。”“胡髯”即頷下的鬚。《中華大字典》：“胡，俗謂鬚曰胡子”，“胡子”當是“鬍髭”的記音（參王鳳陽）。

    • 舍:〔shè / ㄕㄜˋ〕，房舍

    舍：金文。金文“舍”所從的“余”除了表聲，還象簡易的房屋，“亼”爲屋頂及屋樑，“屮”爲木柱（陳獨秀、楊桓、馬敍倫），“口”除了可能是分化符號之外，還可能象房屋的下基。本義是房舍。于省吾、季旭昇認爲“舍”字是借“余”的字形加“口”分化出來。

    金文通假作“予”，表示給予、賜予，令鼎：“余其舍女（汝）臣十家。”引申表示施發、發布，善夫克鼎：“王令善夫克舍令（命）于成周遹正八（師）之年”，“舍令”即發號施令。《詩·鄭風·羔裘》：“彼其之子，舍命不渝”，意謂發號施令無所渝失（于省吾）。又通假作“余”，第一人稱代詞，中山王鼎：“含（今）舍（余）方壯”。又用作人名。又通假爲“娛”（谷建明、魏宜輝），巢編鐘：“永寶是舍（娛）”，意謂永遠珍重此鐘，以供娛樂。

    “捨”是“舍”的分化字，表示放下、捨棄。《説文》：“捨，釋也。从手，舍聲。”

    古時候用“捨”的地方也有用“舍”的，什麼時候區分的比較開了，我也沒查到。現在港臺地區是區分的比較開的，大陸也認爲“捨”合併簡化爲“舍”。

    嘗（尝，嘗試）、※嚐（尝，品嚐） #

    甲骨文金文小篆隶书楷书行书草书

    金文从“旨”，“尚”声，“旨”是人口所好，有甘美之意，“嘗”从“旨”有品尝、试味之意，引伸为尝试。

    早期金文从“冂”（上有两短横，为饰笔），从“旨”，“冂”是“堂”、“尚”的初文，作为“嘗”的声符，后加“口”为从“尚”。

    个人认为后世书家多爱写作从甘，只是为了字形好看。能被称为书家的人自然会写篆书，自然知道从旨，只是旨和甘形近义近写出来又好看，因此多爱写作甘。甘的长横使这个字不显得瘦长，楷书第二例即便从旨，也将匕变形写出长横。

    “嚐”是“嘗”表示本義的分化字，又加上“口”形表義。本義是以口辨別滋味。“嚐”出現的時代較晚，例子如唐代唐彥謙《蟹》：“充盤者熟堆琳琅，橙膏醬渫調堪嚐。”《水滸全傳》第75回：“阮小七叫上水手來，舀了艙裏水，把展布都拭抹了，卻叫水手道：‘你且掇一瓶御酒過來，我先嚐一嚐滋味。’”

    僵（僵硬）、殭（僵，殭屍） #

    僵：從“人”，“畺”聲，本義爲倒下。《説文》：“僨也。从人，畺聲。居良切。”

    ① 向後倒下。《呂氏春秋·貴卒》：“管仲扞弓射公子小白，中鉤，鮑叔御公子小白僵。”高誘注：“僵，猶偃也。”

    ② 不活動。《靈樞經·癲狂》：“癲疾始作，先（而）反僵，因而脊痛。”

    詞例：僵持、僵局、僵硬、僵固、僵住、死而不僵。

    殭：《説文》無。

    ① 死而不朽。《玉篇》：“殭，死不朽也。”引申爲僵硬。唐·盧仝《月蝕》：“森森萬木夜殭立。”

    ② 仆倒。《三國志·魏書·武文世王公傳》評裴松之注引《魏氏春秋》：“故語曰：‘百足之蟲，死而不殭。’以扶之者眾也。”用作使動。《潛夫論·忠貴》：“掊死深穽，銜刀都市，殭屍破家，覆宗滅族者，皆無功於民氓者也。”

    個人以爲：二字基本通用，死而不朽這種意義，或者説與死沾邊的意義可能多用“殭”，因爲它有個“歹”字旁。

    “死而不殭”也可作“死而不僵”。臺灣字典裏有“死而不僵”，無“死而不殭”，且“殭”只有“死而不朽”義，詞例有“殭屍”“殭尸”“殭蠶”（殭屍和殭尸同），“殭屍”亦可作“僵屍”“僵尸”。

    周（周圍）、※週（週期、週日） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文象刻有花紋之玉器。朱芳圃以爲字象方格縱橫，刻劃文采之形，當爲“彫”、“琱”之初文。本義是彫琢玉器。

    “周”字甲骨文釋義有二：一説以爲象田疇之貌（魯實先），一説則以“周”爲“彫”、“琱”之初文，字象有彫飾的玉器（孫常敘），《説文》：“琱，治玉也。一曰石似玉。从玉周聲。”按因“周”字爲“實”“畫”等字的部件，似乎不能理解爲田疇，且金文中“周”有用作“琱”之本義的，故這裏採用後一種説法。“周”因借用作國名，所以金文又在字形下方別加“口”形，甲金文國名之字往往加“口”作分化符號。西周晚期字形上部漸與“用”字同形，戰國文字多承此而來，小篆亦因此誤從“用”，《説文》：“周，密也。从用、口。”

    週：從“辵”，“周”聲。“週”是“周”的後起字。《正字通》：“週，俗周字。”“週”表示周圍、四周。如唐代顏真卿《湖州烏程縣杼山妙喜寺碑銘》：“唐太宗文皇帝升極之六年春二月，移於此山，山高三百尺，週迴一千二百步，蓋昔夏杼南巡之所。”《水滸全傳》第19回：“原來這石碣村湖泊，正傍著梁山水泊，週 圍盡是深港水汊，蘆葦草蕩。”又表示環繞。如清代葉夢珠《閱世編·建設》：“方廣百餘步，外週土垣，內列含倉廩。”

    “週”表示周期。如唐德宗《南效赦文》：“邊軍鎮守及諸道諸軍防秋將士，經三週年未改轉者，宜與甄敘。”元代關漢卿《趙盼兒風月救風塵》第一折：“娶到他家裏，多無半載週年相棄擲。”

    “週”也表示遍、全、滿。如唐代嵩嶽諸仙《嫁女詩》：“人間甲子週千歲，靈境杯觴初一巡。”元代無名氏《無俗念》：“變化無窮，包含不盡，運用無蹤跡。虛空打破，光明遍週無極。”

    “週”還可作量詞。如清代文康《兒女英雄傳》第39回：“老爺看了一週只不曾見著他家那位姨奶奶；纔要問起，還要問問他家今日到底是有件甚麼事。”又表示星期，如“本週”、“下週”。

    升、昇 #

    升，甲骨文，金文，小篆，甲金文从“斗”从一点，“斗”是用来舀水的长柄勺子，“升”象用斗盛酒或粮食，以便献祭神祇，引申为一切升进之意，又用为容量单位。甲金文“升”“斗”形近，两字只差一点，这小点是所盛载的酒或粮食。有点的是“升”，无点的是“斗”。“斗”大于“升”，一斗的液体或粮食，倒在升里，便会满潟、满溢出来，反映“升”比“斗”更小、更容易装满，甲骨文“升”字旁边的小点便象满溢出来的液体或粮食，所以用有一点的字形表示“升”，没有一点的表示“斗”（孙雍长）。

    ① 容积单位。② 上升，登。《诗·小雅·天保》：“如月之恒，如日之升。”③ 通“登”。收成。《吕氏春秋·孟夏纪》：“农乃升麦。”《礼记·月令》作“农乃登麦”。《谷梁传》襄公二十四年：“五谷不升为大饥。”

    昇，《说文》无，《说文新附》有。① 太阳上升。“日照水而东昇”② 登上。汉·王逸《九思·哀岁》：“昇车兮命仆，将驰兮四荒。”③ 升官，晋级。④ （昇平）太平。

    个人认为升包含昇的全部义项，昇实为升的异体或始终未彻底分化的分化字，由于加从日而在某些场合使用。

    栖、棲 #

    棲，本作“西”，表示月已西落，鸟已入巢。战国秦系文字改为从“木”，“妻”声，作“棲”。本义是鸟类歇宿。“棲”为《说文》“西”字之重文。

    西，甲骨文有两种字形，一种象鸟巢之形，本义是鸟巢。日已西落，鸟已入巢，故又是“棲”的初文（罗振玉）。后假借为东西方的西。一说以“西”名日落之方乃引申用法，而非纯粹假借，理由在于“西”原为鸟之归巢，由此联想到夕阳西下之时，故名日落之方曰西（杜学知）。另一种是“囟”的假借，“囟”“西”古声同韵近，故得相借，后逐渐演变成西形（张玉金、季旭升）。

    据《王力古汉语字典》和台湾国语辞典，“棲”和“栖”同字。《说文》无“栖”字。不知“栖”字最早见于何处，应用是否普遍。

    佔（占，佔領、佔比）、占（占卜） #

    • 佔（占）:〔zhàn / ㄓㄢˋ〕，佔領、佔比
    • 占:〔zhān / ㄓㄢ〕，占卜

    占，甲骨文。甲骨从“卜”从“口”，或从“”从“口”，会观看卜兆而判断吉凶之意。甲骨文先有从“卜”从“口”之形，后来有从“”从“口”之形，“口”形或写于“”形之内，或写在“”形之下，或有把“口”形简化为一撇者（王国维、裘锡圭）。从从口之字用法相同，多用于占辞之前。

    占〔zhàn / ㄓㄢˋ〕：① 据有（后起义）。这个意义后来写作“佔”。② 估计。《汉书·宣帝纪》：“流民自占八万余口。”③ 口授。《后汉书·卷四十五·张俊传》：“俊自狱中占狱吏上书自讼，书奏而俊狱已报。”【口占】不用起草而随口成文。《汉书·朱博传》：“阁下书佐入，博口占檄文。”

    從“卩”，“龹”聲。本義是大小腿相連關節的後邊可彎曲之處，與“厀”（膝）相對。《説文》：“卷，厀曲也。从卩，龹聲。”

    （龹）〔juàn / ㄐㄩㄢˋ〕：飯糰。《説文》：“摶飯也。从廾，釆聲。釆，古文辦（此處可能有誤，應爲辨，見後面説明）字。讀若書卷。居券切。”

    釆〔biàn / ㄅㄧㄢˋ〕：金文。金文“釆”構形初義不明。一般解作“辨”，亦有謂是“播”之初文。張世超等認爲“釆”從“力”從數點，數點象穀粒種子，會翻土播種之意，是播種的“播”的初文。後增從“田”“手”作“播”。姑備一説。《説文》：“釆，辨別也。象獸指爪分別也。凡釆之屬皆从釆。讀若辨。，古文釆。”

    我查到的字所引的《説文》都將“釆”説成古文“辦”（办）（看《説文》影印版也是如此），可能是把中間的“刀”誤寫成“力”了。

    卷〔quán / ㄑㄩㄢˊ〕：曲。《詩·大雅·卷阿》：“有卷者阿，飄風自南。”《莊子·逍遙遊》：“卷曲而不中規矩。”引申爲髮曲。《詩·陳風·澤陂》：“有美一人，碩大且卷。”又《小雅·都人士》：“彼君子女，卷髮如蠆。”這個意義也作“鬈”。《詩·齊風·盧令》：“其人美且鬈。”《集韻》：“鬈，《説文》髮好也，引《詩》：‘其人美且鬈。’一曰髮曲。”

    卷〔juǎn / ㄐㄩㄢˇ〕：捲起來。《詩·邶風·柏舟》：“我心匪席，不可卷也。”這個意義後來寫作“捲”。

    卷〔juàn / ㄐㄩㄢˋ〕：書分爲若干部分，每一部分爲一卷。引申爲考卷、試卷。

    捲〔juǎn / ㄐㄩㄢˇ〕：同“卷”。把東西捲成筒狀。北周庾信《詠畫屏風》詩：“玉柙珠簾捲，金鉤翠幔懸。”

    捲〔quán / ㄑㄩㄢˊ〕：通“拳”。拳頭。

    布（布料）、※佈（頒佈，佈告） #

    甲骨文金文小篆隶书楷书行书草书

    從“巾”，“父”聲。本義是麻織品。“布”引申爲散布、分布；鋪設、布置；宣告、宣布；陳述；布施、施予等義。

    書法中“布”字先寫撇再寫橫。因爲不管把上面當成“又”還是“父”都是先寫撇。這種字還包括：有、右、希、若等。“左”“在”是先寫橫。因爲“左”和“右”正好反著，“在”從“才”從“士”，“才”的筆順是先寫橫。

    佈：同“布”，表示遍布、安排、宣告等意思。《正字通·人部》：“佈，通作布。”除了表示麻織品外，“佈”和“布”通用。港臺多喜歡在引申義上用“佈”。

    采、※採 #

    采：甲骨文，金文。甲金文從“爪”從“木”，甲骨文或不從“木”而從“果”，象手在樹上採摘果實之形。“采”是“採”的初文，本義是採摘。

    採：港臺在采集義上多用“採”，丰采等義不用“採”。

    豆、※荳 #

    豆：甲骨文，金文。“豆”在古代原指盛醃菜、肉醬等食品的器皿，本義是高圈足的食器，後假借爲豆麥之“豆”。後來又爲豆麥之“豆”的假借義分化出“荳”字。不過，“豆”在“荳”字出現後仍被普遍用來表示豆麥之“豆”。

    注（注意）、※註（註解） #

    “注”是灌注，註释本灌注之引申义。六朝以后始有“註”作为专用字。《玉篇》：“註，解文意也。”《广雅·释言》：“註，疏也。”

    託（托，託付）、托（托盘） #

    “託”金文从“言”，“乇”声，用作人名。“乇”甲骨文，构形初义不明，用作“宅”“亳”的声符。

    託：① 嘱托，委托。② 寄托，依靠。③ 假托，推托。

    辨析託、侂、托。《说文》收有託、侂，均释作寄也。段玉裁注：“此（侂）与託音义皆同。俗作托，非也。”侂亦作仛。《广雅·释诂四》：“仛，依也。”託、侂实同一词。古籍中侂很罕用。《玉篇》引《论语》：“可以侂六尺之孤。”托是后起字，见《玉篇》。原用作以手承物义。“以手承物”本是依託、寄託的引申义，现“托”成为它的专用字。宋代以后，又兼用于“託”字各义。现託字简化为托。其实电脑字库里亦有个“讬”字，可能是未推行的简化字或只在必要时使用。

    咨、※諮（諮詢） #

    咨：① 咨询，商量。② 叹词。③ 公文之一种（后起义）。

    諮：同“咨”。咨询，跟别人商量。《国语·晋语》四：“询于八虞，而諮于二虢。”按，《说文》諮作咨。目前港台仍用諮。

    志（志向）、※誌（日誌） #

    志，金文。从“之”从“心”，之者往也，故志疑会心之所往之意，或言心意之所趣向，亦即志向。故“之”不只标声，亦施义符之职。① 志向。② 记载。《逍遥游》：“齐谐者，志怪者也。”引申为记事的书，如天文志、地理志等。③ 旗帜。《史记·刘敬叔孙通列传》：“设兵张旗志。”④ 通“痣”。

    誌，① 记述，记录。② 记事的文章或书籍。如墓誌、地方誌。③ 通“痣”。

    “誌”实际是“志”在记录和记事文两个意义上的分化字。

    贊（赞）、※讚（讚美、讚歎） #

    讚是贊的后起字。大陆认定为贊在部分意义上的异体，港台在部分意义上仍沿用讚。

    讚：① 称赞，赞美。② 辅佐，赞助。③ 古代一种以赞颂人物为主的文体。

    譁（哗，諠譁）、嘩（哗，嘩啦） #

    《王力古汉语字典》认为譁也作嘩。为声音嘈杂义。所谓諠譁即是声音既大又嘈杂。諠为声音大。现在嘩又为象声词嘩啦的嘩（〔huā / ㄏㄨㄚ〕），不可作譁（〔huá / ㄏㄨㄚˊ〕），读音亦不同。

    準（准，準備、標準）、准（准許、准將） #

    準，从水，隼声。本义为水之平。《説文》：“平也。从水，隼聲。” 段玉裁注：“謂水之平也。天下莫平於水，水平謂之準。”

    准，俗“準”字。唐宋以后公文用语，有比照、依据之义。“准此”即“照此”之义。后又用于允许义（準之后起义），多写作“准”。台湾异体字字典中说：“今已分用。‘准許’‘准據’等用‘准’；‘水準’‘準備’等用‘準’。”

    理论上，所有用“准”的地方都可以用“準”，但习惯上或实际中（公文中准此没有写作準此的）用“准”的情况如下：① 表示允许。准許、准予、不准。② 表示确定。准於某日上任。③ 表示比照、依据。准此辦理。④ 折算、相抵。關漢卿《竇娥冤·楔子》：“我有心看上他與我家做個媳婦，就准了這四十兩銀子，豈不兩得其便。”⑤ 类比、即将成为。准尉、准新娘。

    个人理解：与“水之平”意义十分相关的用“準”，看起来无关的意义用“准”。

    球、毬（球，繡毬） #

    球：① 美玉。② 玉磬。今借作毬，如足球，羽毛球。

    毬：一种游戏用具。今一般写作球。
