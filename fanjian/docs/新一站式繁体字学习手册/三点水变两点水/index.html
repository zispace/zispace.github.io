    三点水变两点水

    • 決（决）、沖（冲）、況（况）、減（减）、湊（凑）、盜（盗）、羨（羡）、淒（凄）、涼（凉）、淨（净）

    三点水变两点水 #

    決（决）、沖（冲）、況（况）、減（减）、湊（凑）、盜（盗）、羨（羡）、淒（凄）、涼（凉）、淨（净） #

    決：從“水”從“夬”，本義是打開缺口，疏通水道。

    沖：詳見沖。

    況：金文從“水”從“兄”，“兄”是聲符，用作人名，意義不詳。字書中的“況”表示寒涼。

    減：金文、小篆從“水”，“咸”聲，本義是減少。

    湊：《説文》：“水上人所會也。从水，奏聲。倉奏切。”

    盜：甲骨文。《説文》：“私利物也。从㳄，㳄欲皿者。徒到切。”

    㳄〔xián / ㄒㄧㄢˊ〕：甲骨文。甲骨文從欠從數點，“欠”象人跪坐向前張口，數點象口液，全字象人張開口流口水形，或加“手”，象以手抹拭口液形，引申爲羨溢、泛溢（參于省吾）。甲骨文“㳄”後世分爲兩字，一作“涎”，表示口液；一作“羨”，自口液溢出，引申表示氾濫、羨餘，通作“衍”。又自垂涎引申慕欲、貪羨之義，中古時期“㳄”、“羨”、“涎”、“唌”還常混用無別（參張政烺）。

    羨：小篆“羨”字從“羊”從“㳄”，甲骨文“㳄”象人張口流口水形，引伸爲羨溢，卜辭表示泛濫；林義光謂“羨”字象見羊美而涎欲下，孤備一説。自垂涎引伸出貪羨之義，今天説一人貪求某物仍曰垂涎，亦曰羨慕，即本於此。

    淒：從“水”，“妻”聲。本義爲帶雨之雲湧現（段玉裁）。

    涼：從“水”，“京”聲，本義爲薄酒。

    淨：從“水”，“爭”聲。古作“瀞”，表示清潔無垢。
