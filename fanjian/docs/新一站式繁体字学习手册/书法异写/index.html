    书法异写 #

    最近才發現“毀”和“敘”的大陸標準寫法不符合字源（但書法中會經常出現），這些字不算繁簡字，也不在繁簡字表裏。倒不是糾結於每個字非要符合字源，我認爲一個好的正字標準應該在所有常用寫法中找最符合字源的一個。然而，兩岸三地的正字標準似乎都不符合這個我覺得合理的要求。

    這裏列的書法異寫和俗字到底哪個算俗字哪個算異寫，不好界定。像“雞”變“鳮”這種明顯的俗字一般都歸類到某種簡化方式裏了。

    來（来）、萊（莱）、睞（睐） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文“來”是麦子之“麥”的象形初文，后假借为来往之来，久借不还。一说“來”被借用作去来之来，并加注义符“夊”（象脚掌向下形，参见“夊”）以明示其行动义。然而，由于来去之来字常用，为刻写之便，又省掉“夊”（参洪家义）。而“麥”字原本的来往之来义，却用作麦子之麦义。

    先就字形方面说“來”字的演变。甲骨文“來”象麦子，是麦子之麦的本字，一般以为指小麦。字形中间是直上的麦穗，两边叶子下垂，叶下是茎和根。其字上端或加斜或横画。麦穗直上，因为麦茎坚强，中空有节似竹（参罗振玉、张哲）。

    不少甲骨文“來”字上端加斜或横画，有学者认为其字最初本应无横或斜画，后来增一斜或横画以辨其初义（麥子之來）与假借义（往來之來）。不过，在实际甲骨文文例中未见明确而一致地应用此区分（参季旭昇）。即，不管是否增加一横画，都有表达麥子之來的情况，亦有表达到來之來的例子。或者说横画可能是无深意的赘笔（参裘锡圭），或说是饰笔（参《古文字谱系疏证》）。另有学者认为横画代表麦穗（如李裕）。

    金文承甲骨文字形，有的作整株麦子之象形，有的在字形上端加一横画，但（多）不封顶。加横画的字形较为常见。

    彔、綠（绿）、錄（录）、剝（剥）、祿（禄）、碌 #

    甲骨 彔金文 彔小篆 彔隶书 錄楷书 錄行书 錄草书 錄

    彔〔lù / ㄌㄨˋ〕，甲骨文疑为井鹿卢（后作“辘轳”）之形，即设置在井上用来绞起汲水斗的简单装置。其上象桔槔，下象汲水器，小点象水滴形。今字作“辘”，与“轳”字连文。（李孝定）

    大陆以录为标准字形，港台以彔为标准字形。孰好孰坏应该分析下录、彔、彖三个字形来分析。

    彖，甲骨文作，金文作，小篆作。甲骨文从豕从一横，一横是箭的省形，象豕中箭矢之形。“彖”与甲骨文“彘”字形近，故郭沬若怀疑古“彖”“彘”本一字。卜辞用作名词，表示牺牲。金文有两种形体，第一种从豕从一斜画（象箭形）穿过豕身，第二种从豕从一扁圆形，象绳索圈着豕身（康殷）。两字形皆象捕豕之形。部分学者认为两字形象豕猪中箭倒地，或以绳索将豕拖仆于地，引伸有坠落之意（参张世超）。“彖”字上部“彑”形金文本作“彐”形，是“豕”形颈部讹变而成。古籍中“彐”与“彑”形通用，如“彙”与“彚”“彝”与“彞”“彔”与“录”等。金文“彖”旧说读作“墜”，最新释读作“惰”，表示懈怠。

    可见，港台标准完全是遵从小篆。大陆一般是从俗，不过普通人怎么写我无从考证，从《词典网》查到的历代书家字形，彖也写作彐字头（緣），不知大陆为什么做彑字头。

    从字理上看，小篆的彑字头并不比彐字头有优势，不如统一写作彐，也符合书法审美。

    奧（奥）、澳、粵（粤）、襖（袄）、嶴（岙）、噢、懊 #

    奧里面是釆不是米。

    甲骨文金文小篆隶书楷书行书草书

    奧，同“”，从“宀”，“”声，本义为室内西南隅。

    〔juàn / ㄐㄩㄢˋ〕，《说文》：“抟饭也。从廾，釆声。釆，古文辦（辨）字。读若书卷。居券切。”

    釆〔biàn / ㄅㄧㄢˋ〕，甲骨文，金文，小篆，构形初义不明，一般解作“辨”。张世超等认为“釆”从“力”从数点，数点象谷粒种子，会翻土播种之意，是播种的“播”字的初文。后增从“田”“手”作“播”。姑备一说。

    甲骨文金文小篆隶书楷书行书草书

    粵，从“亏”，从“宷”，用作助词，表示审慎的语气。

    宷〔shěn / ㄕㄣˇ〕，悉也。《说文》：“知宷谛也。从宀从釆。式荏切。徐锴曰：‘宀，覆也。釆，别也。包覆而深别之。宷，悉也。’審，篆文宷从番。”（審是审的繁体字）

    大陆奥字形里面皆无撇，港台标准有撇。按播、番、释等字都有撇，奥字当有撇。不知道什么原因，奥、粤两字有撇和无撇的字有不同编码（即属于两个不同的字），澳、懊等字（可能剩余的字皆如此）只有一个编码，没有区分有撇和无撇。此种情况下，同一个编码的字，大陆的字库（例如宋体）显示的是没有撇的，港台的字库（例如MingLiU）显示的是有撇的。

    悤（△匆△怱）、蔥（葱）、總（总）、聰（聪） #

    大陆里面是个夕（点出头）。

    甲骨文金文楚文字三晋文字小篆

    悤，甲金文从一粗点从“心”，在心开处画一个粗点表示心有孔窍，心有孔窍则人心通彻，人就聪明（裘锡圭、沈培）。是“聰”的古字，本义是聪明。

    甲金文所从的粗点或写成一竖之形，竖笔中间或加圆点为饰，圆点后来发展为一横笔。战国竹简“悤”字加注“凶”声，字形下部可能是“凶”和“心”的合写（裘锡圭）。后来“悤”字的中间作左右两斜笔交叉形，或作重叠的叉形（即从“囪”从“心”，参汗简字形），“囪”左下省略则成“匆”，故异体作“怱”。“聰”字异体作“聦”。“蔥”字又作“葱”，陈汉平认为“悤”为心中之空窍，“蔥”亦因其叶中空而得名。汉代隶书“悤”旁又有写作“忩”，从“公”为声符。今天的简化字“总”，就是“忩”的变形（裘锡圭）。

    今大陆和港台皆以匆为悤忙的悤的正字；大陆以葱为正字，港台以蔥为正字。

    楷书 聰行书 聰草书 聰

    溫（温）、醞（酝）、慍（愠）、氳（氲）、瘟、蘊（蕴） #

    〔wēn / ㄨㄣ〕，《说文》：“仁也。从皿，以食囚也。官溥说。乌浑切。”就是说用器皿盛着食物给囚犯吃。段玉裁认为这是溫的本字，《说文》中溫解作水名，看来本来不表示温暖，是同音假借。然而《汉字全息资源应用系统》给出的甲骨文作，金文作，小篆作。我没记错的话这甲骨文分明是囚字吧？但是查囚的甲骨文居然查不到，有可能是我以前看的书有问题，可能没有发现囚的甲骨文？没有找到清楚的解释。有人说东汉的《汉故豫州从事尹府君铭》中出现溫写作温。

    在《汉字全息资源应用系统》中搜索“溫”，会得到一种甲骨文写法：。看字形应该是人在皿中且身上有水。没找到专业解释，但和溫应该是没有继承关系，只是意义上应当隶定作溫吧。

    冊（册）、刪（删）、柵（栅）、跚、珊、姍（姗） #

    甲骨文金文小篆隶书楷书行书草书

    甲金文象以绳索编在一起的竹简之形，象有两道编绳之形，本义是简册。师酉簋、弭吊簋“册”字上部有两竖笔上有圆点之形，竖笔上的圆点多延伸为一短横，与“竹”字形近，是《说文》古文（册）字顶部从“竹”的讹变来源（商承祚、张世超等）。“典”字所从的“册”字上部也有相近的变化。

    大陆以册为标准字形，港台以冊为标准字形。

    跚、珊似乎沒有另外一種電腦字形。有一部分筆畫小異的字就是這樣，沒有因爲字形的不同而佔用兩個編碼，這就導致字庫只能選擇一種字形，不能兩種字形都有，例如，大陸字庫的“跚”字形就選“册”，港臺的就選“冊”。

    尌、樹（树）、廚（厨）、櫥（橱）、躕（蹰） #

    甲骨文金文小篆隶书 樹楷书 樹行书 樹草书 樹

    “尌〔shù / ㄕㄨˋ〕”之甲骨文作“权”，从“木”从“又”，“又”象手形，象以手植树之形，本义是树立，是“樹”的初文。

    金文加从“豆”为声符，“豆”上之“木”形省作“屮”，故从“木”从“豆”之形讹为“壴”，与“鼓”之初文“壴”字相混。由于“木”形讹为“屮”，故又于“尌”旁加注义符“木”作“樹”。

    “又”形至秦简、小篆作“寸”，古文字往往于“又”下加点成“寸”字。

    虛（虚）、墟、噓（嘘）、覷（觑） #

    甲骨文金文小篆隶书楷书行书草书

    从“丘”，“虍”声。本义是大丘，即大山。段玉裁《说文解字注》：“虚本谓大丘，大则空旷，故引伸之爲空虚。”郝懿行义疏：“虚，今作墟。古无墟字，皆以虚为墟也，后人虚旁加土以别于空虚，因而经典亦多改虚为墟。”

    丘，甲骨文，金文，小篆。象两个小山丘之形，本义是土丘、小山。一说“丘”象四方高中央低的地形，孔子名丘，就是因为他的头顶中间是凹的（参《史记·孔子世家》）。甲骨文象二峰形，与“山”字作三峰之形有别。金文讹变为象两人背对背站立的“北”形，为小篆字形所本。战国文字“丘”下从“土”（），与《说文》古文形近。

    隋、隨（随）、橢（椭）、墮（堕）、髓 #

    甲骨文金文小篆隶书楷书行书草书

    隋，《说文》：“裂肉也。从肉从隓省。徒果切。”

    隓，金文从“阜”从二“又”二“土”，象以手推“阜”上的土落下形，引伸有堕下义。二“圣”后来变为二“左”，应是因为“圣”“左”形近，“左”又与“墮”音近之故（参裘锡圭）。本义是堕土、堕下。“隓”是“墮”的本字，异体作“”“隳”。

    髓没有简化，有一个髄字被列为异体。但我认为和隨、随的关系一样，顶多是书法异写或者俗体，只是不知道为什么髓没有简化。

    吳（吴）、誤（误）、娛（娱）、虞 #

    大陆以吴为标准字，非繁简关系。

    甲骨文金文小篆隶书楷书行书草书

    金文从“大”从“口”，会人张大口之意，指大声喧哗。“吳”或从“夨”（读作虞）为声符。“吳”字后多借作国名或人名。

    “大”本解作一张开两臂的人，“大”之侧加上“口”，其实象一人张口大叫的模样。李伯谦认为“大”“夫”古本一字，“大”读作“夫”，亦是“吳”的声符。许慎《说文》训曰：“吳，姓也。亦郡也。一曰吳，大言也，从夨、口。，古文如此。”许慎以“大言”解“吴”一说正说明此理。《诗经·周颂·丝衣》“不吴不敖”和《诗经·鲁颂·泮水》“不吴不扬”两句话，毛亨《传》两处皆云：“吴，哗也。”“吴”与“敖”“扬”并称，意指喧哗叫喊，与许慎释为“大言”之说相合。

    不过，“吴”字的使用，早在金文时期便几全解作专名。如用作国名，即指太王之子太伯仲雍奔荆蛮所立之国。吴王夫差矛：“吴王夫差自乍（作）甬（用）鈼。”或用作人名，吴方彝盖：“王乎（呼）史戊册令（命）吴。”“吴”又用为“虞”，用作职官名，即虞人，负责掌管山泽田猎之事。同簋：“（司）昜（場）、林、吴（虞）、牧”，即指掌管场人、林衡、虞人、牧人等官员。

    战国竹简“吴”用作国名，《上博竹书七·吴命》简3正：“吴青（请）城（成）于楚”，指吴国向楚国求和。“吴”亦读作“虞”，《上博竹书二·子羔》简1：“又（有）吴（虞）是（氏）之乐正质夔之子也”，“有虞氏”即帝舜。另字亦读作“无”，《上博竹书一·容成氏》简5：“禽兽朝，鱼鳖献，有吴（无）通”，指献禽兽来进见，以鱼鳖来朝贡，彼此互通有无。

    传世文献中，“吴”几亦全用作专名。唯《方言》中有“吴，大也”一训，段玉裁《说文解字注》认为“吴”从大言“引申之为凡大之称。”

    亘、亙、恆（恒） #

    甲骨文金文小篆隶书楷书行书草书

    亘，《说文》：“求亘也。从二从囘。囘，古文回，象亘回形，上下所求物也。须缘切。徐锴曰：‘回，风回转，所以宣阴阳也。’”亘读作〔xuān / ㄒㄩㄢ〕。杨树达认为：“亘”即“漩”字，上下两横代表河岸，“回”代表漩涡；又引申为盘旋、回环、辗转、曲折等义；盘桓的桓即系亘字衍生出来的。

    甲骨文金文小篆隶书楷书行书草书

    “亙”是“恆”的初文，甲、金文从“月”在“二”中，会月在天地间永恒之意（何琳仪）。

    由于月亮大部分时间都是缺的，称为弦月，又称娥眉月，故以缺月为恒常，本义是恒久、恒常（徐灏）。何琳仪认为上下两横象天地之形。战国金文在“月”旁加“卜”，从“外”，与《说文》“恆”之古文相同。至小篆“月”又讹变为“舟”，后加心旁作“恆”，表示持久不变的心志。

    亘和亙字形相近，隶变后经常不分。但从字音上能区分哪些字是亘，哪些字是亙。例如，宣、桓、垣等韵母为uan；恆、揯、緪、堩等音为gen或geng。

    但由“堩”字可见，这个字并没有分配两个电脑字符编码（类似恒跟恆）。对电脑来说，只有一个“堩”字，要怎么显示，由字体库决定。因此，大陆的字体库（比如宋体）会将这个字显示成“堩”，而台湾的字体库（也只一部分）会将这个字显示成“”。可将“堩”字复制到筆順字典網搜索栏测试，会发现它在搜索栏中显示成“”，这个网站使用的字体会将“堩”显示成“”。

    屬（属）、囑（嘱）、矚（瞩） #

    甲骨文金文小篆隶书楷书行书草书

    金文从“尾”，“蜀”声，初义不明，疑本义为连续（许慎），动物的尾巴与身体相连，故从“尾”有连续之意（徐锴）。徐锴《说文解字系传》：“屬，相连续，若尾之在体。”《说文》：“屬，连也。从尾，蜀声。”

    动物的尾巴与身体不但相连，而且有附属关系（较小的尾巴附着于较大的身体），故“屬”又有附属之意。引申之，又有类属、归属、从属、统属、下属、部属、亲属、家属。

    金文表示用作官府名、行政机构名，疑取附属之意。汉帛书“屬”通假为“躅”，“踯躅”表示徘徊不前的样子。

    稟（禀）、凜（凛）、檁（檩） #

    甲骨文金文小篆隶书楷书行书草书

    金文从“㐭”从“禾”，“㐭”亦是声符，象仓禀中储存着稻米之形，本义是藏谷物的处所，即仓禀、粮仓。或不从“禾”而从“米”，象粮仓中储存稻谷之形，本义是粮仓。陈梦家认为“㐭”象露天的谷堆之形，今天的北方农民在麦场上，作一个圆形的低土头，上堆麦秆麦壳，顶上作一亭盖形，涂以泥土，谓之“花蓝子”，与“㐭”的字形相似。

    “㐭”“禀”“廪”古本一字（李孝定），金文加“禾”“米”等义符，表示这是堆禾、米用的。《马王堆帛书·老子乙本卷前古佚书》加义符“广”，表示当时的“㐭”多是有屋顶的（季旭昇）。另参见“[嗇]({{ relref path=“行草书楷化.md/#嗇啬牆墙從嗇爿聲墻薔蔷檣樯” }})”。

    大陆好像将“稟”认定为“禀”的旧字形。

    盍、蓋（盖）、闔（阖） #

    甲骨文楚文字小篆隶书 蓋楷书 蓋行书 蓋草书 蓋

    盍〔hé / ㄏㄜˊ〕，甲骨文从大从口，“大”象器盖，“口”象器皿，会盖皿相合之意。楚文字“大”形变为“去”形，“口”变为“皿”。小篆从大从血，血为皿中有物。“去”为“盍”的初文，“盍”为“蓋”的初文，“盍”本义为覆盖、器盖。

    去，从大从口，构形本义有两种看法，第一种认为是呿的初文，表示张大嘴巴，本义是开口（裘锡圭）；第二种认为口象洞口，大为人形，古人穴居，会离家外出之意，与“出”字义近。此外，从大从口的去也是盍的初文。盍的初文（古韵属盍部）与呿的初文（古韵属鱼部）是同形异字。

    卻（却）、腳（脚） #

    甲骨文金文小篆隶书楷书行书草书

    卻，从“卩”，“”声，表示使之退。卩，跪跽人形。〔jué / ㄐㄩㄝˊ〕，《说文》：“口上阿也。从口，上象其理。凡之属皆从。其虐切。，或如此。臄，或从肉从豦。”

    大陆视“卻”为旧字形，港台以其为标准字形。

    雇（※僱）、顧（顾） #

    甲骨文金文小篆隶书楷书行书草书

    雇，甲骨文，从隹，户声，本义是一种鸟名，读作〔hù / ㄏㄨˋ〕。《说文》：“九雇，农桑候鸟，扈民不婬者也。从隹，户声。……侯古切。”后借用做雇佣、典雇等义，读作〔gù / ㄍㄨˋ〕。后又写作僱。大陆以僱为雇的异体。从字典看，台湾将雇和僱并行，但是僱不用于形容词情况。例如，雇工可理解爲被雇傭的人，是個偏正結構的名詞，不作僱工。僱工會被理解成動賓短語，即僱請工人。類似的還有雇農（受雇在農場從事耕作，以賺取工資維生的農民）、雇主、雇員都不作“僱”。其實說白了，“僱”就是個動詞，明確爲動詞的場合就可以用它。但其實所有場合下都可以用“雇”，“僱”只不過是個偏旁類化出來的字，例如“僱傭”。

    寡，金文。前两体为西周金文，从宀从頁，象一人在屋内独处（季旭昇），有单独、孤独之意，本义是独居无伴，引申为少。第三体为战国晚期金文，从頁，两侧加饰笔。

    顧，金文有三体，前两体是西周金文，但实在看不出是不是从西周金文“寡”；第三体是战国晚期金文，可明显看出从鳥、从战国晚期金文“寡”声。可能以鸟的回头张看表示回顾。小篆以雇为声。

    臥（卧） #

    甲骨文金文小篆隶书楷书行书草书

    从“人”，从“臣”，本义为伏着休息。《说文》：“休也。从人、臣，取其伏也。凡卧之属皆从卧。”字中“臣”为“目”之象形，与人臣之“臣”无关。

    大陆视“臥”为旧字形，港台以其为标准字形。

    屆（届） #

    甲骨文金文小篆隶书楷书行书草书

    屆，《说文》：“行不便也。一曰极也。从尸凷声。古拜切。”

    凷，《说文》：“墣也。从土，一屈象形。苦对切。塊（块），凷或从鬼。”

    大陆视“屆”为旧字形，港台以其为标准字形。

    茲（兹） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文及金文均写作“”，字象两束丝，于甲金文多被借作指示代词用。从“艸”的“兹”字见于战国秦简文字。

    本义是丝线。“”与“絲”皆象两束丝之形，不过“”无上下两端之绪形，两字于甲骨文用法有别。金文“”“絲”开始用法相混，故李孝定、张世超等认为“”“絲”是同字异体，晚周或春秋以后始分化为二字。《说文》训“”为微，乃“絲”之引申义，蚕丝幼细，故引申有幽微之义。

    甲金文借来表示近指代词，与“此”同义，典籍作“茲”，师同鼎：“用鑄（茲）尊鼎”。

    大陆视“茲”为旧字形，港台以其为标准字形，但作偏旁时（如磁、慈等字）又写成“兹”。

    禿（秃）、頹（颓） #

    甲骨文金文小篆隶书楷书行书草书

    “禿”的字形最早见于秦印、秦陶文中，作偏旁使用。“禿”字本从“禾”，从“人”，汉简和汉印亦同。小篆“人”形变为“儿”，为后世所本。“禿”的构形初义不明，《说文》以为其本义是没有头发。

    大陆视“禿”为旧字形，港台以其为标准字形。

    毀（毁） #

    甲骨文金文小篆隶书楷书行书草书

    金文從“殳”從“兒”，“兒”的下部加“土”形訛變爲“”，“殳”象手持捶撃武器。字會以武器敲打兒子之意，本義是毀壞。《小爾雅·廣言》：“毀，壞也。”

    《說文解字·土部》：“毀，缺也。從土、毇省聲。”《金石文字辨異·上聲·紙韻》“毀”下引《魏鍾繇薦季直表》作“毁”。《正字通·殳部》“毀”下注云：“俗作毁，非。”《中文大辭典·殳部》“毁”下云：“毀之俗字。”

    敘（叙） #

    敍，小篆作。從攴，余聲。本義是次第、次序。《說文》：“敘，次弟（第）也。从攴，余聲。”如《尚書·洪範》：“五者來備，各以其敘，庶草蕃廡。”孔傳：“言五者備至，各以次序，則眾草蕃滋廡豐也。”《淮南子·本經》：“是以天覆以德，地載以樂，四時不失其敘，風雨不降其虐，日月淑清而揚光，五星循軌而不失其行。”在排列次序和序言義上與“序”通用。

    戰國楚系文字以從“余”從“攴”的字形表示去除的意思。如《九店楚簡》簡28“敘（除）不詳”、《上博二·從政（甲篇）》簡5“敘（除）十怨”、簡6“不恭則無以敘（除）辱”。此爲戰國楚系文字的用法，秦系文字以“除”的字形表示去除義。

    “攴”隸變也作“攵”，表示手持東西擊打，通常在字形中用於表示鞭策、督促的意義，比如“教”“改”等字。“敘”這個字可能就是表示加以管理使之有序。歷代書法中很多書家會把“敘”寫作“叙”。這個字沒有出現在繁簡字表裏，只是字形標準認定不同，大陸主要從俗。
