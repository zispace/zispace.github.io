    “亼”在甲骨文及金文中都没有独立成字，但却广泛地用作造字的部件，意义是向下张口，可称之为“倒口”。

    “”《说文》：“，三合也。从入一，象三合之形。凡之属皆从。读若集。”此中所谓“三合之形”，象覆罩着的器盖形（谷衍奎）。但“”很容易与另一部件“亼”相混，必须清楚地区别。

    “亼”和“”古文字字形相近，为了区分，一个隶定为亼，一个隶定为。

    “侖”字的构形可以有两种解释。首先可把“侖”视为从“”（三合）从“冊”。即解作“集合及整理卷册”的意思。另一种解释是把 “侖”视作从“亼”（倒口）从“冊”。《说文》的解释是：“侖，思也。从亼从冊。”此中，“亼”可解向下张口，而“冊”则解以绳索编列得整齐有序的竹简，这两个部件合起来，似意会一套条理井然的说话或思想， 一如人之能“出口成册”。由于“侖”本已由口（亼）所出，因此可视为“論”的初文。章太炎《国故论衡·文学总略》即直称：“論者， 古但作侖”。

    帝，甲金文象束绑柴薪的形状，“帝”是“禘”的古字，指在郊外烧柴而祭，本义是一种烧柴的祭祀。后因“帝”借来表示上帝、天帝、皇帝，加“示”旁为“禘”字来表示禘祭的本义。金文或于上部加一横为饰笔，为小篆字形所本。

    适，金文，從“辵”從“舌”，構形初義不明。王子适匜（第二例）的“适”字所從的“舌”旁上部稍有訛變，與一般的“舌”字上部作開叉之形有別。金文用作人名，王子适匜：“王子适”。古人以“适”爲名，如孔子的弟子南宮适。

    小篆作，《説文》：“适，疾也。从辵𠯑聲。讀與括同。”根據《説文》，古代的“适”與“括”同音，本義是疾。現代漢語中“适”是“適”的簡化字，表示適合，與古代的“适”是同形異字。

    腦，从“肉”，从“巛”，从“囟”，本义为脑髓。“腦”同于“匘”。《説文》：“匘，頭髓也。从匕。匕，相匕著也，巛象髮，囟象匘形。”段玉裁注：“匘，俗作腦。”

    惱、瑙等都未找到甲金文相关字形。唐碑中“腦”有作者，“惱”的行书宋米芾作，宋苏轼作。

    亂（乱）、辭（辞） #

    甲骨文金文小篆隶书楷书行书草书

    金文作“”，“”是“亂”的初文。金文从“爪”从“幺”从倒放的工字形饰笔，从“又”。“爪”“又”皆象手形，或从“”，“幺”“”皆象丝线之形，全字象上下两手在整理丝絮之形，本义是治丝。

    丝线在未整理之前是乱的，故分化出“亂”字；而治理丝线带出治理一义，分化出“”（典籍作“司”）字。故“”是“亂”“”的初文。金文或加从四口（“㗊”），“㗊”是“亂”的声符（陈英杰、孟蓬生）。郭沫若认为因为治丝时声音嚣骚之故。

    對（对）、懟（怼） #

    甲骨文金文小篆隶书楷书行书草书

    甲文从“又”从“丵”从“土”，陈秉新认为上“丵”下“土”此形亦是声符，“丵”象凿击工具，字象手持工具开辟土地。

    金文在“丵”下加“丰”（象艸木），字形结构与“封”“邦”字相近。因为开辟土地之后，要树立地界，先秦树立地界多半以植树为之，“對”的本义是辟地分疆，植树于土，立下地界，后假借为应对、应答之意。《詩·大雅·皇矣》：“帝作邦作對”，表示帝开辟邦国、开辟领土。

    《说文》小篆加从“口”，可能是为应对、对答的“對”的而造的分化字，只是这个字没有流传下来，后世仍然用“對”字（季旭升）。

    業（业）、鄴（邺） #

    金文小篆隶书楷书行书草书

    “業”象两个并立的铜人举笋（悬钟磬之横木）之形，故“業”是悬挂钟磬的架子的象形。

    金文从二“業”从“去”，与《说文》古文形近，“去”的古文字有两个来源，“業”字所从的“去”是“盍”的初文，读作“盍”，象器盖相合之形（参裘锡圭），是“業”的声符。后期金文只从一个“業”从“口”，“業”的下部和“去”字共享笔画。

    金文中有用作基業或功業之業，如中山王器銘：“以內絕邵公之業。”從“去”的“業”字通假爲“協”，表示輔相、佐助，秦公簋：“保業（䢃）氒〈又─有〉秦”，表示保佑輔助秦國。又表示安定之意，鐘：“業妥（綏）厚多福”，表示安定而多福之意。

    夢（梦） #

    大陆顶上写作艹。实际不是草。

    甲骨文從“爿”從“”，“”亦聲。“”是“㝱”的初文，象人在做夢時手、眼睛、眉毛等有所動作。（謝明文）本義是做夢。

    戰國文字“夢”所從的“”和“蔑”所從的偏旁有混同的情況。《説文》分爲“㝱”和“夢”兩個字。“㝱”是在甲骨文的基礎上加上“宀”旁。許慎以爲“㝱”字從“疒”，其實本應作“爿”。（裘錫圭）《説文》“㝱”的篆形在“夢”上多了一橫筆。《説文》：“㝱，寐而有覺也。从宀从疒，夢聲。”而《説文》的“夢”字則是把意符“爿”換爲“夕”。《説文》：“夢，不明也。从夕，瞢省聲。”其實“㝱”和“夢”是一字之異體。（謝明文）

    習（习） #

    甲骨文楚文字小篆隶书楷书行书草书

    甲骨文从“日”，“彗”声，本义是暴晒，故从“日”。假借为习惯、学习。根据唐兰的研究，甲骨文“彗”的字形与后来的“羽”字相近，但不是“羽”，而是“彗”。因为甲骨文没有“羽”字，而甲骨文“雪”字的“雨”下从此羽形，对比《说文》“雪”字的小篆作“䨮”，从“彗”声，可知此羽形实为“彗”。

    至小篆，“習”所从的“彗”讹变为“羽”，“日”讹变为“白”。

    甲骨文表示重複、因襲，《合集》31672：“習一卜”。《合集》31673：“習二卜”。《合集》31674：“習三卜”。“習一卜”表示重複一次占卜，“習二卜”“習三卜”分別表示重複二次、三次之占卜。郭沫若認爲這些習卜，一次占卜用三個龜甲或三片牛肩胛骨，三龜或三骨爲一習。

    戰國竹簡表示學習，《郭店簡·性自命出》簡60-61：“蜀（獨）凥（處）鼎（則）習父兄之所樂”，意謂一個人獨處時要多做父兄喜歡的事。《郭店簡·語叢三》簡10：“起習文章，嗌（益）”，劉釗認爲“起”表示啟發、開導，全句意謂啟發學習禮樂制度，有益處。

    “習”後來又用作姓氏，《漢印文字徵》：“習封之印”。

    未简化的“百褶裙”的“褶”字即从習声，“摺”亦从習声，当然，在普通话中已看不出怎么表声。不过熠熠生辉的“熠”读〔yì / ㄧˋ〕，在普通话中又跟“習”押韵。可能是“習”读音演变比较复杂吧。

    轟，从三“車”，表示很多车发出的声音，后泛指巨大的声响。行书，草书。草书作个参考吧，取自《草书韵会》，不知道谁写的。

    邊（边） #

    甲骨文金文小篆隶书楷书行书草书

    金文從“辵”從“自”從“丙”從“方”，“自”是“鼻”的初文，古音與“鼻”相同，“自”“丙”都是“邊”的聲符（張世超）。從“方”有旁邊之意。“邊”的本義是旁邊。

    古“方”“旁”通假，《儀禮·大射禮》“左右曰方”鄭玄注：“方，出旁也。”（參丁山）

    金文表示邊境、邊遠地區。大盂鼎：“我聞殷述（墜）令（命），隹（唯）殷邊侯、田（甸）雩（與）殷正百辟，率肆于酉（酒）。”意思是説我聽聞殷朝墜喪天命，殷朝邊遠地區的諸侯和朝中的官員都沉湎於酒。又用作地名，散氏盤：“至于邊柳”。

    “邊”小篆作“”，《説文》：“，行垂崖也。从辵聲。”

    旁，小篆，因此有个隶变形式“㫄”，也是“邊”的楷书第二例的写法所从。

    方，甲骨文；旁，甲骨文。“方”甲骨文從“一”從“刀”，以“一”指示“刀”的鋒芒，本義是刀的鋒芒，後借用作方圓之“方”（裘錫圭）。“方”字之甲骨文及金文構形大致相同，皆從“一”從“刀”，古漢字以“□”表示方圓之“方”，但由於“□”在古文字裏容易跟“丁”的勾廓寫法和“圓”字初文相混，所以很早就以假借字“方”來表示方圓之“方”。“方”與“亡（芒）”造字的原理一致，都以“一”指示“刀”的鋒芒，且二字古音相近，足證二字同源。後來古人借用鋒芒之“芒”的表意初文來表示方圓之“方”，因此分化出“方”字。

    對於“方”字構形的本義尚有其他説法，如葉玉森以爲象“刀”懸掛於架上形；朱芳圃以爲“方”是“枋”的初文，以“一”指示持“刀”之處，與刀鋻相同。按“一”所指示之處爲刀身，而非刀柄，諸説似不合“方”字構形。至於徐中舒以爲“方”字象“耒”之形，何琳儀以爲象“一”橫於“刀”身，表示以刀分物，皆可參考。

    “旁”造字本義不明。“旁”多表示旁邊、廣泛、另外的等義項。

    這（这） #

    “這”指代較近的時間、地點或事物，不見於甲金文及《説文》。《廣韻》魚變切《集韻》牛堰切，音彥。《玉篇》迎也。《正字通》周禮有掌訝，主迎。訝古作這。毛晃曰：凡稱此箇爲者箇，俗多改用這字。這乃迎也。

    “這”亦可作金、元曲文理的襯字，位於語句中間，無實義，如元代關漢卿《謝天香》：“待道是癲狂睡囈，兀的不青天這白日。”

    在東漢石碑《薌他君祠堂石柱題記》中有“堂雖小，經日甚久，取石南山，更逾二年，這今成已。”字句，個人認爲應當是“現今已成”的意思，和現在“這”字義同。

    慶（庆） #

    > 简单说就是“鹿”字头、“愛”字底。
    甲骨文金文小篆隶书楷书行书草书

    甲金文从“廌”从“心”。金文或从“廌”从“文”，“廌”是神兽，“文”、“心”象神兽身上的花纹，本义是廌身的花纹。由美丽的花纹引申出美善、喜庆之义（魏宜辉）。

    小篆从“鹿”从“心”，“鹿”是“廌”的讹变，廌的尾巴讹变为“夂”，古人在吉祥的典礼中用鹿皮作为礼物，以表达美好的心愿，因此有庆贺、喜庆之义。《説文》：“慶，行賀人也。从心从夊，吉禮以鹿皮爲贄，故从鹿省。”

    競（竞） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文从二“人”或“大”，字顶部象人带头饰之形，全字疑象二人头带饰物并肩竞逐之形（商承祚、李孝定），本义是竞跑，引申为竞争。《莊子·齊物論》：“有競有爭”，郭象注：“並逐曰競。”《説文》：“競，彊語也。一曰逐也。从誩，从二人。”

    至金文演变为从“誩”，《説文》“誩，競言也”“讀若競”。“誩”有争论之意，是“競”的义符兼声符。西周中期金文有于二“大”之间加一圆形，吴振武认为象两人争球之形。姑备一说。或认为圆形只是饰笔。

    个人认为隶书之后又多附会成从二竟，兼表字音。

    戲（戏） #

    甲骨文金文小篆隶书楷书行书草书

    “戲”从戈，䖒声，本义是三军的主力军以外的部分军队。《説文》：“戲，三軍之偏也。一曰兵也。从戈，䖒聲。”张世超等认为金文表示军之偏师，師虎簋：“令（命）女（汝）更乃且（祖）考，啻（嫡）官（司）（左）右戲緐荊。”

    “戲”又表示角鬥，《國語·晉語九》：“少室周爲趙簡子之右，聞牛談有力，請與之戲，弗勝，致右焉。”韋昭注：“戲，角力也。”

    高鴻縉認爲“戲”同音通假作“嬉”，故有玩耍、游戲等義。《史記·孔子世家》：“孔子爲兒嬉戲，常陳俎豆，設禮容。”

    聲（声）、磬、罄、馨 #

    甲骨文甲骨临摹小篆隶书楷书行书草书

    甲骨文从“殸”（磬）从“耳”，“殸”亦是声符，“殸”象手持棍棒敲打悬挂着的石磬，旁边的“耳”表示石磬的乐音传到耳朵里，本义是乐音，后泛指一切声音。甲骨文亦有只从声从耳者，省去敲击动作。

    古時“聲”、“音”有別，“聲”的含義比較廣泛，凡是傳到耳朵裏的都可以叫“聲”，包括人語聲、歌聲、鳥獸聲、音樂等。“音”多指音樂，指有旋律、有節奏的聲，各種樂器和諧配合演奏出來的聲才叫“音”（王鳳陽）。《禮記·樂記》：“聲成文謂之音。”宮、商、角、徵、羽，聲也；絲、竹、金、石、匏、土、革、木，音也。《樂記》曰：‘知聲而不知音者，禽獸是也。’”意謂“宮、商、角、徵、羽”等五個音階叫做“聲”，用絲、竹、金、石、匏（葫蘆）、土、革、木八種材料製成的樂器演奏出來的叫做“音”。後來“聲音”二字常常連用，其分別已漸漸模糊。

    甲骨文疑用來表示馨香的“馨”（于省吾），《合集》27632：“聲（馨）黍其登”，意謂進獻芳香的黃米。

    “耳”字的笔顺有两种，处于字的下方时往往最后一笔写竖，从“聲”的行书即可看出。楷书亦遵循这个原则。

    蘇（苏） #

    甲骨文金文小篆隶书楷书行书草书

    陸（陆）、逵 #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文從“阜”從二“圥”，早期金文從二“阜”從“圥”，“圥”是聲符，“阜”象山陵，作爲“陸”的意符。本義是大土山，引申爲陸地。《楚辭·九嘆·憂苦》：“巡陸夷之曲衍兮，幽空虛以寂寞。”王逸注：“大阜曰陸。”意謂遊覽山丘和平原，以及彎曲的水澤，感到空虛寂寞。

    春秋以後，或加“土”爲意符。或不從“圥”而改從二“六”爲聲符。至小篆簡化爲從“阜”從“圥”從“土”。

    “逵”是“馗”的異體字。《説文》：“馗，九達道也。似龜背，故謂之馗。馗，高也。从九从首。渠追切。逵，馗或从辵从坴。”

    棗（枣） #

    甲骨文金文小篆隶书楷书行书草书

    从二“朿”，会枣树多刺之意，本义是枣树。“棗”与“棘”皆从二“朿”，两者利用偏旁之间的位置作区别，偏旁并列的为“棘”，上下重叠的为“棗”。“棗”即酸枣树，是多刺的灌木，比“棘”更高大（裘锡圭）。战国文字多从二“來”。以上两例金文乃战国金文，从二“來”。

    廟（庙，※庿） #

    廟，異體作“庿”，指敬奉祖先神位以供祭祀的屋宇。金文從“广”從“”（古潮字），從“朝”只一見，“广”或作“宀”，皆表示屋宇。“朝”“廟”韻母相同，但聲母相隔。不過“朝”“廟”意義相關，《説文通訓定聲》：“古者行禮必於廟，謀事必於廟。”“廟堂”除指祭祀祖宗之處，又指“朝廷”。故段玉裁以“廟”爲會意字。戰國金文從“广”從“苗”聲，與《説文》古文同。上博楚簡“廟”從“宀”從“苗”，是“庿”的異體。

    趙（赵） #

    金文。金文從“走”從“肖”，“肖”是聲符，本義是疾行、急速行走（參漢語大字典）。《説文》：“趙，趨趙也。从走肖聲。”段玉裁改爲“趍趙也。”金文用作人名，弔趙父劍鞘飾：“弔（叔）趙父乍（作）旅爯”。又用作姓氏，趙孟庎壺：“趙孟庎”。

    肖，金文。從“月”從“小”，“小”亦是聲符，會月光漸漸消滅之意，本義是月光減弱。何琳儀認爲是“宵”的初文，可備一説。《説文》誤以爲從肉，不確。從“肖”的字多有漸小、消減之義，如“消”字表示消失、消減，“削”表示刻削、削減，“陗”、“峭”表示陡峭，山勢越高，山便越尖峭銳小，“梢”是樹木枝梢，有末尾之義，“稍”的本義是禾末，“髾”的本義是髮尾、頭髮末梢，“銷”、“焇”有熔化、銷毀之義（參金國泰）。

    金文用作姓氏，即“趙”氏，二十九年相邦趙戈：“廿九年，相邦肖（趙）豹”，廿七年大梁司寇鼎：“梁廿又七年，大梁司寇肖（趙）亡智鑄”。

    類（类） #

    注意下面是“犬”不是“大”。

    類：小篆。從“犬”，“頪”聲，本義爲種類，即多種相同或相似事物的綜合。《説文》：“種類相似，唯犬爲甚。从犬，頪聲。”《易·乾》：“本乎天者親上，本乎地者親下，則各從其類也。”

    頪〔lèi / ㄌㄟˋ〕：難曉也。从頁、米。一曰：鮮白皃（貌），从粉省。臣鉉等曰：難曉，亦不聰之義。盧對切。《説文解字注》曰“難曉也。謂相佀（似）難分别也。頪類古今字。類本專謂犬，後乃類行而頪廢矣。”

    盤（盘） #

    金文。從“皿”，“般”聲。初文作“凡”，象側立的盥盤之形。後加“攴”旁，“凡”又訛爲“舟”，寫作“般”。金文或增“皿”旁（表示器皿）或“金”旁（表示以青銅製作）。漢代的盥盤多以木製，故又從“木”作“槃”。“盤”與《説文》“槃”的籀文相合。《説文》：“槃，承槃也。从木，般聲。鎜，古文从金。盤，籀文从皿。”

    “凡”是“盤”的象形初文，後借作平凡之凡；後增“攴”旁表示動作，“攴”又訛變成“殳”，“凡”訛變成“舟”，作“般”，又借來表示一樣、相同等義；後又在“般”下加“皿”“金”“木”等表示本義。

    糞（粪） #

    上米下異。

    甲骨文，甲骨文第二例象一手执帚、一手执箕。小篆。甲骨文從雙手持筲箕，以盛垃圾，表示灑掃除塵。本義是掃除。《禮記·曲禮上》：“凡爲長者糞之禮，必加帚於箕上，以袂拘而退，其塵不及長者。”陳澔集説：“糞，除穢也。”意謂凡是幫長者打掃的禮節，必須把掃把放在筲箕的上面，並用衣袖遮蔽而引退，不讓塵土沾到長者。

    其，甲骨文，箕之本字，本義是簸箕。簸，從箕，皮聲。本義是揚去穀米中的糠秕雜質。

    謄（誊） #

    《説文》：“迻（移）書也。从言，朕聲。徒登切。”我理解：所謂移書就是轉寫，就是在別的地方再寫一遍。

    这个字和滕、腾、縢等字一样，都是形声字，只是不知道什么原因简化了这个字，成了和卷、拳、券等字一样了。

    櫃（柜） #

    從“木”，“匱”聲，本義爲小匣，後泛指盛放衣物、書籍、文件等用的器具。

    頭（头） #

    甲骨文金文小篆隶书楷书行书草书

    金文從“豆”從“頁”，“豆”標聲，“頁”是意符，象人形而強調頭部，本義是頭。一説青銅器中的“豆”象人頭頸相連之形（孫常敘），故“豆”除了是聲符以外，還是意符。
