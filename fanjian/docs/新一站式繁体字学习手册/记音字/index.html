    记音字

    歷（历，歷史）、曆（历，日曆）、瀝（沥）、櫪（枥）、靂（雳） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文从“秝”从“止”，异体从“林”，会在禾边林下经过之形，会经历之意（罗振玉等）。“歷”的本义为经过，用于四时运行，便引伸出曆法之“曆”（陈定民）。

    另一种说法认为“歷”非会意字，而是从止秝声的形声字，《说文》：“秝，读若歷”。此说认为足之所经皆有经过之义，不必定要在禾林边过（李孝定）。《说文》：“歷，过也。从止厤声。”按“秝”“厤”可为纯粹声符，亦可能兼为义符。古时出门便为禾田、树林，以“秝”“止”会经过之意，亦在常理中。故“歷”可能是形声字，亦可能是形声兼会意字。

    金文从“秝”从“止”外，加从“厂”，“厂”为“石”之省，于此用义不明，或只为繁笔，林义光以为有推移之意，高田忠周以为象人之所居，皆可商榷。亦有不从“止”，简作“厤”，“厤”“歷”同字异体。

    “曆”用于日曆、曆法、陰曆（阴历）、陽曆（阳历）等义，是“歷”的分化字。“歷”用于歷史、經歷、歷日月而彌（弥）新等义。

    賓（宾）、儐（傧）、擯（摈）、濱（滨）、嬪（嫔）、繽（缤）、檳（槟）、殯（殡）、鬢（鬓） #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文從“宀”從“人”，或加從“止”。“止”象來客，“賓”象主人在屋下迎接客人進來（葉玉森）。本義是賓客。“人”或作“万”形（“丏”之初文），“丏”作爲聲符，是聲化的結果，“丏”、“賓”音近。甲骨文用作祭名及人名。

    金文從“宀”從“万”從“貝”，“貝”是賓客的饋贈，古者賓客至必有物贈之，其贈之事謂之賓，故其字從貝（王國維）。故“賓”又有贈送之意。

    華（华）、嘩（哗，嘩啦）、譁（哗，喧譁、譁變）、樺（桦）、鏵（铧） #

    大陆可能是七、八笔连起来写吧，不确定。

    甲骨文金文小篆隶书楷书行书草书

    “華”是“花”的古字，金文上象花朵盛放之形，下象莖葉（“于”形），“于”爲聲符。《詩經·周南·桃夭》：“桃之夭夭，灼灼其華。”

    一説上象蓓蕾（高鴻縉），然而金文構形並不象花苞。金文用作地名、國族名、人名等。邾公華鐘：“鼄公華擇厥吉金”。

    “華”小篆作“”，“”“華”（）音義皆同，古本一字，《説文》誤分爲二。《説文》：“，艸木華也。从亏聲。凡之屬皆从。荂，或从艸从夸。”《説文》：“華，榮也。从艸从。凡華之屬皆从華。”“榮也”一義是由花朵盛放引申而來。

    義（义）、儀（仪）、議（议）、蟻（蚁） #

    顶上的羊不出头。

    甲骨文金文小篆隶书楷书行书草书

    甲金文從“羊”，“我”聲，從“羊”的字多有美善之義，與“善”、“美”從“羊”之意同。本義疑爲威儀美盛之貌，是“儀”的初文。一説“我”也是意符，“義”的初義可能是“威儀”、“儀仗”的“儀”，專指軍隊、士兵的威儀，故從“我”（象古兵器）。另一方面，古人可能認爲莊重肅穆的威儀能夠流露美善、正義之德，故從“羊”（不直接指動物，而引申指美善之事或物）。故“義”的本義是威儀淑善、清正，引申爲正義之德。後世言“義軍”、“義師”、“起義”，大概亦由此而引申。

    我：甲骨文，金文。甲金文象有鋸齒的長柄兵器，本義是一種鋸，或形近鋸的古兵器。假借爲第一人稱代詞，表示自己。郭沫若認爲象兵器的“我”即《詩經》與“斧”並舉的“錡”，“我”是“錡”的象形初文，指刃部有齒的鉞形武器。“我”所象的兵器，羅西章認爲即是陝西扶風縣莊白村出土的三齒兵器，曾憲通、陳煒湛認爲即曾侯乙墓出土的多戈戟。漢字中唯一保留了“我”兵器義的，大概只有“義”字（關子尹）。

    乂〔yì / ㄧˋ〕：甲骨文，金文。甲金文象古老時代用來剪除雜草的剪刀之形，其功用相當於今天的鐮刀，本義是鐮刀，用作動詞，表示割草。後來加“刀”旁，造“刈”字專門表示收割。

    乂和义都是義的俗字，“乂”另兼正字。

    審（审）、瀋（沈，瀋陽）、嬸（婶） #

    甲骨文金文小篆隶书楷书行书草书

    金文“宀”從“米”從“口”，睡虎地秦簡、馬王堆漢帛書下從“日”，構形初義不明，至小篆“米”訛變爲“釆〔biàn / ㄅㄧㄢˋ〕”，“口”訛變爲“田”。

    金文表示確實，五祀衛鼎：“正迺訊厲曰：‘女（汝）貯田不（否）？’厲迺許曰：‘余審貯田五田。’”

    《説文》：“宷，悉也。知宷諦也。从宀从釆。審，篆文宷从番。”

    釆〔biàn / ㄅㄧㄢˋ〕：甲骨文，金文。金文“釆”构形初义不明。一般解作“辨”，亦有谓是“播”之初文。张世超等认为“釆”从“力”从数点，数点象谷粒种子，会翻土播种之意，是播种的“播”字的初文。后增从“田”、“手”作“播”。姑备一说。

    瀋：① 汁。《左傳·哀公三年》：“無備而官辦者，猶拾瀋也。”杜預注：“瀋，汁也。”② 水名。在今瀋陽附近。

    《説文》：“瀋，汁也。”“瀋”“汁”兩字同源，北土呼汁爲瀋，只是方言的區別。

    沈〔chén / ㄔㄣˊ〕：“沉”的本字。甲骨文，金文。甲骨文從“水”從“羊”，或不從“羊”而從“牛”或“玉”，象沈牛或羊或玉於水中之形，以對河神進行祭祀。金文轉變爲形聲字，從“水”，“冘”聲，爲小篆字形所本。以“冘”为声符的字还有“忱”也读〔chén / ㄔㄣˊ〕。“沉”只是一种讹写。

    沈〔shěn / ㄕㄣˇ〕：① 古書中用作“汁”義時，也有作“沈”的。《禮記·檀弓下》：“爲榆沈，故設撥。”“榆沈”即榆樹皮汁。② 春秋國名。③ 姓。

    沈〔tán / ㄊㄢˊ〕：沈沈。宮室深邃貌。《史記·陳涉世家》：“入宮，見殿屋帷帳，客曰：‘夥頤！涉之爲王沈沈者！’”

    節（节）、癤（疖）、櫛（栉） #

    甲骨文金文小篆隶书楷书行书草书

    本指竹的節理，即竹子各段相連處突出的部位。“節”始見於戰國金文，從“竹”“即”聲，與小篆同。“節”是竹節，兩條節紋之間謂之一節；古時用作身份、權力的憑證，圓筒形的竹節剖成兩半，分由帝王及執事官員持有，或由守關者及獲特權人員持有，兩半吻合才有效。早期破竹爲節，後以青銅鑄造，仍取象竹節之形，如戰國時期的鄂君啟舟、車節，是水、陸路經商的免稅通行證。而虎符亦可稱爲“節”，如辟大夫虎符的銘文自稱爲“辟大夫信節”。可見“符”、“節”表示憑信，義同通用。

    畢（毕）、篳（筚） #

    大陸十筆寫完，中豎連作一筆，最後寫。

    甲骨文金文小篆隶书楷书行书草书

    甲骨文從“田”從，金文從“田”從“”，、“”象用來捕獵鳥獸的有柄的網。“”是“畢”的象形初文。（參季旭昇）“畢”的本義是捕獵用的網。鄭玄認爲“畢”的形制特點是網小而柄長。《禮記·月令》鄭玄注曰：“罔（網）小而柄長謂之畢。”“田”是意符，表示與田獵有關。

    養（养）、癢（痒）、痒 #

    大陆标准：第5、7笔连写；第9笔为点。

    甲骨文金文小篆隶书楷书行书草书

    “養”的字形最早見於小篆和秦簡，從“食”，“羊”聲。本義是牧養牲畜。《説文》：“，古文養。”“”又作“䍩”，從“羊”從“攴”（同攵），字形見於戰國楚簡，如《郭店簡·六德》簡33“求（養）新（親）之志”。“”是“養”的異體，秦代統一文字後多用“養”而不用“”。

    甲骨文中有從“羊”從“攴”的“䍩”字，金文字形從“羊”從“”，一般釋爲“牧”的異體。商承祚以爲字形與《説文》古文同，象以手持鞭而牧羊：“牧牛則字從牛，羊則字從羊也。後以從牛之字爲牧，而以爲養矣。”一説西周以後放牧的“牧”字未見寫作“䍩”，因此，戰國竹簡和《説文》古文的“䍩”只是“養”的異體，和甲金文的“䍩”字無關。（參《字源》）

    痒〔yáng / ㄧㄤˊ〕：《説文》：“瘍也。从疒，羊聲。似陽切。”

    瘍〔yáng / ㄧㄤˊ〕：《説文》：“頭創也。从疒，昜聲。與章切。”

    癢〔yǎng / ㄧㄤˇ〕：《説文》無。《釋名》：“癢，揚也，其氣在皮中，欲得發揚，使人搔發之而揚岀也。”《大廣益會玉篇》：“痒，餘兩切。痛痒也。《説文》曰：瘍也。癢，同上。”《類篇》：“癢，以兩切。膚欲搔也。”《廣韻》：“痒，皮痒。癢，上同。”《洪武正韻》：“癢，痛癢。亦作痒。又皮瘍。”

    説文中似乎沒有瘙癢一義的字。從後世字書的解釋也很難判斷出痛癢和頭創是不是一回事，或者是不是引申義。但是從後世字書的解釋看，把“癢”“痒”認爲是同字，和“瘍”在意義上相同。《集韻》更是把“瘍”“痒”“傷”列在一處，解釋爲頭創。

    叢（丛，※樷※藂） #

    甲骨文金文小篆隶书楷书行书草书

    《説文》︰“叢，聚也。从丵，取聲。”又《説文》︰“丵，叢生艸也。”“叢”的本義當是草木叢生（湯可敬今釋）。朱駿聲疑“叢”从丵、聚省，會草木聚生之意。字亦作“藂”或“樷”，即“叢”之或體或異體字。可備一説。

    釐（厘） #

    甲骨文金文小篆隶书楷书行书草书

    “釐”的初文作“”，“”從“來”從“人”從“又”從“攴”，象以手持杖打麥之形，乃獲麥之象形字，獲麥象徵足食，故引申有福祉之義（參李孝定）。參見來、麥。

    金文加從“里”爲聲符，“釐”是“”的後起字。、釐甲金文從“來”，金文或從木，至小篆訛作從未。古“來”、“”、“釐”同聲，“來”是“”的聲符，小篆所從之“未”應是“來”之訛。《詩·周頌·思文》：“貽我來牟”，《漢書·劉向傳》引作“貽我釐麰”，可證“來”、“釐”古同音。

    在厘米一词中，台湾也作厘，称釐米为旧作。《正字通》：“厘，俗廛字。《集韻》作，後因譌作厘。舊註知厘作毫釐非，不知同廛亦非。”从《正字通》看，厘在古代既作廛的俗体，也作釐的俗体。

    简化字保留了“廛”“瀍”“躔”等字，只将“纏”简化为“缠”，可能是因为“纏”比较常用。参见廛。

    憲（宪） #

    大陆标准，第四笔是横，第六笔出头，与“害”相同。

    甲骨文金文小篆隶书楷书行书草书

    早期金文上從，下從“目”，學者多認爲爲“害”省，後期在目下加從“心”，“”聲，構形初義不明。加藤常賢認爲象頭上戴着帽子，掩蓋眼睛之形，本義爲覆蓋頭之物，即帽子。張世超等則認爲“憲”本義是目明善察，猶“聖”本義是耳聰善聽。一説疑爲“瞎”的初文（徐在國）。

    金文用作人名，又表示賢明睿智，史牆盤：“（憲）聖成王”，表示賢明的周成王（參金文形義通解）。

    《説文》：“憲，敏也。从心从目，害省聲。”

    害，金文。金文最早見於西周中期，上部所從不明，下部皆從口，後與上部竪劃相連爲古形。字象蓋子合在器皿之上，本義是蓋子，借爲損害。金文、古籍中“害”、“割”、“介”、“丯”皆音近通用，戰國末年睡虎地秦簡“害”已寫作從宀從丯從口，與甲、金文構形有異，當爲小篆所本。

    “害”金文構形多爭議，未有定論。一説象矛頭之形，爲“”之初文，《説文》：“，矛屬，從矛，害聲。”（何琳儀）。一説象蓋器相合形（郭沫若、戴家祥）。一説爲“舍”之異文，上象屋宇，下象基台（馬叙倫）。一説害爲被刀剔割的模型（許誰雄）。一説“害”又爲“簠”字所從部件，當爲胡簋之“胡”本字（陳秉新）。無論如何，金文“害”不從“宀”，故與房舍無關。
