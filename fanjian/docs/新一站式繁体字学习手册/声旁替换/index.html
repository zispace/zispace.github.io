    登、燈（灯）、鄧（邓）、蹬、凳、瞪、镫、櫈 #

    甲骨文金文小篆隶书楷书行书草书

    甲骨文会双手捧食器登上祭台进献祭品之意，故“登”有进义。

    甲骨文從二“止”（“”）從“廾”（同“”，象雙手形）從“豆”（盛食物的容器），“豆”亦是聲符。象雙手捧食器登上祭台進獻祭品之形，故“登”有進義。反映古代獻祭時手腳並用。這種字形在甲骨文多用作人名。

    甲骨文或省去二“止”形，這種字形在甲骨文多用作本義，表示進獻祭品，猶其是獻禾、米、黍。《合集》30306“登黍于宗”。《合集》34591：“王其登米”。這種用法的字形或加從“米”爲意符，象進獻米之形。

    此外，甲骨文還有從“皀”從“廾”，此字形可讀爲“徵”，表示徵召，《合集》6619：“勿登（徵）人乎（呼）伐羌”（楊樹達）。在殷墟花園莊東地甲骨文字中，有字形只作“皀”，而用爲“登”的例子，《花東》48：“皀（登）自丁黍”，即進獻來自“丁”的黍（沈培）。

    金文從“”從“豆”從“廾”，或不從“”。班簋不從“”而從“阜”，也有上升之意。

    金文表示登上，散盤：“登于厂湶”。這種用法的“登”皆寫作從“”。又表示升遷，就職，班簋：“登于大服”。又用作國名，文獻作“鄧”，鄧公鼎：“鄧公乍旅尊鼎”。古以“登”（鄧）名國者有二，一在楚國附近，曼姓。一在陝西，媿姓（陳夢家）。又用作祭名，相當於文獻的“烝”。姬鼎：“用登（烝）用嘗”。十四年陳侯午錞：“以登（烝）以嘗”。《爾雅·釋天》：“冬祭曰烝”，郭璞注：“進品物也。”《禮記·王制》：“天子諸侯宗廟之祭，春曰礿，夏曰禘，秋曰嘗，冬曰烝。”《春秋繁露·四祭》：“蒸者，以十月進初稻也。”

    襄、讓（让）、釀（酿）、壤、嚷、瓤、攘、囊 #

    甲骨文金文小篆隶书楷书行书草书

    “襄”字的構形初意不明。疑象人伸手解衣，致力於耕作之形（許慎、劉心源、高鴻縉）。一説甲骨文象人頭頂戴物之形（何琳儀）。一説象人舉起兩手，捋起衣袖之形，是攘臂的“攘”的初文。此説的問題是字形下部的人還有手形。早期金文與甲骨文形同，後期金文加從“衣”爲意符。或加“土”旁，是土壤的“壤”的專字。

    甲骨文用作地名，《合集》24234：“才（在）襄”。金文用作人名。又表示輔助，毛公鼎：“襄辥（乂）氒（厥）辟”，表示輔助他的君主。

    种:〔chóng / ㄔㄨㄥˊ〕

    甲骨文金文小篆隶书楷书行书草书

    甲金文“重”會人背負重物之意，本義是沉重。甲金文從“人”從“東”（象囊橐之形），商代象形金文作人背負囊橐之形，意會人背負重物，本義是沉重（柯昌濟、何琳儀）。從“東”除表意之外，也是“重”的聲符。甲骨文所從的“東”（囊袋）或置於人形之下，而西周以後金文“人”與“東”共用筆畫，中豎合寫，爲小篆字形所本。部分字形下加“土”旁，表示人站在地上。

    金文有幾種用法：一、是一種容器、量器的名稱，典籍作“鐘”（此處不知是否原文有誤，一般字典都説“鍾”是容器，“鐘”是樂器），春成侯壺：“春成侯中府，爲重（鐘），冢（重）十八益（鎰）。”二、用作地名，井侯簋：“易（賜）臣三品：州人、重人、墉人。”三、用作姓氏，即典籍的“董”氏，十年洱陽令戈：“左庫工帀（師）重（董）棠”（參孫敬明、蘇兆慶）。

    戰國竹簡“童”通假作“重”，如《上博楚竹書二·容成氏》：“飤（食）不童（重）味”，表示吃飯時不用兩種菜餚。

    种〔chóng / ㄔㄨㄥˊ〕：姓。宋代有种師道。有的字書也釋爲“稚”，可能是“沖”的別字。參見種。

    “鐘”和“鍾”本來既可表示鐘鼓，又可表示酒器。“童”和“重”古音亦相近。如《列子·楊朱》：“朝之室也聚酒千鐘，積麴成封。”後來“鐘”專門用來表示樂鐘，“鍾”則表示酒器。（裘錫圭）

    《說文》以為“鍾”的本義是酒器。《說文》：“鍾，酒器也。从金，重聲。”徐灝《說文解字注箋》：“許以鍾爲酒器，鍾爲樂器，判然各異，但此二字古相通用。”

    “鐘”的意義比較單一，就是表示鐘錶或鐘鼓。

    “鍾”表示酒鍾，還表示聚集、匯聚，如《左傳·昭公二十一年》：“天子省風以作樂，器以鍾之，輿以行之。”杜預注：“鍾，聚也。以器聚音。”《國語·周語》：“澤，水之鍾也。”還表示專注、專一，如《世說新語·傷逝》：“聖人忘情，最下不及情；情之所鍾，正在我輩。”《隋書·蘭陵公主傳》：“蘭陵公主字阿五，高祖第五女也。美姿儀，性婉順，好讀書，高祖於諸女中特所鍾愛。”

    從字典上看，鍾是姓，鐘不是姓。南京紫金山又名“鐘山”，康熙字典作“鐘山”，但臺灣國語辭典作“鍾山”，搜不到鐘山這一名稱的來歷，但從一些遺蹟上的刻字看，應該是“鐘山”。

    纖：從“糸”，“韱”聲，本義爲細小。

    韱〔xiān / ㄒㄧㄢ〕：《説文》：“山韭也。从韭，㦰聲。息廉切。”

    㦰：《説文》：“絕也。一曰田器。从从持戈。古文讀若咸。讀若《詩》云“攕攕女手”。子廉切。臣鉉等曰：㦰，銳意也。故从从。”金文從从，從戈，造字本義不明。

    纖〔xiān / ㄒㄧㄢ〕表示細小；縴〔qiàn / ㄑㄧㄢˋ〕表示牽拉的繩子。兩字讀音意義皆不同，很好區分。

    簽：後起字，《説文》無。① 在文書上寫上意見或題寫姓名。② 通“籤”。官府交差吏逮捕犯人的籤牌。

    籤：① 削尖的小竹片。② 有特殊作用的竹片。《陳書·世祖紀》：“每雞人司漏，傳更籤于殿中，乃勑送者必投籤于階石上，令鎗然有聲。”③ 簽署意見或題寫姓名於簡。④ 特指寺廟中寫有字句等的卜具，以竹片製成。⑤ 古代漏壺上記時的竹籤，上有刻度。⑥ 用籤刺入。

    辨析：籤、簽二字皆爲清母鹽韻三等字。簽是後起字。表示署名、在文書上寫上意見或題寫姓名、用指籤牌等本作“籤”，後來通常作“簽”。表示刺入義和名詞卜具義時，通常用“籤”不用“簽”。

    “簽”實際是從“籤”分化出來的。現如今，表示簽署、簽字等抽象義時用“簽”；表示竹籤、抽籤、書籤等具象義時用“籤”。這兩個意義比較好分辨。唯獨“標簽”“便簽”這種詞用哪個字似乎不好分辨，臺灣的字典給出的解釋是“標簽”或作“標籤”，但其實看了上面“辨析”部分的內容後，這幾個詞到底應該用哪個字其實很清楚了。答案就是：本應作“標籤”，後來因爲“籤牌”必有題字，從這種角度講，亦可作“簽牌”，故“標籤”亦可作“標簽”。其實簽字、簽訂等義從“籤”中分化出來無可厚非，標籤、標簽的混亂算是其副作用吧。

    識：從“言”，“戠”聲。本義是記識、記得。

    戠：甲骨文從“戈”從倒三角形，倒三角形是“樴”的象形初文，“樴”即是“杙”（本字“弋”），象下端很尖的木樁，以便植入泥土中，在“戠”字中作爲聲符。本義是木樁。

    甲骨文及金文又加從“口”，在兩旁加點爲飾筆。《爾雅·釋宮》：“樴謂之杙。”郭璞注：“橜也。”邵晉涵《爾雅正義》説：“古謂之橜，又謂之樴，又謂之杙。其狀不一，或衺而銳，或大而長，其用至廣。”（參陳劍）

    甲骨文用作祭名，又表示牲畜毛色，過去又讀爲“蝕”或“食”，認爲“日有戠”表示日食，陳劍認爲不應表示日食，應讀爲“異”，表示自然界的異常現象，又讀爲“待”，表示等待（裘錫圭）。金文讀爲“識”，表示知識，何尊：“爾有唯小子，亡（無）戠（識）。”又通作“織”，“織衣”即“繒衣”（繒爲絲織品的總稱），表示衣服，豆閉簋：“易（賜）女（汝）戠衣。”

    雍、雝、擁（拥）、臃、癰（痈）、瓮（※甕） #

    甲骨文金文小篆隶书楷书行书草书

    “雝”與“雍”同字，甲骨文從“隹”從“水”從正方形，或從兩方形相疊，構形初義不明。

    一説方形是“宮”字初文，“雝”是璧雝（古代大學）的本字，象璧雝的宮外有水有鳥止息（羅振玉、陳邦懷）。一説方形是“雝”的初文，象連環形（于省吾）。一説象鳥足被環所扣，無法高飛。引申爲阻塞、壅塞等意思（徐中舒）。金文上承甲骨文字形，方形變圓，與“吕”形混。

    甲骨文用爲人名、地名。金文表示熟食，通“饔”，余阝王鼎：“用雝（饔）賓客”，表示用熟食宴饗賓客；表示壅塞，通“壅”，毛公鼎：“勿雝（壅）律庶民”；又用作象聲詞，逨鐘：“雝雝”。

    個人認爲兩口形訛變成了邑，邑又異寫成乡，水變爲巛（同川）。隸書第一例還能看到從巛從乡的混合體。可以理解“亠乡”是“邕”的連筆寫法。

    注意，“邕”可能與“雝”字裏的“邕”不是一字。有人認爲金文即“邕”字，從巛，從邑，像城邑的四邊有水環繞；有人認爲當釋爲“川阝”。

    户（※戶）、扈、滬（沪）、妒 #

    甲骨文金文小篆隶书楷书行书草书

    甲金文象一扇門之形，《玉篇》：“户，所以出入也。一扉曰户，兩扉曰門。”後由門户引申爲人家、住户。

    甲骨文用作本義，表示門户之户，“己巳卜，其啓庭西户”（《鄴》3.41.6）。金文用作族氏名。簡帛也用作本義，《馬王堆·老子甲本》：“不出於户，以知天下。”

    《説文》：“户，護也。半門曰户。象形。凡户之屬皆从户。㦿，古文户从木。”其中，户護比擬，屬於音訓。

    大陸以“户”爲正體，臺灣以“戶”爲正體，所有從“戶”的字，例如“房”“肩”等，頂上都寫作一撇。臺灣的電腦字體也是如此。只是本文用字體是大陸的，因此“房”“肩”等字頂上是點。之所以“户”和“戶”可以在同一字體中區分，是因爲這兩個字佔用了兩個字符編碼（即計算機能識別出是不同的字），字體的設計者必須爲這兩個字符編碼分別設計對應的圖形符號。

    我在書法字典軟件中查了下古人寫的楷書，“戶”比“户”多一些，不同的字不一樣，大體六四開吧。其他以“户”爲偏旁的字也是如此。

    懇，小篆字形從“心”，“豤”聲。又作“懇”，從“心”，“貇”聲。“懇”同“”。本義是真誠、誠懇。

    《正字通》：“，懇本字。”本義是真誠、誠懇。《説文新附》：“，悃也。从心，豤聲。康恨切。”唐·玄應《一切經音義》卷四引《通俗文》：“至誠曰懇。”如《後漢書·光武十王列傳》：“其後數陳乞，辭甚懇切。”《三國志·魏書·董二袁劉傳》：“豐懇諫，紹怒甚，以爲沮眾，械繫之。”

    《説文》：“豸，獸長脊，行豸豸然，欲有所司（伺）殺形。凡豸之屬皆从豸。池爾切。”段玉裁注：“許言獸者，謂凡殺物之獸也。《釋蟲》曰：‘有足謂之蟲，無足謂之豸。’按凡無足之蟲體多長，如蛇蚓之類。正長脊義之引伸也。”

    徐中舒認爲有足之猛獸如豹、貔、豺等字皆從“豸”，是知《爾雅·釋蟲》及段注所説皆非其本義。

    唐蘭首先釋讀此字爲“豸”。一説象貓科動物，長脊獸，如貓、虎之類（馬敍倫、馬如森）。豸本應讀作“貓”音，如“貌”從“豸”標聲。姑備一説。

    小篆字形從“土”，“豤”聲。隸楷字形從“土”，“貇”聲。本義是翻土耕作。《説文新附》：“墾，耕也。从土，豤聲。康很切。”如《管子·治國》：“民事農則田墾，田墾則粟多，粟多則國富，國富者兵彊，兵彊者戰勝，戰勝者地廣。”《韓非子·顯學》：“今商官技藝之士，亦不墾而食，是地不墾，與磐石一貫也。”

    舂、樁（桩） #

    甲骨文金文小篆隶书楷书行书草书

    舂〔chōng / ㄔㄨㄥ〕，甲骨文從“廾”（象雙手，在很多字形中隸變爲“大”）從棍棒（金文改棍棒爲“午”，“午”是“杵”的象形）從“凵”（金文改“凵”爲“臼”，其實“凵”是“臼”的初形，古人掘地爲“臼”），會雙手持杵在臼中舂搗之意。本義是舂搗。

    “臼”象舂搗穀物用的器皿，用石或木製成，中間凹下。古人把穀物置於臼中，兩手（廾）持棍棒或杵自上而下擊打，以去皮殼，或把穀米碎成粉末。《説文》：“舂，擣粟也。從廾持杵臨臼上。午，杵省也。古者雝父初作舂。書容切。”徐鍇《説文解字繫傳》作“舂，擣米也。”

    甲骨文“凵”形或以“口”形代替，表示中間凹的器皿之意同。

    樁〔zhuāng / ㄓㄨㄤ〕，從“木”，“舂”聲，本義爲木橛，也泛指楔入地中的樁柱。《説文》：“橛杙也。从木，舂聲。啄江切。” 唐·李白《大獵賦》：“下整高穨，深平險谷，擺樁栝，開林叢。”王琦注：“樁，杙也。”

    樁〔chōng / ㄔㄨㄥ〕，擊；刺。《玉篇·木部》：“樁，橦也。”

    塊（※凷）、屆（△届） #

    塊，本義爲土塊。《説文》小篆作“凷”：“凷，墣也。从土，一屈象形。苦對切。（塊）凷或从鬼。”

    屆，從“尸”，“凷”聲。本義是行動不便。《説文》：“屆，行不便也。一曰：極也。从尸，凷聲。古拜切。”段玉裁注：“此與艐義相近。艐，船著沙不行也。”張舜徽《説文解字約注》：“行不便者，謂行爲物阻，不得前進也。在人爲屆，在舟爲艐，其事一耳。行不得進，故屆字又有極義，謂有所止也。”例如《詩·小雅·節南山》：“君子如屆，俾民心闋。”俞樾平議：“言君子所行如不便於民，則上下之情不通而民之心閉矣。”

    “屆”表示極限、盡。《爾雅·釋言》：“屆，極也。”郭璞注：“有所限極。”如《詩·大雅·瞻卬》：“蟊賊蟊疾，靡有夷屆。”鄭玄箋：“屆，極也。”“蟊”和“賊”是害蟲。

    “屆”表示至、到。如《尚書·大禹謨》：“惟德動天，無遠弗屆。”孔傳：“屆，至也。”《文選·曹子建〈七啟〉》：“入乎泱漭之野，遂屆玄微子之所居。”

    “屆”可作量詞，相當於“次”、“期”。如徐珂《清稗類鈔·考試類·倪恩齡場前中進士》：“咸豐庚申會試，應試者不及歷屆之半，以粵寇肆擾，各省亂事未已，無力成行也。”茅盾《送考》：“接着她就訴説這學校每屆考試報名的如何多，取的如何嚴格。”

    羲、犧（牺）、曦 #

    甲骨文金文小篆隶书楷书行书草书

    “羲”字甲骨文從“我”（甲商金）從，金文從“義”（甲金）從，疑◎象截去頭的豬形，全字會宰殺犧牲以獻祭之意。另見義。

    甲骨文從“我”從，“我”是一種長柄及有鋸齒的戈形兵器，疑象截去頭的豬，兩小點表示宰割的碎片，全字是宰殺犧牲以獻祭之義（秦建文）。金文從“義”，秦建文認爲字形象既宰豬又宰羊，姑備一説。按“義”應是“羲”的聲符。此説的疑點在於宰殺犧牲應該用刀而非用戈。

    甲骨文用作地名，金文用作人名，直至戰國的詛楚文，才出現犧牲一詞，“羲”是“犧”的初文，詛楚文：“圭玉羲（犧）牲”。圭玉和犧牲皆是祭品，故相提並論。《説文》：“犧，宗廟之牲也。从牛羲聲。賈侍中説：此非古字。”《説文》：“羲，气也。从兮，義聲。”

    “伏羲”是古帝名，傳説中的三皇之一。他創造了八卦和書契，教導人民狩獵、捕魚和放牧，又將祭祀用的犧牲、牲畜拿來烹調，以供庖廚，所以“伏羲”又稱“庖犧”。相傳“伏羲”、“女媧”都是人類的始祖。

    驚（惊）、警 #

    > “敬”字大陸和臺灣的寫法不同，大陸都類化爲草字頭了。
    金文 茍金文 敬小篆隶书楷书行书草书

    驚，從“馬”，“敬”聲，本義爲馬受突然的刺激而行動失常。《説文》：“馬駭也。从馬，敬聲。舉卿切。”《左傳·襄公二十八年》：“慶氏之馬善驚。”

    敬，金文從“攴”，“茍〔jì / ㄐㄧˋ〕”聲，“茍”（非苟）是“敬”的初文，本義是恭敬、肅敬。《説文》：“敬，肅也。从攴、茍。居慶切。”中山王方壺或加“心”爲意符，作“憼”。戰國竹簡“茍”字或改爲“羌”聲（“羌”字從“羊”聲）。

    金文表示恭敬、敬慎，又通假爲“警”、“儆”，表示警告、使戒懼，中山王方壺：“以憼（警）嗣王。”《睡虎地秦簡·日書甲種》簡85背1：“外有火敬（警）”。《睡虎地秦簡·秦律十八種》簡196：“慎守唯敬（儆）”，意謂謹慎警戒。

    “警”“儆”互通。《説文》：“警，戒也。从言从敬，敬亦聲。居影切。”“儆，戒也。从人，敬聲。《春秋傳》曰：儆宮。居影切。”

    茍〔jì / ㄐㄧˋ〕，甲金文從“羊”省從“卩”，與“羌”字形近，“茍”疑象羌族人恭敬跪坐之狀，是“敬”的初文。古代羌人被殷人當作奴隸，故以羌人來表示恭敬。“羌”兼充當“茍”的聲符（于省吾）。後引申爲謹慎、儆戒、警惕。“羌”與“茍”的分別在於“羌”字的人形是站立的，“茍”字的人形是跪坐。

    甲骨文表示警戒，《合集》20390：“令茍（警）舊（師）”，意謂命令告戒舊地的軍隊。又用作地名。地名未必能反映“茍”的原義，反而可能是因爲“茍”的正面意義而以之爲名。

    金文表示恭敬，大保簋：“大保克茍（敬）亡（無）遣（譴）”，意謂太保能自敬慎，沒有災譴。師虎簋：“茍（敬）夙夜勿灋（廢）朕令”，意謂朝夕虔敬，不要荒廢我的命令。又用作人名。

    《説文》：“茍，自急敕也。从羊省从包省从口。口，猶愼言也。从羊，羊與義、善、美同意。凡茍之屬皆从茍。己力切。，古文羊不省。”許説中的“敕”是警誡的意思，而“自急敕”則有“自省”、“自我警惕”的意味，這與後起的“敬”、“警”等字都有意義上的聯繫。

    “茍”與“苟”易相混。《説文》：“苟，艸也。从艸，句聲。古厚切。”

    洶（汹）、胸（※胷） #

    匈，从“勹”，“凶”聲，是“胸”的古字，金文有字，從“凶”從“肉”，與《説文》或體同形，構形初義不明。《説文》：“匈，聲也。从勹，凶聲。許容切。，匈或从肉。”

    金文通讀作“容”，表示盛載。少府銀圜器：“（容）二益（鎰）”。

    據《説文》，乃“匈”之或體，《史記·高祖本紀》：“項羽大怒，伏弩射中漢王，漢王傷匈。”

    “匈”也通作“兇”，表示兇惡。《漢書·佞幸傳·石顯》：“顯聞眾人匈匈，言己殺前將軍蕭望之。”

    “匈”也作爲“匈奴”的指稱，同時可泛稱北方少數民族。《舊唐書·突厥傳上》：“又匈虜一敗，或當懼而修德，結怨於我，爲患不細。” 西方亦以Hunnorum （拉丁文）、Huns（英文）、Hunnen（德文）、Hunové（捷克文）指稱從中亞遷徙過去的民族。但歐洲的“匈人”是否直接承襲自“匈奴人”，學界存在爭議，姑備一説。“匈奴語”歷史文獻並不豐富，其性質與來源亦在學界存在爭議。

    另見“凶”。

    洶，從“水”，“匈”聲，本義爲水波騰涌貌。《説文》：“涌也。从水，匈聲。”唐杜甫《水會渡》：“大江動我前，洶若溟渤寬。”

    “汹”字始見《漢隸字源·上聲·腫韻》引《周憬功勳銘》，《隸辨·上聲·腫韻》“汹”字下注曰：“《説文》作‘洶’，碑省從‘凶’。”其後《集韻·上聲·鍾韻》、《類篇·水部》均以爲“洶”之或體。

    胸，從“月”，“匈”聲，本義爲頸、腹之間，軀幹的一部分。《周禮·考工記·梓人》：“以胸鳴者。”唐溫庭筠《南歌子》：“手裏金鸚鵡，胸前繡鳳凰。”

    “胷”同於“胸”，表示軀幹的一部分。《字彙·肉部》：“胷，同胸。”大陸和臺灣皆以“胸”爲正體。

    膚（肤） #

    金文小篆籀文楷书行书草书

    金文從“肉”從“虍”從，“虍”是“虎”的省文，在“膚”字中作爲聲符，“肉”是義符，表示皮肉。

    金文用作人名。戰國簡帛用作本義，表示皮膚，《郭店楚簡·唐虞之道》：“脂膚血氣”，《上博楚竹書二·魯邦大旱》：“夫山，石以爲膚，木以爲民。”

    “膚”是“臚”的異體字，《説文》：“臚，皮也。从肉盧聲。力居切。膚，籒文臚。”

    “盧”“慮”“虜”“膚”這幾個字都以“虍”爲聲符，但中間這個“田”其實來源不同，是形似部件的類化。“盧”字的“田”是“爐”的象形（參見盧）；“慮”底部是“思”（參見慮）；“虜”字的“田”是“毌〔guàn / ㄍㄨㄢˋ〕”（貫的上部，將物體穿在一起，臺灣標準仍寫作毌）（參見虜）；“膚”字的“田”我沒有找到解釋。

    慮（虑）、濾（滤） #

    金文小篆隶书楷书行书草书

    慮，金文有兩種字形，第一種（上曾大子鼎，春秋早期）從“心”，“膚”聲，第二種（中山王鼎，戰國晚期）從“心”，“呂”聲。本義是思慮。

    張世超等認爲“膚”古讀如“盧”，與“呂”同音。按金文鄂君啟舟節的“氵膚江”，讀作“廬江”。兩種字形是“慮”的聲符更換字。秦簡、小篆字形下部訛作“思”。“心”是義符，古人認爲心是思慮的器官，《孟子·告子上》：“心之官則思，思則得之，不思則不得也。”

    金文用義有三。一、表示思慮、謀劃，中山王鼎：“（謀）慮皆從”。《爾雅·釋詁》：“慮，謀也。”《説文》：“慮，謀思也。从思，虍聲。良據切。”《禮記·大學》：“安而後能慮”。二、善慮也，般殷鼎：“心聖若慮”，表示心靈通悟而又有深謀遠慮（參孫敬明、何琳儀、黃錫全）。三、憂也，中山王鼎：“亡（無）遽惕之慮”，《孫臏兵法·十問》：“三軍之士皆勇而毋慮。”張震澤注：“毋慮即無慮，謂無所憂慮。”

    按小篆變爲從思、虍聲，這個字的簡化方式當歸爲意符替換，但爲了和“膚”“虜”相呼應，故放在此處。

    爺，從“父”，“耶”聲，本義爲父親。古樂府《木蘭詩》：“軍書十二卷，卷卷有爺名。”《正字通》：“爺，以蛇切。音耶。俗謂父爲‘爺’。古《木蘭詩》阿爺無大兒。《杜甫詩》爺孃妻子走相送。通作‘耶’。”可見“爺”爲後起字。

    犀、遲（迟） #

    甲骨文金文小篆隶书楷书行书草书

    “遲”甲骨文從“彳”，“屖”聲，金文作“遟”，從“辵”，“屖”聲。“彳”、“辵”意思相通，皆有行走之意，本義是慢慢地行走；引申爲慢、久、晚、遲到、延遲、逗留、停滯、遲鈍等義。“遲”字又有安適、閒雅之義，參見“屖”。小篆把“遟”的聲符“屖”改換爲“犀”聲，成爲現在的“遲”字。戰國竹簡“遲”從“”聲，寫作“”，是“遟”、“遲”的異體，《説文》：“遲，徐行也。从辵，犀聲。《詩》曰：‘行道遲遲。’直尼切。，遲或从。，籒文遲从屖。”

    屖〔xī / ㄒㄧ〕，金文，從“尸”（象蹲坐的人）從“辛”（象兵器），本義疑爲受傷，是“夷傷”之“夷”的初文（何琳儀）。引申爲堅硬、遲緩等義。後來堅硬義借“犀”來表示，遲緩義作“遲”。

    金文多從“貝”，“商”聲。古代“賞”、“商”本同一字，應是假借“商”作“賞”，後加“貝”分化出“賞”，目的是突出“賞”涉及財貨的賜予。本義是賜予。

    金文中，賞賜字商末周初作“商”，後加形符“貝”以示財貨的賜予，入東周則字多作“賞”。償還之“償”西周寫作“賞”，東周早期未見此字。睡虎地秦簡字作“賞”，與賞賜之“賞”同形。曶鼎“賞”字從“尚”省聲，中山王方壺“貝”省作“目”形。《説文》：“賞，賜有功也。从貝，尚聲。書兩切。”

    鑽（钻） #

    鑽，從“金”，“贊”聲。本義是穿孔的工具，又表示穿孔、刺穿。《説文》：“鑽，所以穿也。从金，贊聲。”

    贊，小篆。戰國秦系文字從“貝”從“㚘”，小篆“㚘”形訛爲“兟”形，遂寫成“贊”。本義是輔佐、輔助。《説文》：“贊，見也。从貝从兟。”段玉裁注：“此以疊韻爲訓，疑當作所以見也。謂彼此相見必資贊者。《士冠禮》‘贊冠者’、《士昏禮》‘贊者’注皆曰：‘贊，佐也。’《周禮·大宰》注曰：‘贊，助也。’是則凡行禮必有贊，非獨相見也。”如《左傳·昭公元年》：“國無道而年穀和熟，天贊之也。”《周禮·秋官·司民》：“內史、司會、冢宰貳之，以贊王治。”鄭玄注：“贊，佐也。”

    運（运） #

    運，從“辵”，“軍”聲，本義爲移動。《説文》：“迻徙也。从辵，軍聲。王問切。”

    “軍”金文作，从車，勹（螾）聲或匀聲。金文表示軍隊，中山王鼎：“親率三軍之眾，以征不宜（義）之邦”。

    氈（毡）、顫（颤）、壇（坛）、檀 #

    《説文》：“，撚毛也。从毛，亶聲。諸延切。”

    亶，從“㐭”，“旦”聲。本義是穀物多。《説文》：“，多穀也。从㐭，旦聲。”如《詩·小雅·十月之交》：“擇三有事，亶侯多藏！”楊樹達云：“《詩》文言多藏，故以訓多穀之亶狀之。‘亶侯’者，猶言‘亶兮’也。”

    《説文》：“㐭，穀所振入宗廟粢盛，倉黃㐭而取之，故謂之㐭。从入、回，象屋形，中有户牖。凡㐭之屬皆从㐭。力甚切。廩，㐭或从广从禾。”

    壇、罈合併簡化爲坛，詳見“壇、罈”。

    襪（袜）、袜（袜胸） #

    袜:〔mò / ㄇㄛˋ〕，袜胸

    襪，從“衣”或從“韋”，“蔑”聲，本義爲襪子，即穿在腳上的衣物。《釋名·釋衣服》：“襪，末也，在腳末也。”《集韻·月韻》：“韤，《説文》‘足衣也。’或從衣。”漢·張衡《南都賦》：“脩袖繚繞而滿庭，羅襪躡蹀而容與。” 詳見蔑。

    袜〔mò / ㄇㄛˋ〕，婦女所使用的抹胸、肚兜。隋·煬帝·喜春遊歌：“錦袖淮南舞，寶袜楚宮腰。”或稱爲“袜胸”、“袜腹”。

    《康熙字典》：“《唐韻》莫撥切。《集韻》莫葛切，並音末。《廣韻》袜肚。《類篇》所以束衣也。《隋煬帝詩》錦袖淮南舞，寶袜楚宮腰。又《集韻》勿發切，音韈。《類篇》同襪。《玉篇》脚衣。《漢·雜事祕辛》約縑迫袜，收束微如禁中。”

    钁（䦆）、鐝（镢） #

    钁，① 大鋤。② 挖掘。《説文》：“钁，大鉏（鋤）也。从金，矍聲。居縛切。”

    鐝，《龍龕手鑑》：“鐝俗，正。其月反。”《集韻》：“，磨也。”

    古代字書普遍認爲“鐝”同“”，是磨的意思，并沒有提及“鐝”與“钁”的關係。臺灣異體字字典也未提及二字之間的關係，也認爲“鐝”是“”的異體。《王力古漢語字典》中無“鐝”字。1956年簡化字總表裏“鐝”簡化成“镢”，“钁”簡化成“䦆”，被分爲兩個字，但現在大陸的字典裏兩個字是相同的意思，有些字典甚至將“镢”的繁體字標注爲“钁”。

    個人認爲，“钁”就是“镢头”的“镢”的繁體字，而“鐝”就是的異體。簡化字總表裏把“钁”和“鐝”分別簡化可能是工作不細緻，也可能當時就是認爲“钁”和“鐝”是兩種不同的農具，抑或者就是想把“镢”當的簡化字，只是後來編寫字典的人把“䦆”“镢”两字弄混了？
