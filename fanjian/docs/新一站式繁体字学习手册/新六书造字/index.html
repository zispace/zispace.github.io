    新六书造字 #

    國（国）、摑（掴）、幗（帼）、蟈（蝈） #

    甲骨文金文小篆隶书楷书行书草书

    “國”字草書的筆順是：“或”字的橫和戈勾連寫，然後再逆向起筆順時針畫出“口”字和一撇，最後寫“口”字下的一橫。“戈”在草書中經常先寫橫再寫彎勾，“成”字的草書亦是如此，只不過彎勾寫成豎勾了。

    金文從“囗”（〔wéi / ㄨㄟˊ〕），“或”聲。“囗”象國家的邊界、領域。本義是國家。戰國文字或不從“囗”而從“匚”“”（），是表示區域、邊界的義符。“或”是聲符。《説文》：“國，邦也。从囗从或。”段玉裁注：“《邑部》曰：‘邦，國也。’按邦、國互訓，渾言之也。《周禮》注曰：‘大曰邦，小曰國。’邦之所居亦曰國，析言之也。”“《戈部》曰：‘或，邦也。’古或、國同用。邦、封同用。”徐灝曰：“邦謂之國，封疆之界謂之域，古但以或字爲之。”如甲金文用“或”來表示“國”，何尊：“宅茲中或（國）”，表示居住在國家中部。古代“國”“域”音義皆近，故一説應釋爲“宅茲中或（域）”，表示在中心的區域居住。從“囗”的“國”見於西周，金文用作本義，彔卣：“淮尸（夷）敢伐內國”。蔡侯鐘：“建我邦國”。

    以前的俗字一般寫作“囯”，裏面是“王”；從“玉”可能有美好的意思；有人説宋元就有“国”字，我沒找到；不管怎麼説，一般認爲這個簡化字是個會意字。

    或：甲骨文，商朝金文西周早期。甲骨文象有納柲的“管銎斧”或“柲帽”之形，是一種兵器。（李學勤、謝明文）商代晚期的金文與甲骨文同，後來柲形與“◎”分離。謝明文認爲“◎”的脫落可能是變形音化，以“◎”（“員”的初文）作爲“或”的聲符。“◎”的周圍或加上飾筆，“柲”形又演變爲象“戈”形，爲《説文》小篆所本。後來的人根據訛變的字體，又誤把疆域、國家之類的假借當作“或”的本義或引申義，於是把“或”附會成一個會意字，故《説文》云：“或，邦也。从囗从戈，以守一。一，地也。”

    竄（窜）、攛（撺）、鑹（镩）、躥（蹿） #

    竄：小篆。從“鼠”，從“穴”，本義爲隱藏。《説文》：“（墜）[匿]也。从鼠在穴中。”《左傳·定公四年》：“天誘其衷，致罰於楚，而君又竄之。”杜預注：“竄，匿也。”

    類似的會意字還有“突”，會犬從穴中突然竄出之意，表示突然、忽然。

    “窜”爲從穴、串聲的形聲字。

    眾（众，※衆） #

    大陸以“衆”爲正體，港臺以“眾”爲正體。個人以爲以“眾”爲正體更合適。

    甲骨文金文小篆隶书楷书行书草书

    甲骨文從“日”從三“人”，象眾人在日下勞作，本義是眾人，引申爲眾多。李孝定云：“字從日者，蓋取眾人相聚，日出而作之意。”金文所從“日”形或訛變爲“目”，爲小篆所本。《説文》：“眾（衆），多也。从乑、目，眾意。”

    “众”其實不是古字形，是俗字，從三人會意。

    陰（阴）、蔭（荫） #

    甲骨文金文小篆隶书楷书行书草书

    “陰”從“阜”，“侌”聲，本義指山的北面，水的南面，皆是背向陽光之處，故稱“陰”。而山的南面，水的北面，是陽光照射之地，故稱“陽”。

    從“阜”的字多與山有關，而“侌”從“今”從“云”，“今”是聲符，從“云”可能是因爲多云的天氣即稱爲“陰天”的緣故。金文用作本義，敬事天王鐘：“江漢之陰陽”，表示長江、漢水的南面和北面。㠱伯盨：“其陰其陽，以征以行。”又用作地名。

    楚簡的“陰”字不從“阜”，直接寫作“侌”，《郭店楚簡·太一生水》簡2：“是以成侌昜。”“侌昜”即“陰陽”。《説文》：“陰，闇也。水之南，山之北也。从阜侌聲。”

    甲骨文以爲“陰”，從今省從隹，原爲一種鳥名，在甲骨文裏借爲陰晴之陰。

    金文有從水從酓聲（㱃（飲本字）省文）和從阜從酓聲的“陰”字。

    《説文》有字：“雲覆日也。從雲今聲。於今切。（侌），古文或省，，亦古文。”

    “阴”“阳”實爲俗字，以日陽月陰會意，同時保留了原字的“阝”旁，屬表意特徵字。另見陽。

    塵（尘） #

    甲骨文，漢帛書似從“土”從三“鹿”省（省去下部，只剩鹿首），《説文》小篆則從“麤”從“土”，會眾鹿奔跑，塵土飛揚之意。本義是塵土。

    “尘”是俗字，以小土會意。

    傘（伞，※繖） #

    《説文》作“繖”：“繖：蓋也。从糸，散聲。蘇旱切。”“傘”象幕蓋之形，本義爲傘蓋，古代儀仗用具的一種。《集韻》：“繖，蓋也。或從巾，亦作傘。”

    “伞”是將“傘”簡略後的象形字。

    郵（邮） #

    從“邑”從“垂”（邊陲），本義是驛站，即古代傳遞文書、官員來住及運輸時用作中途休息的地方。《説文》：“郵，境上行書舍。从邑、垂。垂，邊也。”如《孟子·公孫丑上》：“德之流行，速於置郵而傳命。”《睡虎地秦簡·秦律十八種》簡2-3：“近縣令輕足行其書，遠縣令郵行之。”意謂距離近的縣，文書由走得快的人專程遞送，距離遠的縣由驛站傳送。

    垂：小篆，從土，聲，本義爲邊際、邊疆，後作“陲”。《荀子·臣道》：“邊境之臣處，則疆垂不喪。”楊倞注：“‘垂’與‘陲’同。”

    ：《説文》：“艸木華葉。象形。凡之屬皆从。是爲切。，古文。”

    “邮”爲從邑，由聲的形聲字。

    竈（△灶） #

    > 結構：穴土黽。大陸以“竈”爲正字，港臺以“灶”爲正字，後者實爲俗字。
    甲骨文金文小篆隶书楷书行书草书

    金文第一例伯作井姬鼎，從穴從火；第二例秦公簋（春秋中期偏晚），從穴從黽，與小篆字形同。金文還有從穴從火，告聲的字（公孫子編鎛，戰國中期）。金文用作本義，即生火做飯的竈。

    《説文》：“：炊竈也。从穴，鼀省聲。則到切。，竈或不省。”

    “鼀：圥鼀，詹諸（蟾蜍）也。其鳴詹諸，其皮鼀鼀，其行圥圥。从黽从圥，圥亦聲。七宿切。，鼀或从酋。”

    《康熙字典》：“《説文》炊竈也。亦作窖。”

    “灶”爲俗字，從火從土，牽強會意，不如從穴從火。

    窮（穷） #

    甲骨文金文小篆隶书楷书行书草书

    本作“竆”，從“穴”，“躳”聲。因爲“躳”又作“躬”，故寫作“窮”。出土文獻《郭店簡·唐虞之道》簡2的字形從“宀”從“躳”，“宀”和“穴”兩個偏旁在古文字中常常通用。竹簡中還有從“穴”（或從“宀”）從“身”的字形，亦讀爲“窮”，一般認爲是“竆”的省形。（參何琳儀）如《郭店簡·老子乙本》簡14“其甬（用）不[宀身]（窮）”。

    “窮”的本義是極、盡、完結。《説文》：“竆，極也。从穴，躳聲。”如《尚書·微子之命》：“作賓于王家，與國咸休，永世無窮。”《孔傳》：“爲時王賓客與時皆美，長世無竟。”“無窮”和“無竟”皆表示無盡、無限。又如《史記·秦始皇本紀》：“後世以計數，二世三世至于萬世，傳之無窮。”

    “穷”爲俗字，應該是以力在穴中被困會意。

    《説文》：“躳：身也。从身从呂。居戎切。躬，躳或从弓。”

    竊（窃） #

    甲骨文金文小篆隶书楷书行书草书

    戰國楚系文字從“米”從“攴”，“禼”聲，見包山楚簡和清華簡（字形都是這樣，我是辨認不出）。也有省去“米”旁，從“攴”，“禼”聲，字形見郭店簡（）。《説文》小篆從“穴”從“米”，“禼”“廿”皆聲。從“廿”聲的寫法未見其他材料。馬王堆帛書字形從“萬”聲，“萬”和“禼”是音近的聲符。本義是偷盜。如《左傳·僖公二十四年》：“竊人之財，猶謂之盜，況貪天之功以爲己力乎？”《清華壹·楚居》簡4-5：“乃[米禼攴]（竊）鄀人之穜以祭。”

    禼：《説文》：“：蟲也。从厹（），象形。讀與偰同。私列切。，古文禼。”“竊”字所從的“禼”是一種變了形的寫法。

    點（点） #

    從“黑”，“占”聲，本義爲細小黑色斑痕。《説文》：“小黑也。从黑，占聲。多忝切。”

    “点”爲俗字，表音特徵字，省黑的上半部，將灬理解爲意符，以占爲聲。

    淚（泪） #

    從“水”，“戾”聲，本義爲眼淚。

    戾：從“犬”，從“户”，以犬從門户之下曲身而出會意，本義是屈曲，引申表示違背、暴虐、災禍等。《説文》：“戾，曲也。从犬出户下。戾者，身曲戾也。”用作屈曲意的，如《呂氏春秋·盡數》：“飲必小咽，端直無戾。”《齊民要術·種榆白楊》：“既非叢林，率多曲戾。”

    “泪”爲俗字，從水從目會意。

    繭（茧） #

    大陸標準寫作艹頭即可。

    未見到秦簡牘之前的文字。睡虎地秦簡甲一三背，小篆。

    《漢語多功能字庫》：從“糸”，從“虫”，從“黹”省，本義爲蠶或其他昆蟲成蛹前吐絲做成的殼。《説文》：“蠶衣也。从糸从虫，黹省。古典切。絸，古文繭从糸、見。”

    黹：甲骨文，金文，小篆。甲金文象刺繡花紋之形，本義是針線繡成的花紋。屈萬里認爲是“黼”的初文。另見帶。

    “茧”算是半表意特徵字。

    獻（献） #

    > 結構：虍鬲犬。
    甲骨文金文小篆隶书楷书行书草书

    甲金文從“鬳”從“犬”，“鬳”“犬”皆是聲符。“鬳”是“甗”的初文，是盛食物的器皿，用來祭祀，表示用甗進獻食物之意。本義是進獻。甲骨文用義不詳。金文表示進獻，羌伯簋：“獻帛”。《説文》：“獻，宗廟犬名羹獻，犬肥者以獻之。从犬，鬳聲。”

    “献”算是以南爲聲特徵字。

    鬳〔yàn / ㄧㄢˋ〕：甲骨文，金文。甲金文從“鬲”，“虍”聲（象虎頭），是爲“甗”而造的形聲字，本義是古代炊器，後加“瓦”爲意符。

    “鬳”字以“虍”標聲後，所從之甗形多類化爲“鼎”。後來“鬳”字加從“瓦”而爲“甗”，以表器皿之意。

    甲骨文讀爲“獻”，表示進獻，《合集》26954：“獻羌”，表示進獻羌人。

    寶（宝，※寳） #

    甲骨文金文小篆隶书楷书行书草书

    寶字甲骨文作貝、玉在屋裏之形。貝、玉於商代作貸幣用，置於屋內，表示貯藏財富，有藏寶之意，或會珍重、寶愛之意。《説文》：“寶，珍也。从宀从王从貝，缶聲。”

    金文寶字多有加“缶”作聲符（“缶”的上部或聲化爲“枹”的初文），小篆承此形。或增“畐”爲聲符。又，“缶”、“畐”同見於一字（如楚公家鐘），屬多聲之例。另有字形只從宀從缶，省去玉、貝（見杞白每氏壺）。

    “寶”有個異體字“寳”，貌似是將“缶”替換爲“尔”，實際是從宀從珍從貝。

    “宝”是俗字，從宀從玉會意。

    開（开） #

    甲骨文金文小篆隶书楷书行书草书

    金文“開”從“閂”從“廾”，“廾”從二“又”，象兩手之形，“門”上有橫木，即門閂，全字會兩手除去門閂開門之意，本義是開門、門啟。金文與《説文》古文（金文圖例實在看不清）同形，用作人名，元年春平侯矛：“冶韓開”，表示姓名叫做韓開的冶煉金屬的工匠。“開”字的構形初義與“闢”字相近，只是“開”比金文“闢”字多了鎖門用的橫木（門閂），參見闢。

    “开”相當於省去門型只保留門栓的會意字。

    滅（灭） #

    甲骨文金文小篆隶书楷书行书草书

    書法在寫戊字結構時，一般都是先寫左丿，再寫一，再寫被戊字包圍的結構，再寫戈勾，再寫戈丿，最後寫丶。草書在寫戊字結構時，一般會先寫左丿，再寫一，再寫戈勾（有時甚至寫成豎），再寫被戊字包圍的結構，再寫戈丿，最後寫丶或不寫。從行書和草書字例的遊絲引帶可以看出一二。

    從甲骨文字形可以看出是手持一物扑火之形。舊《一站式》説“所執之物形似甲骨文的‘戌’（斧鉞一類的兵器），金文寫作‘烕’或加‘水’寫作‘滅’。”我不記得從哪裏摘抄的了。現在再查《漢語多功能字庫》，只有如下解釋：從“水”，“烕”聲，本義爲除盡。《説文》：“盡也。从水，烕聲。”《易·剝》：“剝床以足，以滅下也。”

    “灭”相當於用蓋子壓住火的會意字。

    護（护） #

    “护”從手，户聲。詳見護。
