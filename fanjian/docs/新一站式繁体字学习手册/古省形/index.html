    古省形 #

    麗（丽）、儷（俪）、灑（洒）、洒、曬（晒）、驪（骊）、鸝（鹂） #

    甲骨文金文小篆隶书楷书行书草书

    甲金文象一只鹿有双角之形，特别强调鹿头上的一对美丽的鹿角，取义于丽角成双成对，本义是两个、一双、偶。《小尔雅·广言》：“麗，兩也。”《玉篇》：“麗，偶也。”由漂亮的丽角引申，“麗”有美丽、美好之意。

    古文字为了描画一事物，往往把与该事物相连或相关的事物都画出来，如为表示眉毛之“眉”，把眼睛也画出来。“麗”字以一对丽角表示一双之意，把鹿的身体也画出来，才能清楚显示所画的是鹿角，《说文》古文和篆文应该是鹿角之形的讹变。金文第二例（战国·陈丽子戈）与说文古文同。

    区分洒、洗、灑：

    洒，从“水”，“西”声，本义为洗涤，为后世“洗”之本字，今“洒”又作为“灑”的简化字，意谓把水或其他物质散开。《说文》：“洒，滌也。从水，西聲。古文爲灑埽字。先禮切。”段玉裁注：“洒，涤也。下文云：‘沬，洒面也。浴，洒身也。澡，洒手也。洗，洒足也。’今人假洗为洒，非古字。……”

    洗，本义为洗足。《礼记·内则》：“足垢燂汤请洗。”甲骨文为洗的初文，象在桶中洗“止”。

    灑，《说文》：“汛也。从水，麗聲。山豉切。”“汛”的本义是洒水，季节性的涨水为晚起义。《新唐书·房玄龄传》：“玄龄敕子弟汛扫庭唐。”

    《论衡·讥日》：“沐者去首垢也，洗去足垢，盥去手垢，浴去形垢。”“沬”是洗面。“洗涤”的意义本写作“洒”，后来“洗”的词义扩大，由“洗足”变为一般意义的洗涤，就替代了“洒”。“洒”则多用于“洒〔sǎ / ㄙㄚˇ〕水”之义。

    总结：现在，洗就是洗涤；表示洒水义时，洒是灑的异体（大陆和港台都定灑为正字，洒为异体），但在表示洒家一词时，洒为正体，好像没有写作灑家的。

    晒为曬的俗字。

    爾（尔）、璽（玺，※壐）、邇（迩）、彌（弥）、獼（猕） #

    甲骨文金文小篆隶书楷书行书草书

    甲金文象三足的绕丝、缠丝、纺线的架子，上有锐头，中有器身，下有竖足，并有丝线围绕。本义是绕丝、纺线的架子（络丝架、络子），是“檷”的初文（林义光），后借用为第二人称代词。金文“爾”的上部写作“尔”，“尔”其实从“爾”字分化出来，是通过省去“爾”字下半部分而成。“爾”“尔”本为一字，故简化字用“尔”来表示“爾”。许慎认为“爾”的本义是“麗爾”，“麗爾”即“麗廔”，表示窗棱交错，格格相连的样子，此义罕用（张标）。《说文》：“爾，麗爾，猶靡麗也。从冂从㸚，其孔㸚，尒聲。此與爽同意。”

    “汝”和“爾”在春秋以前有粗略的分工，“汝”（古作“女”）多用于主语和宾语位置，如《左传·僖公四年》：“五侯九伯，女实征之。”《左传·定公四年》：“余将杀女”；“爾”多用作定语，《论语·公冶长》：“各言爾志”。而“汝”多用于单数，如《论语·为政》：“子曰：‘由﹗诲汝知之乎﹗知之为知之，不知为不知，是知也。’”意谓孔子说：“子路，教你对待知或不知的正确态度吧﹗知道就是知道，不知道就是不知道，这就是聪明智慧（杨伯峻）；“爾”多用于复数，《书·汤誓》：“王曰：‘格尔众庶，悉听朕言。’”意谓王说：“你们过来，听我说话。”这只是粗略的分工，也有例外的情况（王凤阳）。

    战国竹简第二人称代词“爾”字多写作“尔”，《郭店简·缁衣》简3：“《寺（诗）》员（云）：‘情（靖）共尔（爾）立（位），好氏（是）贞（正）植（直）。’”此句见于《诗经·小雅·小明》，意谓《诗经》云：“安分敬守你的职位，只喜欢正直。”（参刘钊）。战国秦简用“壐”来表示爾，《睡虎地秦简·日书甲种》简25背贰：“鬼恒召（诏）人曰：“壐（爾）必以（某）月日死。”意谓鬼常常告诉人说：“你必在某月某日死去。”

    璽，小篆从土、爾声，战国文字从金或土、尔声。楷书从玉，与籀文同。

    處（处，※処）、咎 #

    甲骨文金文小篆隶书楷书行书草书

    “處”字金文從“人”從“几”，“虍”（字象虎頭形，突出虎口，本義爲虎頭）聲，會人坐在几上休息之意，本義是居處、止息。多於人形下加“止”，噩君啟車節省去“虍”。一説象虎盤踞之形，字右下方的几原應指虎的後腳（孫雲鶴）。

    金文用作本義，表示居處，中山王圓壺：“不能寧處”。又表示居住，牆盤：“青（靜）幽高且（祖），才（在）微霝（靈）處。”

    《説文》：“処，止也。得几而止。从几从夂。處，処或从虍聲。”。“處”字後來引申爲事情發生的所在、地方，處所等意思。

    按《漢語多功能字庫》的例子，“処”只見一例，是省文，故將此簡化方式定位古簡寫。

    “咎”上部和“処”無關，只是簡化字“处”的原因使人們覺得“咎”的上部同“処”。“咎”的上部從“人”，和“臥”一樣，大陸都定爲從“卜”，臺灣標準字還保持從“人”。

    咎：甲骨文，金文（商）（戰國晚期）。甲骨文從“夊”從“人”，金文從“人”從“各”。“夊”象向下的腳，早期甲骨文“夊”多寫在“人”的上面。一説象用腳踐踏人命（許進雄、劉興隆），表示滅頂、滅身之災；一説“夊”或“各”自外反加諸“人”身，表示遭逢不順遂或橫逆的事情（關子尹）。

    “咎”字的本義是災禍，但構形未有定論。第一説象足踩人頂，表示殺身之禍，較爲直接。第二説有金文字形和甲骨辭例、古書爲證，亦有道理。查《説文》對“咎”的解釋爲：“咎，災也。从人从各。各者，相違也。”此中所謂“相違”，概指臨於人身上的各種不順遂的或橫逆的事情解。首先，甲骨文從“夊”從“人”本已可指有事情及於人身，而金文“咎”從“各”，“各”的本義即是來臨，對於“人”而言，有“逆來”之意，即可指人所遭逢的或“衝著人而來”的事情。雖然人的遭逢可包括吉凶，但大抵由於約定俗成或系統的使用，“咎”字後來所涉之各義項，如災禍、責備、憎惡、甚至過失，都帶有《説文》所指“相違”這一負面的意思。甲骨文多記錄神祇降災咎給人，古書亦多有降咎、降災的記載。《尚書·呂刑》：“降咎于苗”，指降災於苗民。《尚書·大禹謨》：“天降之咎”。總的而言，“咎”是一極爲抽象的會意字（關子尹）。

    各：甲骨文，金文。甲骨文“各”象腳朝向洞穴，表示人來到和進入洞穴，本義是到來，與“出”相反。甲骨文的“各”字從“凵”（或“口”）從“夊”（象腳趾向下的腳），金文則從“口”從“夊”。“凵”或“口”象洞穴，古人穴居，“各”字象腳朝向洞穴，表示人來到和進入洞穴，與“出”的意思相反。“各”是“”的初文，本義是來到。《方言·一》：“，至也。”《方言·二》“，來也。”（裘錫圭） 甲金文或從“彳”旁，表示在道路的旁邊。由外而至到訪家中的稱爲“客”。“各”“”字“進入”之義古書多借“格”“徦”字表示。

    啓（启，※啟） #

    甲骨文金文小篆隶书楷书行书草书

    甲金文從“”從“口”，“”象以手開門之形，是“啟”的初文，本義是開啟。甲骨文或省去“又”作“启”。《説文》：“启，開也。从户从口。”段玉裁注：“按後人用啟字訓開，乃廢启不行矣。”古人穴居，“口”疑象户內之山洞（參張世超等）。或認爲“口”只是飾筆，古文字往往加“口”以增繁，如“高”、“石”等字。

    西周金文皆從“户”從“又”從“口”，戰國以後始見改從“又”爲從“攴”（攵）（卅三年鄭令劍）、“殳”（鄂君啓舟節）。西周金文從“户”從“攴”或“戈”之形是“肇”的初文。用法方面，甲骨文表示禀告，又讀作“晵”，表示天晴。金文表示開闢，中山王鼎：“闢啟封彊（疆）”，中山王圓壺：“大啟邦宇”。又表示啟迪、啟發，叔向父禹簋：“廣啟禹身”，士父鐘：“用廣啟士父身。”一説“廣啟”之“啟”表示佑助、輔助（楊樹達）。又用作族氏名、人名、地名等。啟封令癰戈：“啟封”。“啟封”是戰國時魏地，後因避漢景帝劉啟諱，改稱“開封”。

    楷書從戈、從又、從攵的字形皆有淵源；從“石”的字形是因爲“户”在隸書中像“石”。

    大陆以“啓”爲正字，港臺以“啟”爲正字。
